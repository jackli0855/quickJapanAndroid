( 商業用語的轉換 語句篇 )

學習目標

①　學習如何將日常用語轉換成商業用語

上一單元，我們介紹了如何將日常生活的字彙，轉換成適合在工作場合中使用的得體商業用語，本單元中，我們將更進一步介紹在商業場合使用的「慣用句」，在日文中有一個奇妙的現象，愈是尊敬的用法，句子長度就會愈長，商用日文的慣用句也是一樣，長度會比我們所習慣使用的句型更長，因此，得多花一點時間記憶下來。

すみません／ごめんなさい ( 抱歉。 )

商業用語

申(もう)し訳(わけ)ございません( 非常對不起。 )

どうも／ありがとう／すみません  ( 謝謝、感謝。 )

ありがとうございます  ( 非常謝謝您。 )

わかりました。( 知道了。 )

商業用語

かしこましました。承知(しょうち)いたしました。

( 您的意思我了解了。 )

日常用語

今見(いまみ)てきます。( 我現在去看看。 )

ただいま確認(かくにん)してまいります。( 我現在前往確認。 )

商業用語

日常用語

ちょっと待(ま)ってください。( 請等一下。 )

商業用語

少々(しょうしょう)お待ちください／ませ  ( 請稍候片刻。 )

どうですか。( 如何? )

いかがでしょうか。( 您意下如何? )

じゃ、そうします。  ( 那麼，就那麼做。)

それでは、そのようにいたします

( 那麼，就照那樣進行。 )

日常用語

今(いま)いません  ( 現在不在。 )

商業用語

ただいま、席(せき)を外(はず)しております

( 目前不在座位上。 )

日常用語

誰(だれ)ですか？どちらさまですか？

( 你是誰?  您是誰? )

商業用語

失礼(しつれい)ですが、お名前(なまえ)を教(おし)えていただけますか

( 不好意思，可以請您告訴我您的名字嗎? )

失礼(しつれい)ですが、お名前(なまえ)をお聞(き)きしてもよろしいでしょうか。

( 不好意思，可以請教您的名字嗎? )

日常用語

何(なん)の用(よう)ですか( 有什麼事? )

商業用語

失礼(しつれい)ですが、

ご用件(ようけん)をお伺(うかが)いしてもよろしいでしょうか。

( 不好意思，可以請問您有什麼要事嗎? )

わかりません。( 不知道。 )

わかりかねます。( 我不太清楚。 )

申(もう)し訳(わけ)ございません、私(わたし)ではわかりかねます。

( 非常抱歉，我不太清楚)
