日本人でもよく間違う敬語（2）

學習目標

①　了解日本人生活中經常使用的錯誤敬語表現。

情境一 詢問對方是否在公司時

客：田中様(たなかさま)はおられますか。

( 請問田中先生在嗎? )

錯誤

問題出在「おられます」這個用法上。「おられます」為動詞「おる」＋「～られる」，將動詞原形改為被動形「～られる」的形式，可以當作尊敬語使用，因此「おられます」這個用法看起來敬意十足。但是「おる」這個字是「いる」的謙讓語，只能用在自己身上表示謙卑，所以「おられます」為謙讓語「おる」加上尊敬語「～られる」，那麼倒底是用來表示謙卑、還是表示尊敬呢?  這是不適當的敬語用法。

正確說法:

田中様(たなかさま)はいらっしゃいますか。

情境二 傳達上司的話時

社員(しゃいん)：社長(しゃちょう)はそうおっしゃられました。

( 社長那樣說了。 )

錯誤

「おっしゃられました」為「言う」的尊敬語「おっしゃる」加上表示敬意的「～られる」形式，二項尊敬語合而為一，看起來敬意十足，非常適合用來談論社長，不是嗎?  錯錯錯，像這種在同一個字當中出現二次尊敬用法的情況，稱之為「二重敬語」，就像中文的「我很非常十分尊敬你」，相同意思的詞彙用得太多，反倒會顯得十分多餘，完全沒有尊敬的意思存在。在日文敬語中，一次最好只使用一種敬語形式，要不用「おっしゃる」、要不用「言われる」，二種都是正確的用法，但是如果合在一起，變成「おっしゃられました」，那麼就是錯誤的敬語了。

正確說法:

社長(しゃちょう)はそうおっしゃいました。

情境三 詢問時間、目的地

社員(しゃいん)：部長(ぶちょう)、いつから出張(しゅっちょう)に参(まい)られるのですか。

( 部長，您從什麼時候開始去出差呢? )

錯誤

在日商公司中，詢問出差、行程相關事情時，經常聽到這句話。這句的重點在於「参られる」，「～られる」為尊敬語形式，因此「参られる」看起來也像是尊敬語。不過，和情境一相同，「参る」是「来る／行く」的謙讓語，例如「私は台湾から参ります」「私は来年そちらにまいります」等等，和尊敬語「～られる」無法連用。必須使用「いらっしゃる」「行かれる」等等的尊敬語形式。

いつから出張(しゅっちょう)にいらっしゃるのですか。

情境四 和客戶的談話

社員(しゃいん)：打(う)ち合(あ)わせの件(けん)、お聞(き)きになっていらっしゃいますでしょうか。

( 關於討論的事，您聽說了嗎? )

錯誤

「お聞きになっていらっしゃいますでしょうか」這句看起來很長，其實將敬語省略之後，其實原句就是「聞いているでしょうか」。「聞いて」變成「お聞きになって」，「いる」變成「いらっしゃいます」，最後加上表示委婉推測的「でしょうか」，在一個句子中使用三種尊敬語。和情境二相同，這樣的「多重敬語」，表面上聽起來尊敬，其實是多餘的用法，也無法明確表達敬意。在一個句子中，只使用一項敬語表現，這是最正確的使用方法。

正確說法:

打(う)ち合(あ)わせの件(けん)、お聞(き)きになっていますか。

打(う)ち合(あ)わせの件(けん)、聞いていらっしゃいますか。

打(う)ち合(あ)わせの件(けん)、聞いていますでしょうか。

情境五 書信用語

社員(しゃいん)：いつでもご連絡(れんらく)してください。

( 無論何時都可以和我們連絡。 )

錯誤

動詞「連絡する」為「連絡」的意思，如果要說「請連絡」，表示請對方做某項動作時，可以說「連絡してください」或是「ご連絡ください」，二種都是正確的說法，但是，如果將二者合而為一，說成「ご連絡してください」可就是錯誤的敬語了。

「ご連絡してください」就是「ご連絡する」＋「ください」，而「ご＋動詞＋する」則是「謙讓語」的用法，不可以用在對方身上。因此，儘管有「～ください」，但是「ご連絡してください」仍然是不適當的敬語表現。

正確說法:

いつでもご連絡(れんらく)ください。

情境六 使役形相關用語

本日(ほんじつ)は休業(きゅうぎょう)させていただきます。

( 本日公休。 )

契約(けいやく)の件(けん)について説明(せつめい)させていただきます。

( 為您說明關於契約的事情。 )

錯誤

在商店門口，時常可以看到寫著「本日は休業させていただきます。

」的告示，表示當天商店停止營業。「させていただきます」是動詞使役形「させて」加上表示「得到」的授受動詞「いただきます」，整體意思就是「允許讓我…(進行某項動作) 」，也因為聽起來較客氣有禮，因此時常被拿來作為「敬語」使用，在店門口擺上「本日は休業させていただきます。」的告示，比起直接說「本日は休業します。」，看起來顯得禮貌多了。

但是，由於「させていただきます」是「允許讓我…」的意思，因此只能用在必須取得別人許可的時候，例如在開會中說「アドバイスをさせていただきます (請容許我提出建議) 」、生病時說「明日は休ませていただきます (請允許我明天請假」等等情況。

像是「暫停營業」這一類事情，並沒有必要得到其他人的許可，實際上也是店家老闆自己一個人做的決定，因此使用「本日は休業させていただきます。」，意思就是「請準許我今天暫停營業」，感覺將會十分奇怪。路過的人可能心裡會想「就算有人不準許，你還不是會照常停止營業」、「最好你停止營業也需要別人的同意啦」，因此，在不需要對方同意的事情上，是不適用「させていただきます」這個文法的。

正確說法:

本日(ほんじつ)は休業(きゅうぎょう)いたします。

契約(けいやく)の件(けん)について説明(せつめい)いたします。
