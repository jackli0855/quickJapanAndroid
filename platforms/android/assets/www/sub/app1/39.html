单元39    强调の「の」

             （加强语气的「の」）

 

 

学习目标

 ①　了解助词「の」的加强语气功能。

②　了解加强语气「の」的使用方法。

③　了解「动词＋のです」和「动词ます形」的区别

 

我们在先前的众多单元中，吃过了诸如可能形假设用法等等文法大餐，为了怕各位消化不良，这次就介绍一点较为轻松易懂的文法概念。

 

本单元要介绍的是「加强语气」，什么是加强语气呢?  简单来说，就是当说话者想强调自己的想法或强烈表达自己的主张时，会使用不同的字句来加强说话的语气，让对方明确知道「我想表达这件事」。例如「我不吃青椒→ 我就是不吃青椒！」 「我想出国→ 我就是想出国啦！」这类型的语句。

 

日文中，有许多加强语气的方式，其中一种就是本单元要介绍的，使用助词「の」来达到加强语气的效果。相信连没有学过日文的人就知道，日文中的助词「の」的基本意思是「的」，例如「私のパソコン(我的计算机)」，不过如果「の」不用来连接名词和名词，而放在一个句子的句尾的话，就会给人有加强语气的感觉，例如「私は学校に行きたい ( 我想去学校 )」「私は学校に行きたいのです ( 我就是想去学校 )」。

另外，许多人会将「～のです」和「动词ます形」当作同样的文法，例如「行くのです」「行きます」，以下也会针对这个部分进行介绍。

 

の

 

助词「の」除了表示「…和…」的意思之外，若是放在句尾，则会有加强语气的意思，请参照以下用法和中文翻译，相信会比较容易理解。

 

基本用法:  い形容词?动词常体  ＋　の　＋　だ／です。

　　　　　名词?な形容词　＋　なの　＋　だ／です。

 

例：

 

彼女(かのじょ)が作(つく)ったカレーライスはおいしい。

彼女(かのじょ)が作(つく)ったカレーライスはおいしいのだ！

（她煮的咖哩饭很好吃  → 她煮的咖哩饭真的很好吃!　）

 

渡辺(わたなべ)さんは英語(えいご)がうまい。

渡辺(わたなべ)さんは英語(えいご)がうまいのだ！

（渡边先生的英文很好 →　渡边先生的英文真的很好!）

 

新(あたら)しいアパートは広(ひろ)い。

新(あたら)しいアパートは広(ひろ)いのだ！

（新公寓很大 →　新公寓真的很大!）

 

私(わたし)は日本語(にほんご)の先生(せんせい)になる。

私(わたし)は日本語(にほんご)の先生(せんせい)になるのだ！

（我成为日语老师 →　我一定要成为日语老师!）

 

弟(おとうと)はピーマンを食(た)べない。

弟(おとうと)はピーマンを食(た)べないのだ！

（弟弟不吃青椒 →　弟弟绝对不吃青椒!）

 

犯人(はんにん)は金田一(きんだいち)さんだ。

犯人(はんにん)は金田一(きんだいち)さんなのだ！

（犯人是金田一先生 →　犯人是金田一先生!）

 

私(わたし)が買(か)いたいものはあのパソコンだ。

私(わたし)が買(か)いたいものはあのパソコンなのだ！

（他想买的是那台计算机 →　他想买的就是那台计算机!）

 

私(わたし)は国中(くになか)さんのことが好(す)きだ。

私(わたし)は国中(くになか)さんのことが好(す)きなのだ！

（我喜欢国中小姐 →　我就是喜欢国中小姐!）

 

金城武(かねしろたけし)は日本(にほん)では有名(ゆうめい)だ。

金城武(かねしろたけし)は日本(にほん)では有名(ゆうめい)なのだ！

（金城武在日本很有名 →　金城武在日本真的很有名!）

 

 

 

不过，在口语会话中，为了发音方便，常会将「の」发音成「ん」的形式。

在会话中，几乎所有人都会念成「ん」，请特别注意。

 

例：

 

渡辺(わたなべ)さんは英語(えいご)がうまいんだ！

新(あたら)しいアパートは広(ひろ)いんだ！

私(わたし)は日本語(にほんご)の先生(せんせい)になるんだ！

弟(おとうと)はピーマンを食(た)べないんだ！

犯人(はんにん)は金田一(きんだいち)さんなんだ！

私(わたし)が買(か)いたいものはあのパソコンなんだ！

私(わたし)は国中(くになか)さんのことが好(す)きなんだ！

 

由于「の＋だ／です」的句型有加强语气的意思，因此也可以用来回答以「为什么」开头的疑问句。

 

例：

 

どうして学校(がっこう)に来(き)ませんでした？　（为什么不来学校?）

風邪(かぜ)を引(ひ)いたんです。（我感冒了。）

風邪(かぜ)を引(ひ)きましたから。（因为我感冒了。）

 

どうしてご飯(はん)を食(た)べないの？（为什么不吃饭?）

食(た)べたら太(ふと)くなるんだ。（吃了就会变胖啊。）

食(た)べたら太(ふと)くなるから。（因为吃了会变胖。）

「动词＋のです」和「动词ます形」的差异

 

由于「动词＋のです」和「动词ます形」，都是以「です」和「ます」结尾，因此许多人会误以为两者是相同的东西，都是用来表示「动词敬体」的文法。

 

例：

 

私は朝ごはんを食べます。

私は朝ごはんを食べるんです。

先生は日本语を教えてくれます。

先生は日本语を教えてくれるんです。

 

「动词ます形」：用来向对方表示敬意、语气。

「动词のです」：用来强烈表达自己的主张，虽有「です」，但是不太礼貌。

 

 

「动词ます形」：用于想向对方表示礼貌时。

「动词のです」：用于想让对方了解自己想法时。常用在争辩或吵架时。

 

例：

 

先生は来周テストがないと言いました。

（老师说下周没有考试。）

先生は来周テストがないと言ったんです！

（老师就是说下周没有考试啊!）

 

彼はパーティーに来ます。

（他来参加派对。）

彼はパーティーに来るんです！

（他会来参加派对!）

 

私は朱と言います。

（我的名字是朱。）

私は朱と言うんです！

（我的名字叫做朱! (不是其他名字)
