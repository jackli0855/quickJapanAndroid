助词「を」

 

说到日文动词的用法，就不能不提到「を」这个助词，它是日文动词的核心要素之一。

 

回想一下中文当中使用动词的情况，「吃饭」「看书」「喝水」「玩游戏」「开门」，我们中文习惯以「动词＋名词」的形式叙述，「吃」是动词、「饭」是名词；「看」是动词，「书」是名词等等。如果少了动词和名词其中一项，意思就会变得不完整，像是「我看」「我书」，有人听得懂那是什么吗?

 

相较于中文「动词＋名词」的形式，日文中的动词，则以不同的方式使用。

 

中国語の動詞：动词＋名词

　　　　　例：吃 ＋ 饭

　　　　　　　看 ＋ 书

              喝 ＋ 水

 

日本語の動詞：名词　を　动詞

　　　　　例：ご飯（饭）　を　   食べる（吃）

   　　　　　　　本(书)　　を　　読む(读)

　　　　　　　　 水(水)　　を　　飲む(喝)

 

现在你应该可以知道，日文的语顺和中文刚好相反，而助词「を」也是中文当中所没有的概念，接在名词和动词中间。这就是「を」的最主要功能。

也许你在一时三刻内很难理解这样的用法，不过根据我们的教学经验显示，不用几天，你就会非常习惯这样的用法了，具体原因并不清楚，不过对于中文母语者来说，这似乎不是太困难的问题。

 

其他例句如下:

 

例：ゲーム を する (玩游戏)

　　ドア を 開(あ)ける (开门)

　　飲(の)み物(もの) を 買(か)う(买饮料)

　　テレビ を 付(つ)ける (开电视)

　　傘(かさ) を さす (撑伞)

　　ボタン を 押(お)す (按按钮)

　　字(じ) を 書(か)く (写字)

　　絵(え) を 描(えが)く (画画)

　　犬(いぬ) を 飼(か)う　(养狗)

 

不不，我现在知道你在想什么，但是现在不是背这些动词的时候，不要急着像背单字卡一样，急着将这些新看到的字全部背起来。关于最好的单字记忆方法，我们会在「最强！不用上课的音速日语学习术」一书中的「字汇技术」中谈到，目前，你只要跟着声音档多念几次，确定发音正确，并且了解文法架构就好。

 

动词的现在过去式

 

我们先前学过名词及形容词的现在式及过去式，那么现在学习动词时，就显得容易多了。我们先将刚才提到的例句，由「动词原形」转换为「动词ます形」，在目前阶段，我们的日文会话都是以「动词ます形」居多。你必须变形成机器人型态。

 

例：

    （Ⅲ动词）ゲーム を する　→　ゲーム を します

　　（Ⅱ动词）ドア を 開(あ)ける　→　ドア を 開(あ)けます　

　　（Ⅰ动词）飲(の)み物(もの) を 買(か)う　→　飲(の)み物(もの) を 買(か)います

　　（Ⅱ动词）テレビ を 付(つ)ける　→　テレビ を 付(つ)けます

 

以上你所看到的，是动词ます形的现在式。还记得「现在式→过去式。现在否定→过去否定」这个变化规则吗? 我们先介绍「现在否定」。

 

动词ます形现在式: ～ます

　　　　　　↓　将「ます」改为「ません」

动词ます形现在否定: ～ません

 

 

例：ゲームをします　→　ゲームをしません (不玩游戏)

　　ドアを開(あ)けます　→　ドアを開(あ)けません (不开门)

　　飲(の)み物(もの)を買(か)います　→　飲(の)み物(もの)を買(か)いません (不买饮料)

　　テレビを付(つ)けます　→　テレビを付(つ)けません (不开电视)

 

 

接着，来理解动词现在式如何变成过去式。「现在式→过去式。现在否定→过去否定」的前面部分。

 

 

动词ます形现在式: ～ます

　　　　　　↓　将「ます」改为「ました」

动词ます形过去式: ～ました

 

 

例：ゲームをします　→　ゲームをしました (玩了游戏)

　　ドアを開(あ)けます　→　ドアを開(あ)けました (开了门)

　　飲(の)み物(もの)を買(か)います　→　飲(の)み物(もの)を買(か)いました (买了饮料)

　　テレビを付(つ)けます　→　テレビを付(つ)けました (开了电视)

 

 

最后，来理解动词「现在否定」如何变成「过去否定」。「现在式→过去式。现在否定→过去否定」的后面部分。

 

 

 

动词ます形现在否定: ～ません

　　　　　　↓　加上「でした」

动词ます形过去否定: ～ませんでした

 

例：ゲームをしません　→　ゲームをしませんでした　(之前没玩游戏)

　　ドアを開(あ)けません　→　ドアを開(あ)けませんでした (之前没开门)

　　飲(の)み物(もの)を買(か)いません　→　飲(の)み物(もの)を買(か)いませんでした

(之前没有买饮料)

　　テレビを付(つ)けません　→　テレビを付(つ)けませんでした

(先前没有开电视)

 

 

我想现在你或许头脑已经一片混乱了， 「ます」「ません」「でした」跳来跳去，为了更方便理解，我们将上面的全部内容，简单整理成以下表格。

 

 

 

现在

过去

肯定

～ます

～ました

否定

～ません

～ませんでした

 

有没有什么更快速记忆的方法？有！

 

「ます！ません！ました！ませんでした！」

 

你将上面那句话多念几次，会发现它的韵律感，就像是念绕口令般，再多念几次，你就能完全记起来了。这个方法虽然原始，看起来也有些笨，不过相信我们，这绝对是记忆的好方法。

 

 

动词的邀请用法

 

最后，我们再学一个会话中非常实用的动词用法，语言学上称为「劝诱用法」，不过我倾向以白话文解释，就是「邀请用法」。什么是邀请用法？简单来说，就是邀请别人做某件事情，例如「一起喝杯咖啡吧!」 「一起去吃饭吧!」「一起去玩吧!」「钟响了，我们一起回到教室吧!」等等，中文当中的「一起...吧!」就是典型的邀请用法。当然日文中也有，而且用一个动词就可以表达。

 

邀请用法：等于中文的「一起...吧！」

 

动词ます形：～ます

　　↓　将「ます」改为「ましょう」

邀请用法：～ましょう

 

例：ご飯(はん) を 食(た)べましょう。(一起吃饭吧)

　　ゲーム を しましょう。(一起玩游戏吧)

　　遊(あそ)びましょう。(一起玩吧)

　　コーヒー を 飲(の)みましょう。(一起喝咖啡吧)

　　ジーンズ を 買(か)いましょう。(一起买牛仔裤吧)

　　ペット を 飼(か)いましょう。(一起养宠物吧)

 

如果想加强或减弱语气，可以在最后加上表示语气的助词「よ」和「か」。聆听一下声音档，你很容易就可以感受到语气上的差别。

 

加强语气:相当于中文的「一起...啦！」

 

例：コーヒー を 飲(の)みましょうよ。(一起喝咖啡啦)

　　ジーンズ を 買(か)いましょうよ。(一起买牛仔裤啦)

　　ペット を 飼(か)いましょうよ。(一起养宠物啦)

减弱语气:相当于中文的「一起...吧，好不好?」

 

例：コーヒーを飲(の)みましょうか。(一起喝咖啡，好不好?)

　　ジーンズを買(か)いましょうか。(一起买牛仔裤，好不好?)

　　ペットを飼(か)いましょうか。(一起养宠物，好不好?)
