①学会比较的说法。

②学会「より」「ほど」「どちらが」「のほうが」的用法。

 

我们学过了名词、形容词、助词的用法，本回开始就要进入应用阶段了。这里所学到的，不再是枯燥的文法例句，而是和我们生活息息相关的语言表现。这次我们要介绍的是日文中「比较」的说法。

 

什么是「比较」说法呢? 像是「我比你高」「冬天比夏天冷」「日本不像台湾一样炎热」等等、具有将二者互相比较意思的语句。日文当中，最常用以表示「比较」的字，就是助词「より」「ほど」，以及「どちらが」「のほうが」二个简短句型。

 

より

 

「より」相当于中文「我比你高」的「比」，常用来表示二者的比较关系。

这里要注意的是「比较物」与「被比较物」的位置。

 

 

より：(比较物)　は　　(形容词)　です。

   　　　　　　↓

      (比较物)　は　(被比较物)　より　(形容词)　です。

　　　

例：台(たい)湾(わん) は 暑(あつ)い です。

　　(台湾很炎热。)

 

台(たい)湾(わん) は 日(に)本(ほん)より 暑(あつ)い です。

    (台湾比日本炎热 。)

 

如上所述，将被「被比较物＋より」放在句子中「は」和「形容词」的中间，就可以达到比较的效果。

 

例：日(に)本(ほん) は 台(たい)湾(わん)より 寒(さむ)い です。(日本比台湾寒冷)

　　電(でん)車(しゃ) は 自(じ)転(てん)車(しゃ)より 速(はや)い です。(电车比自行车快)

　　ワンピース は スカートより 高(たか)い です。(洋装比裙子贵)

　　図(と)書(しょ)館(かん) は スーパーより 静(しず)か です。(图书馆比超市安静)

　　都(と)市(し) は 田(い)舎(なか)より 便(べん)利(り) です。(都市比乡下方便)

　　ひまわり は 草(くさ)より きれい です。(向日葵比杂草漂亮)

 

ほど

 

上面例句中，有一句是「台湾は日本より暑いです。(台湾比日本热)」，那么如果我们想以否定的型式「日本不像台湾那么热」来叙述的话，要怎么说呢？

 

？　日(にほん)本は台(たい)湾(わん)より暑(あつ)くありません  (日本比台湾不热)。

 

相信有不少人会将后面的形容词，直接改成否定形吧。不过很抱歉，虽然文法上没有大问题，不过这种用法是错的。「より」后面不能接否定形。

因此以下的例句也是错的。

 

　？　自(じ)転(てん)車(しゃ)は電(でん)車(しゃ)より速(はや)くありません(自行车比电车不快)。

？　田(い)舎(なか)は都(と)市(し)より便(べん)利(り)ではありません(乡下比都市不方便)。

 

如果我们想以「日本不像台湾那么热」「自行车不像电车那么快」等等否定的形式叙述的话，一般会使用「ほど」这个助词。

 

ほど：相当于中文的「不像…」，用于否定句型中。

 

句型：(比较物)　は　(被比较物)　ほど　(形容词否定形)。

 

例：日(に)本(ほん) は 台(たい)湾(わん) ほど 暑(あつ)くありません。(日本不像台湾那么热)

　　自(じ)転(てん)車(しゃ) は 電(でん)車(しゃ) ほど 速(はや)くありません。(自行车不像电车那么快)

　　ジュース は コーヒー ほど 高(たか)くありません。

　(果汁不像咖啡那么贵)

　　田(い)舎(なか) は 都(と)市(し) ほど 便(べん)利(り)ではありません。

　　(乡下不像都市那么热闹)

　　私(わたし) は 王(ワン)建(ジェン)民(ミン) ほど 有(ゆう)名(めい)ではありません。

(我不像王建民那么有名)

　　期(き)末(まつ)テスト は 中(ちゅう)間(かん)テストほど 大(たい)変(へん)ではありません。

(期末考不像期中考那么累人)

因此，同样一句话，可以有二种不同的说法。

 

例：夜(よ)市(いち) は スーパーより うるさい です。

(夜市比超市吵杂)

　　スーパー は 夜(よ)市(いち)ほど うるさくありません。

 (超市没有像夜市那么吵杂)

 

　　辞(じ)書(しょ) は 雑(ざっ)誌(し)より 厚(あつ)い です。

　　(字典比杂志厚)

　　雑(ざっ)誌(し) は 辞(じ)書(しょ)ほど 厚(あつ)くありません。

    (杂志没有像字典那么厚)

 

询问及回答

 

介绍另一种在会话中相当常见的比较句型，主要用于询问对方的意见，例如:「(在餐厅中) 咖哩饭和蛋包饭，哪一个比较好吃? 」「(购物时) 这件衣服和那件衣服，哪一件比较可爱?」。这句话最广为人知的地方，就是日本节目「料理东西军」了，二位主持人在节目最后二选一时，一定会说「今夜のご注文は、どっち？（今夜的餐点，你选择哪一样？）」。

 

要询问对方意见、以及回答对方问题时，我们可以用以下二种句型:

 

①　询问句型：　名词　と　名词　、どちらが　形容词　ですか。

  (____和____ ，哪一种比较____呢? )

 

解说：

「と」：助词，表示「和」。

 

「どちら」：和先前介绍过「こそあど」的「どれ」用法大致相同，是「哪个」的意思。会话中可以省略为「どっち」。

「が」：虽然我们已经习惯了「～は～ですか。」这个句型。但是这里不用「は」，而改用「が」这个助词。

 

 

例：カレーライス と オムライス、どちらが おいしい ですか。

(咖哩饭和蛋包饭，那一种比较好吃?)

　　この服(ふく) と あの服(ふく)、どちらが かわいい ですか。

   (这件衣服和那件衣服，那一件比较可爱?)

 

　　犬(いぬ) と 猫(ねこ)、どちら が 好(す)き ですか。

   (狗和猫，你比较喜欢哪一种?)

　　自(じ)転(てん)車(しゃ) と バイク、どちらが 便(べん)利(り) ですか。

(自行车和机车，哪一种比较方便?)

 

 

另外，「～、どちらが～」的句型，只能用在二选一的情况，例如「犬と

猫」「自転車とバイク」，如果是三种东西以上，「犬と猫と兎」「自転車と

バイクと車」就不能使用「どちら」，而要改用「どれ」。

 

「どちら」：用于从二样东西当中，选择一样的情况。

「どれ」：用于从三样以上的东西中，选择一样的情况。

 

例：犬(いぬ)と 猫(ねこ)と 兎(うさぎ)、どれが 好(す)き ですか。

(狗和猫和兔子，你比较喜欢哪一种?)

 

　　自(じ)転(てん)車(しゃ)と バイクと 車(くるま)、どれが 便(べん)利(り) ですか。

(自行车和机车和汽车，哪一种比较方便?)

 

　　小(しょう)テストと 中(ちゅう)間(かん)テストと 期(き)末(まつ)テスト、どれが 難(むずか)しい です

　　か。

(随堂考和期中考和期末考，哪一种比较难?)

 

② 回答句型： 名词　のほうが　形容词　です。

      (____(的那一方)比较____。 )

 

解说：

 

「の」：连接名词和名词，相当于中文的「的」。

「ほう」：汉字写作「方」，相当于中文中「那一方」的「方」。

 

例：A：カレーライス と オムライス、どちらが おいしい ですか。

(咖哩饭和蛋包饭，哪一种比较好吃？)　

B：カレーライス のほう が おいしい です。

(咖哩饭比较好吃)　

　

A：この服(ふく) と あの服(ふく)、どちらが かわいい ですか。

(这件衣服和那件衣服，哪一件比较可爱？)　

B：あの服(ふく) のほうが かわいい です。

   (那件衣服比较可爱)

　　

A：犬(いぬ) と 猫(ねこ)、どちら が 好(す)き ですか。

(狗和猫，你比较喜欢哪一种？)　

B：猫(ねこ) のほうが 好(す)き です。

(我比较喜欢猫)

 

　　A：自(じ)転(てん)車(しゃ) と バイク、どちらが 便(べん)利(り) ですか。

(自行车和机车，哪一种比较方便？)　

　　B：バイク のほうが 便(べん)利(り) です。

(机车比较方便)
