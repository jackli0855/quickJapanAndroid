　　　　　　（动词て型相关句型）

 

学习目标

 

①　学习句型「～てください」

②　学习许可句型「～てもいいです」

③　学习禁止句型「～てはいけません」

④　学习相关疑问词和问句

 

我们在上一单元，学习了日文动词变化的重要部分「て形」，也介绍了「て形」的基本用法─接续复数的句子和动词。如果你能大致理解这些内容，看得懂我们在说什么，那么代表你已经通过最初关卡，剩下的只是时间问题，需要时间慢慢去熟悉、熟练动词变化，一定到变成反射动作为止，什么是反射动作，就像是你一听到「おはよう」就知道是「早安」一样(相信我，不会花很多时间)。

 

但是，「动词て形」的功用远不止如此。如果只有单纯接续的功能，充其量不过是英文中「and」的代替品，实在很难说服大家这是学习日文动词变化时非常重要的部分。「动词て形」之所以重要，是因为它除了接续句子的功能外，还有「万能胶」的功用，「动词て形」能够组成很多很多的日文实用句型、惯用句，换句话说，许许多多的日文句型中，都可以看到「动词て形」的身影，例如「ちょっと待って！」这句话是不是很耳熟呢?

事实上，在进行日常会话时，几乎每讲二句话就会用到「て形」的相关用法，因此，如果想流畅地进行沟通，除了学会「动词て形」的变化方法外，还得学习以「动词て形」为基础的各式句型。

我们会用二单元的篇幅，分二次来介绍「动词て形」相关的惯用句型和用法。本单元介绍三项句型，分别为表示请求的、表示允许的、表示拒绝的、以及相关的问句用法。如果你已经理解了上一单元内容，那么相信一定能够对接下来的内容驾轻就熟。

 

 

～てください

 

 

「～てください」是日文中的请求用法，相当于中文的「请…」。

「ください」亦可写成汉字「下さい」。

 

基本句型:  动词て形＋ください

　　　　　( 请… )　

 

例(れい)：

 

待(ま)つ　　→　待(ま)って　→　待(ま)ってください

書(か)く　　→　書(か)いて　→　書(か)いてください

見(み)る　　→　見(み)て　　→　見(み)てください

食(た)べる　→　食(た)べて　→　食(た)べてください

 

する　　→　して　　→　してください

来(く)る　　→　来(き)て　　→　来(き)てください

 

 

～てもいいです

 

「～てもいいです」是日文中表示许可的用法，相当于中文的「可以…」。

 

「～てもいいです」＝「动词て形」＋「も」＋「いいです」

　　　　　　　　　　　 动作      +   也　+   可以

 

日文直译为「做～动作也可以」之意，也就是「可以…」的意思。

 

一般常以对话的形式出现:

 

Q： ～ てもいいですか？

A： はい、～てもいいですよ。

 

问句使用疑问助词「か」，为「我可以….吗?」之意，回答句使用表示语气的助词「よ」，为「OK，你可以…喔」的意思。　　　

 

例：

 

A：この本(ほん)、借(か)りてもいいですか。

B：はい、借(か)りてもいいですよ。

( A : 我可以借这本书吗?  )

( B : 嗯，可以借啊。 )

 

A：このパソコン、使(つか)ってもいいですか。

B：はい、使(つか)ってもいいですよ。

( A : 我可以用这台计算机吗?  )

( B : 嗯，可以用啊。 )

 

A：冷蔵庫(れいぞうこ)のケーキ、食(た)べてもいいですか。

B：はい、食(た)べてもいいですよ。

( A : 我可以吃冰箱里的蛋糕吗?  )

( B : 嗯，可以吃啊。 )

 

以上介绍的是肯定句，也就是允许对方的情况，那么如果是否定，也就是想拒绝对方的时候该怎么说呢?

 

 

～てはいけません

 

「～てはいけません」是日文中表示禁止的用法，相当于中文的「不行…」。

 

基本句型: 「动词て形」＋は＋いけません。

　　　　 　( 不行… )

 

「～てはいけません」可以使用在上述对话中，表示拒绝对方某动作，也可以用于一般句子中，表示禁止对方进行某动作。

 

基本对话形式 :

 

A：～てもいいですか。

B：いいえ、～てはいけません。

 

例：　

 

A：この本(ほん)、借(か)りてもいいですか。

B：いいえ、借(か)りてはいけません。

( A : 我可以借这本书吗?  )

( B : 不，不行借。 )

 

A：このパソコン、使(つか)ってもいいですか。

B：いいえ、使(つか)ってはいけません。

( A : 我可以使用这台计算机吗?  )

( B : 不，你不行使用。 )

 

A：冷蔵庫(れいぞうこ)のケーキ、食(た)べてもいいですか。

B：いいえ、食(た)べてはいけません。

( A : 我可以吃冰箱里的蛋糕吗?  )

( B : 不，不行吃。 )

 

タバコを吸(す)ってはいけません。

( 不行抽烟。 )

 

法律(ほうりつ)を破(やぶ)ってはいけません。

( 不行违反法律。 )

 

台風(たいふう)の時(とき)に、海辺(うみべ)で遊(あそ)んではいけません。

( 台风来的时候，不能到海边玩。 )

 

ご飯(はん)を食(た)べる時(とき)に、テレビを見(み)てはいけません。

( 吃饭时，不能看电视。 )

 

夜中(よなか)にギターを弾(ひ)いてはいけません。

( 半夜不行弹吉他。 )

 

相关问句用法

 

我们学了「～てください(请你…)」的句型，那么问句该如何说呢?

 

句型:   ～てくれますか。

(可以请你…吗?)

 

关于这项文法，我们之后会再学到，现在先暂时当作惯用句记起来吧!

实际使用时，由于是询问句的关系，听起来会比「～てください」这种直接进行请求的请求句，更为委婉、有礼貌一些。

 

例：

 

　待(ま)ってください　　待(ま)ってくれますか。

　書(か)いてください　　書(か)いてくれますか。

　見(み)てください　　　見(み)てくれますか。

　食(た)べてください　　食(た)べてくれますか。

 

另外，我们刚才介绍过的～てもいいですか问句，也有其他种回答方式。

 

Q：～てもいいですか。( 可以…吗? )

 

A：

①　はい、～てもいいですよ。( 嗯，可以…。 )

②　はい、どうぞ。( 嗯，请。)

③　いいですよ。( 可以啊。 )

 

④　いいえ、～てはいけません。( 不，不行…。 )

⑤　だめです。( 不行。)

 

 

也可以直接使用拒绝的用法，来当作疑问句。

 

Q：～てはいけませんか。( 不可以…吗? )

 

A：はい、～てはいけません。( 是的，不可以…。 )

　　いいえ、～てもいいですよ。( 不，可以…喔。 )

 

例：

 

A：タバコを吸(す)ってはいけませんか。

( 不可以抽烟吗? )

B：はい、吸(す)ってはいけません。

( 是的，不可以抽烟。 )

　　いいえ、吸(す)ってもいいですよ。　

( 不，可以抽烟。 )
