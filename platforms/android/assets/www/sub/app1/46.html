单元46    命令と禁止の言い方

           （命令和禁止的说法）


学习目标

①　学会表示命令的「动词命令形」

②　学会表示命令的「～なさい」用法

③　学会表示禁止的「动词原形＋な」用法

 

本单元介绍的内容，就某种程度来说也算是「非常地实用」(笑)。本回的主题是学习日文中表示「命令」和「禁止」的说法。

 

所谓命令的说法，相信不用解释大家都知道，就是指命令某人去做什么事情，常见于父母亲命令小孩、老师命令学生、上司命令部下、主人命令宠物等等。我们之前学过「～てください」这个句型，是「请…」的意思，虽然是请求别人去做什么事情，但是语气还是没有像命令一般强烈。

日文中的命令说法，是以动词变化的形式来表示的，有所谓的「动词命令形」，相信对于已经克服了许许多多动词变化难关的各位来说，一定是小菜一碟。

 

另外，由于「动词命令形」的语气较为强烈，通常只有男生会使用，如果女生想表达命令语气，像是妈妈早上叫小孩起床的话，就会使用「～なさい」这个句型，将之想成不粗鲁的「动词命令形」即可。

 

 

最后，是日文中「禁止」的说法，用来表示禁止别人做什么样的事情，相当于中文的「别…!」，例如「别动!」「别抽烟!」等等，这时会使用「动词原形＋な」这个文法。同样地，「动词原形＋な」语气也很强烈，不适合女性使用，一般女性表示禁止的说法为「～ないでください」，我们之前也学习过这个句型。

详细文法及例句介绍，请参照以下说明。

 

动词命令形

 

「动词命令形」，顾名思义就是命令他人时使用的句型，相当于中文的「去…!!」。日文中的命令语气，则是以动词变化的形式表现。

 

动词变化方法

 

第一类动词

 

将动词罗马拼音字尾的「u」改为「e」即可。

 

例：

貸(か)す　Kasu →　貸(か)せ　Kase

書(か)く　Kaku →　書(か)け  Kake

泳(およ)ぐ　Oyogu →　泳(およ)げ Oyoge

言(い)う　Iu →　言(い)え  Ie

待(ま)つ　Matsu →　待(ま)て  Mate

帰(かえ)る　Kaeru →　帰(かえ)れ  Kaere

飛(と)ぶ　Tobu →　飛(と)べ  Tobe

読(よ)む　Yomu →　読(よ)め  Yome

 

第二类动词

 

将动词字尾的「る」改为「ろ」即可。

 

食(た)べる　→　食(た)べろ

着(き)る　→　着(き)ろ

忘(わす)れる　→　忘(わす)れろ

寝(ね)る　→　寝(ね)ろ

開(あ)ける　→　開(あ)けろ

見(み)る　→　見(み)ろ　

 

不规则动词

没有规则，请直接记起来吧。

 

する　→　しろ

勉強(べんきょう)する　→　勉強(べんきょう)しろ

買物(かいもの)する　→　買物(かいもの)しろ

来(く)る　→　来(こ)い

 

例：

 

ピーマンを食(た)べろ！

（去吃青椒!）

家(うち)に帰(かえ)れ！

（给我回家!）

早(はや)く宿題(しゅくだい)をしろ！

（快去写作业!）

今(いま)すぐここに来(こ)い！

（现在马上来这里!）

今月(こんげつ)の営業成績(えいぎょうせいせき)を見(み)ろ！

（你去看看这个月的作业绩!）

反省書(はんせいしょ)を書(か)け！

（给我写悔过书!）

 

 

なさい

 

「动词命令形」的语气很强烈，是很强烈的命令语气，一般只有责备他人时才会使用，而且多为男性使用。如果说话者是女性，一般来说会使用「なさ」这个用法来表示命令的语气，最常听到的就是每天早上妈妈叫小孩起来的情况了，「早く起きなさい( 快点起床! )」。

 

基本用法: 动词ます形－「ます」＋「なさい」

（表示轻微命令。）

 

例：

 

早(はや)く起(お)きなさい！

（快起来!）

ピーマンを食(た)べなさい！

（快吃青椒!）

 

 

家(うち)に帰(かえ)りなさい！

（给我回家!）

早(はや)く宿題(しゅくだい)をしなさい！

（快点写作业!）

今(いま)すぐここに来(き)なさい！

（现在马上到这里来!）

今月(こんげつ)の営業成績(えいぎょうせいせき)を见なさい！

（你看看这个月的业绩!）

反省書(はんせいしょ)を書(か)きなさい！

（去写悔过书!）

 

 

动词原形＋な

 

「动词原形＋な」为表示「禁止」的用法，禁止别人进行某项动作，相当于中文的「别…!」。

 

如同「动词命令形」，「动词原形＋な」的语气也相当强烈，多为男性使用，女性则大多使用「～ないでください」这个句型来表示禁止的语气。

 

基本用法:  动词原形＋な。

（别…!）

 

例：

 

仕事(しごと)をサボるな！

（工作别偷懒!）

 

タバコを吸(す)うな！

（别抽烟!）

ここに来(く)るな！

（别来这里!）

夜食(やしょく)を食(た)べるな！

（别吃消夜!）

俺(おれ)のことを忘(わす)れるな！

（别忘记我!）

俺(おれ)の日記(にっき)を見(み)るな！
（别看我的日记!）

勝手(かって)に手紙(てがみ)を開(あ)けるな！

（别随便打开我的信!）

宿題(しゅくだい)をする前(まえ)に寝(ね)るな！

（在写作业前别睡觉!）

変(へん)なことを書(か)くな！

（别写奇怪的事情!）

台風(たいふう)の日(ひ)に海辺(うみべ)に行(い)くな！

（台风天别去海边!）
