本单元要介绍的是「动词使役形」，相当于中文的「让…去做…」，用以表示让某人依照自己的意见去做某样事情。依照自动词和他动词的不同，使用的「动词使役形」句型也不同。说明结束。

详细例句和文法解说，请参照以下篇幅。

使役形变化方式

 

在解说文法之前，我们先来了解如何将动词由「动词原形」变化为「动词使役形」。

 

动词变化方法 :

 

第一类动词

 

将动词罗马拼音字尾的「U」改成「A」 之后，加上「せる」。

但如果动词字尾是「う」改成「わ」 之后，加上「せる」。

 

例：

書(か)く　Kaku  →　書(か)か　Kaka  →　書(か)かせる

泳(およ)ぐ　Oyogu  →　泳(およ)が　Oyoga  →　泳(およ)がせる

言(い)う　Iu  →　言(い)わ　Iwa  →　言(い)わせる

待(ま)つ　Matsu  →　待(ま)た　Mata  →　待(ま)たせる

読(よ)む　Yomu  →　読(よ)ま　Yoma  →　読(よ)ませる

 

第二类动词

 

直接去掉动词字尾的「る」，加上「させる」。

 

食(た)べる　→　食(た)べさせる

着(き)る　→　着(き)させる

忘(わす)れる　→　忘(わす)れさせる

寝(ね)る　→　寝(ね)させる

開(あ)ける　→　開(あ)けさせる

見(み)る　→　見(み)させる　

 

 

不规则动词

没有规则，只能直接记起来。

 

する　→　させる

勉強(べんきょう)する　→　勉強(べんきょう)させる

買物(かいもの)する　→　買物(かいもの)させる

来(く)る　→　来(こ)させる

 

使役形用法

 

「动词使役形」如同字面上的意思，主要用于表示「让对方依照自己的意思去做某件事」，相当于中文的「让…去做…」。同时依照「自动词」和「他动词」的区别，使用的句型也略有不同。

「动词使役形」其实有二层意思，

 

①　表示命令，要求他人依自己的想法去做某件事，

    相当于中文的「叫…去做…」「要…去做…」。

②　表示许可，允许他人、让他人去做某件事，

    相当于中文的「让…去做…」。

 

 

基本用法① :  主词 　は　対象　を　自动词使役形。　

 

 

例：

部下(ぶか)を営(えい)業(ぎょう)に行(い)かせる。

(要部下去跑业务。)

先生(せんせい)は生(せい)徒(と)を立(た)たせる。

(老师要学生站起来。)

先生(せんせい)は生(せい)徒(と)を外(そと)で遊(あそ)ばせる。

(老师让学生在外面玩。)

成績(せいせき)が悪(わる)かったから、子(こ)供(ども)を塾(じゅく)に行(い)かせる。

(由于成绩很差，因此要小孩去补习班。)

成績(せいせき)が良(よ)かったから、子(こ)供(ども)を遊(ゆう)園(えん)地(ち)に行(い)かせる。

(由于成绩很好，因此让小孩去游乐园玩。)

変(へん)な顔(かお)をして、泣(な)いている子(こ)供(ども)を笑(わら)わせる。

(扮鬼脸让哭的小孩子笑出来。)

変(へん)な顔をして、笑(わら)っている子(こ)供(ども)を泣(な)かせる。

(扮鬼脸将正在笑的小孩弄哭。)

娘(むすめ)をプールで自(じ)由(ゆう)に泳(およ)がせる。

(让女儿在游泳池中自由游泳。) 表示许可

娘(むすめ)を無理(むり)にプールで泳(およ)がせる。

(强迫女儿要在游泳池里游泳。) 表示命令

 

 

若后方为他动词时，由于他动词之前必须要有「を」，

因此其使役用法如下所示:

 

基本用法② : 

主词 　は　対象　に　东西 を　他动词使役形。　

 

例：

父(ちち)は新之助(しんのすけ)にピーマンを食(た)べさせる。

(爸爸要新之助吃掉青椒。)

先生(せんせい)は生(せい)徒(と)に教(きょう)科(か)書(しょ)を読(よ)ませる。

(老师要学生念课本。)

俺(おれ)は弟(おとうと)に自(じ)分(ぶん)のパソコンを使(つか)わせる。

(我让弟弟使用我的计算机。)

部(ぶ)長(ちょう)は部下(ぶか)にお客様(きゃくさま)に電(でん)話(わ)を掛(か)けさせる。

(部长要部下打电话给客人。)

中(なか)田(だ)さんは彼(かれ)氏(し)に夜(や)食(しょく)を買(か)わせる。

(中田小姐要她的男朋友去买消夜。)

もう３０歳(さい)なのに、生(せい)活(かつ)費(ひ)などを全(ぜん)部(ぶ)両(りょう)親(しん)に払(はら)わせている。

(明明已经三十岁了，却还是要父母亲付全部的生活费。)

身(み)の安全(あんぜん)のため、警察官(けいさつかん)たちに拳銃(けんじゅう)を持(も)たせている。

(为了安全而让警察们带着手枪。)

どうしてもやりたいと言(い)ったので、息子(むすこ)に空(から)手(て)を習(なら)わせた。

(因为他说无论如何都想学，因此让儿子去学了空手道。)

 

 

特殊的使役形用法

 

另外，还有一些动词使役形用法，不包含在以上二者之中，既不是表示「命令」、也不是表示「许可」，而是表示较抽象的意思，相当于中文的「让…有什么样的感觉」。

这些特殊用法的动词，包括「悩む」「困る」「驚く」「びっくりする」「がっかりする」等等表示感情的动词，以及「考える」「反省する」等等表示思考的动词。

 

如果句型中使用的是以上二种类的动词，就不能解释为「命令」或「许可」，因为文意上不通，因此就必须解释为「让…有…的感觉」。

 

例:

かなり難(むずか)しい宿題(しゅくだい)が出(で)て、頭(あたま)を悩(なや)ませている。

(回家作业很难，让我非常烦恼。)

彼(かれ)はいきなり高校(こうこう)をやめると言(い)って、家(か)族(ぞく)を困(こま)らせた。

(他突然说要从高中休学，让家人很困扰。)

このパソコンは性能(せいのう)がすごくて、見(み)た人(ひと)を驚(おどろ)かせる。

(这台计算机的性能很好，让看到的人都吓一跳。)

もう！そんなことを言(い)って俺(おれ)をがっかりさせるなよ！

(真是的! 别说那种话让我泄气啦!)

講演(こうえん)を通(つう)じて、生(せい)徒(と)に自(じ)分(ぶん)の長(ちょう)所(しょ)と短所(たんしょ)を考(かんが)えさせる。

(透过演讲，让学生思考自己的优点和缺点。)

嘘(うそ)をつく人(ひと)を反省(はんせい)させる方法(ほうほう)を教(おし)えてください。

(请告诉我如何让说谎的人反省自己。)
