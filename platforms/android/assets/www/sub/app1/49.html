学习目标


①了解助词「も」「に」「で」「と」的意思

②了解助词「も」「に」「で」「と」的基本用法

 

经过上一回的练习，我们已经克服了动词入门时最棘手的部分，你已经大致掌握日文动词的技巧了。包括:「区分为动词原形和动词ます形」、现在过去式、以及用替换字尾罗马拼音字母的方式，将动词原形变换为动词ます形。随着文法的加深，必须学习的东西也会愈来愈多，但是基本技巧都是一样，特别是替换字尾罗马拼音字母，这个技巧我们以后会常常用到。实际上，只要熟练后，往后的动词学习上，并不会遇到太大的困难。

 

这一单元，我们要来学习「も」「に」「で」「と」这四个助词，这些助词常常和动词一起使用。什么意思呢? 我们来看看中文的情况:「我也喜欢小狗」「我到学校去」「我和同学一起写作业」，这些例句中，「也」「到」「和」这三个字是助词，本身没有意义，主要的功能是和后面的动词「喜欢」「去」「写作业」一起使用，形成一句完整的句子。日文中的「も」「に」「で」「と」也是一样，和动词一起使用，形成一句意思完整的句子。

 

学习重点有:①了解助词「も」「に」「で」「と」的意思。②了解助词「も」「に」「で」「と」的基本用法。我们将依序介绍下去。

も

 

「も」：等同中文的「也」。是使用频率非常高的助词。

 句型:  (名词)　も　(名词／形容词／动词)　です。

 

例：鈴(すず)木(き)さん は 大(だい)学(がく)生(せい) です。私(わたし) も 大(だい)学(がく)生(せい) です。

(铃木先生是大学生，我也是大学生)

　　私(わたし) は サラリーマン です。彼(かれ) も サラリーマン です。

(我是上班族，他也是上班族)

　　台(たい)北(ぺい)の夜(よ)市(いち) は 賑(にぎ)やか です。九(きゅう)份(ふん) も 賑(にぎ)やか です。

(台北的夜市很热闹，九份也很热闹)

　　学(がく)食(しょく)の料(りょう)理(り) は まずい です。このレストランの料(りょう)理(り) も まず

    い です。

(学生餐厅的料理很难吃，这间餐厅的料理也很难吃)

　　ピザ を 買(か)いました。パスタ も 買(か)いました。

(买了比萨，也买了意大利面)

　　両(りょう)親(しん) は 日(に)本(ほん) に 行(い)きます。私(わたし) も 行(い)きます。

(双亲要去日本，我也要去)

 

 

 

 

 

 

に

 

助词「に」的后面，主要接和「移动」有关的动词(如「行(い)く(去)」「来(く)る(来)」「帰(かえ)る(回家)」），表示「移动的目的地」。

 

「に」：表示「目的地」

 句型:

(名词)は　(场所或地方名称)  に　(行(い)きます?来(き)ます?帰(かえ)ります）。

 

例：私(わたし) は 日(に)本(ほん) に 行(い)きます。(我去日本)

　　私(わたし) は 会(かい)社(しゃ) に 行(い)きます。(我去公司)

　　私(わたし) は アメリカ に 行(い)きました。(我了去美国)

　　彼(かれ) は 学(がっ)校(こう) に 来(き)ませんでした。(他之前没来学校)

　　お客(きゃく)さん は 家(いえ) に 来(き)ました。(客人来了家里)

　　お母(かあ)さん は 実(じっ)家(か) に 帰(かえ)ります。(母亲回去老家)

　　弟(おとうと) は 家(いえ) に 帰(かえ)りました。(弟弟回家了)

 

で

 

助词「で」最基本的用法，就是用以表示「方法．手段」。

 

「で」：表示方法或手段。相当于中文的「利用...」「使用...」。

 

  句型:　 (名词)　は　（名词）　で　（动作）。

 

例：私(わたし) は 鉛(えん)筆(ぴつ) で 字(じ)を書(か)く。(我用铅笔写字)

    私(わたし) は フォーク で パスタを食(た)べます。　(我用叉子吃意大利面)

    私(わたし) は バス で 学(がっ)校(こう)に行(い)きます。(我坐(使用)公交车去学校)

    私(わたし) は 飛(ひ)行(こう)機(き) で 日(にほん)本に行(い)きます。(我坐(使用)飞机去日本)

    私(わたし) は 紙(かみ) で 鶴(つる)を作(つく)りました。(我用纸折了鹤)

    私(わたし) は 紙(かみ)粘(ねん)土(ど) で ドラゴンを作(つく)りました。

(我用纸黏土做了龙)

    私(わたし) は 自(じ)分(ぶん)の力(ちから) で 仕(し)事(ごと)を探(さが)します。

(我用自己的力量找工作)

    私(わたし) は 両(りょう)親(しん)の支(ささ)え で マンションを買(か)いました。

(我以父母亲的援助买了公寓)

 

と

 

在之前的单元中，我们学过「と」这个助词的用法，例如「先生と学生」「松と竹と桜」等等，「と」相当于中文的「和」。而「と」和动词一起使用时，同样也是「和」的意思。

 

「と」:表示连带关系，等同中文的「和」。

  句型: 私 は （人名） と  （动作）。

         (我和某人～～)

 

这里要特别注意的是，中文的「和」，是接在人名之前，像「我『和』『他』一起去旅行」，然而日文相反，「と」必须接在人名之后，例如「私は『彼』『と』日本に行きます。」

 

例：私(わたし)は 先(せん)生(せい)と 職(しょく)員(いん)室(しつ)に 行(い)きました。

(我和老师去办公室)

　　私(わたし)は 父(ちち)と 食(しょく)事(じ)を しました。

(我和父亲去吃饭)

　　母(はは)は ペットと 海(かい)外(がい)旅(りょ)行(こう)を しました。

(母亲和宠物去海外旅行)

　　上(うえ)野(の)さんは 彼(かの)女(じょ)と 映(えい)画(が)を 見(み)ました。

(上野先生和女朋友去看了电影)

　　私(わたし)は 弟(おとうと)と妹(いもうと)と 実(じっ)家(か)に 帰(かえ)りました。

(我和弟弟妹妹回了老家)

　　社(しゃ)長(ちょう)は 部(ぶ)長(ちょう)と 出(で)掛(か)けました。

(社长和部长出去了)

 

句型整理

即使是架构相同的句子，随着助词的不同，意思也会相差很多。请比较看看我们在这一单元学到的四个助词，如下。

 

①私(わたし) は 日(に)本(ほん) に 行(い)きます。

　我去日本。以「に」表示目的地。

②私(わたし) も 日(に)本(ほん)に行(い)きます。

　我也去日本。以「も」表示「也」的意思。

③私(わたし) は 友(とも)達(だち) と 日(に)本(ほん)に行(い)きます。

　我和朋友去日本。以「と」表示「和」的意思。

 

 

④私(わたし) は 飛(ひ)行(こう)機(き) で 日(に)本(ほん)に行(い)きます。

　我坐飞机去日本。以「で」表示交通工具．方法手段。

 

 

以上四个句型，同时也是会话中常用到的句型，请务必完全理解其意思。
