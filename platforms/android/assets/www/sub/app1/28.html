学习目标

 ①　学习「动词ない形」的变化方法

②　了解动词常体的过去式

③  了解动词常体的过去否定用法

④  学习句型「～ないでください」

 

 

我们在先前关于「动词」的章节中介绍过，日文的动词有二种型态: 「动词敬体」和「动词常体」，就像变形金钢有车辆和机器人二种型态一样。「动词敬体」较为尊敬，因此在对老师、上司或其他地位较高的人说话时，必须使用「动词ます形」；另外，「动词常体」则是较普通的说法，当我们和朋友、家人、同辈者、或是辈份较低者(如学弟妹等)说话时，则使用「动词普通形」。就像变形金钢在众人前是车辆，但是在朋友间则会变成机器人一样。

 

由于在开始学习日文的阶段，语言能力尚未纯熟，因此习惯一开始以「动词敬体(动词ます形)」为切入点进行学习，使用「动词ます形」来说话的话，无论说话对象是谁，都不会有不礼貌的情形发生。不过，我们已经将「动词ます形」的时式用法学得差不多了，各位应该也很习惯「动词ます形」的存在了吧。 话虽如此，我们必须学习「活的语言」，也许以「动词

 

ます形」对长辈说话很恰当，但是当你和朋友、家人等关系较亲密的人聊天时，使用「动词ます形」则会有种说不出来的奇怪感觉。因此，我们除了「ます形」之外，也必须学习「普通形」的说法。

我们很早以前就学过「动词原形」了，「动词普通形」和「动词原形」完全相同，不需要重新学习。

 

本单元的重点，在于学会动词的「否定用法」。我们学过「～ません」「ませんでした」，这次的任务，只是将「～ません」「ませんでした」改成常体型态，使各位除了面对长辈之外，和家人朋友也能自在地聊天。

 

 

动词ない形

 

「动词否定形」又称为「动词ない形」，「ない」即为动词原形用以表示否定时会用的助词。

 

动词现在否定： 「动词原形」+ 「ない」＝「动词ない形」

 

动词分三种，「动词ない形」的变化方式也分为三种，基本变化方式和「动词ます形」大同小异。

 

 

第一类动词

 

变化方法: 将动词字尾的u 改成 a 之后，加上「ない」

　　　　　但如果动词结尾是「う」的话，就必须改成「わ」后加上

「ない」。

 

例：

 

行(い)く  iku 　→　行か ika 　→　行かない  (不去)　

読(よ)む  yomu　→　読ま  yoma　→　読まない  (不读)

買(か)う  kau　→　买わ  kawa　→　买わない   (不买)

待(ま)つ  matsu　→　待た  mata　→　待たない (不等)

帰(かえ)る  kaeru　→　帰ら  kaera　→　帰らない (不回家)

売(う)る  uru　→　売ら  ura　→　売らない  (不卖)

 

第二类动词

 

变化方法: 将动词字尾的假名「る」去掉，加上「ない」

 

例：

食(た)べる　→　食(た)べない  (不吃)　

見(み)る　→　見(み)ない  (不看)

 

開(あ)ける　→　開(あ)けない  (不开)

寝(ね)る　→　寝(ね)ない  (不睡)

始(はじ)める　→　始(はじ)めない  (不开始)

忘(わす)れる　→　忘(わす)れない  (不忘记)

 

不规则动词

 

する　→　しない　　(不做)

来(く)る　→　来(こ)ない    (不来)

 

 

动词过去否定

 

 

学会「动词ない形」后，「动词过去否定」的变化方式就显得相当容易了。

 

变化方法: 「动词ない形」－「い」＋「かった」

 

 

第一类动词

 

行(い)く　→　行(い)かない　→　行(い)かなかった　(之前没有去)　

読(よ)む　→　読(よ)まない　→　読(よ)まなかった  (之前没有读)

買(か)う　→　買(か)わない　→　買(か)わなかった  (之前没有买)

 

待(ま)つ　→　待(ま)たない　→　待(ま)たなかった　(之前没有等)　

帰(かえ)る　→　帰(かえ)らない　→　帰(かえ)らなかった  (之前没有回家)

売(う)る　→　売(う)らない　→　 売(う)らなかった  (之前没有卖)

 

 

第二类动词

 

食(た)べる　→　食(た)べない 　→　食(た)べなかった  (之前没有吃)　

見(み)る　  →  見(み)ない　   →　見(み)なかった  (之前没有看)

開(あ)ける　→　開(あ)けない　→　開(あ)けなかった  (之前没有开)

寝(ね)る　  →  寝(ね)ない　   →　寝(ね)なかった  (之前没有睡)

始(はじ)める　→　始(はじ)めない　→　始(はじ)めなかった  (之前没有开始)

忘(わす)れる　→　忘(わす)れない　 →　忘(わす)れなかった  (之前没有忘记)

 

 

不规则动词

 

する　→　しない　→　しなかった　　(之前没有做)

来(く)る　→　来(こ)ない　→　来(こ)なかった　　(之前没有来)

 

～ないでください

 

我们介绍以「动词ない形」为基础的常用句型: 「～ないでください」。

 

 

我们先前学过「动词て形＋ください」为「请…」的意思，同样地，我们也可以「动词ない形＋で＋ください」，中文就是「请不要…」的意思。

 

基本句型: 「动词ない形」＋「で」＋「ください」

           ( 请不要… )

 

例：

 

待(ま)つ　  →　待(ま)たない　→　待(ま)たないでください  (请不要等)

書(か)く　  →　書(か)かない　→　書(か)かないでください  (请不要写)

見(み)る　  →　見(み)ない　  →　見(み)ないでください    (请不要看)

食(た)べる　→　食(た)べない　→　食(た)べないでください  (请不要吃)

 

する　  →　しない　  →　しないでください    (请不要做)

来(く)る　  →　来(こ)ない　  →　来ないでください    (请不要来)
