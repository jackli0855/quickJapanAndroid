单元33    列挙の言い方

            (列举的说法)

 

学习目标

 ①　学习表示列举的句型「～たり」

②　学习表示列举的句型「～とか」「～やら」

③　学习表示列举的句型「～し」　

 

我们在很前面很前面的单元中，首次介绍过「と」「か」「や?など」这些助词，各位还记得吗? 这些助词的功能是「列举事物」，意思就是将许多名词一起接在一句话当中，例如：「猫と犬と兎」「定食かうどんかそば」「自転车やバイクや车など」等等用法。

 

但是先前介绍的这些助词，功能有限，只能连接名词，而无法连接形容词、动词、句子。本单元要介绍的，同样是用于「列举事物」的句型用法，只是更为进阶、更为生活化，包括「たり」「とか」「やら」「し」四项文法句型，可以广泛连接名词、形容词、动词、甚至是连接整个句子。

 

其中「たり」「とか」「し」三项句型用法较相似，需特别注意，以下也将介绍每个句型的使用方法和使用时机。由于在会话中出现频率相当高，因此请务必确切了解、熟练本单元介绍的文法句型。

 

 

 

たり

 

「たり」是日文中最常用于表示「列举」的句型之一，和先前学过的「か」「や、など」不同的是，「たり」的功能更为强大，可以接续动词或整个句子。列举的意思，就是从发生许多动作或事情中，随便抽取一二项出来讲，例如「我周日有看电视、打扫房间、出去玩」，周日做了许多事，但是来不及全部讲出来，因此举了以上三件事来代表，这就是列举用法。

 

另外，「たり」的使用范围很广，无论是口语沟通或者文章写作都可以使用，也就是说，「たり」既是口头用语，也是书面用语。

 

「たり」为「动词た形」加上「り」，可以将复数的「たり」连在一起用，但是最后记得一定要加上「する」，这是日语说话的习惯，加上去整句才算完整，见下列说明。

 

基本用法: 「动词た形」＋り、「动词た形」＋り＋する／します

 

例：

 

私(わたし)は日曜日(にちようび)にテレビ(　)を見(み)たり、部屋(へや)を掃除(そうじ)したり、遊(あそ)んだりしました。

( 我在星期日时一下子看电视、一下子打扫房间、一下子到处玩。)

 

彼(かれ)は行(い)ったり来(き)たりして、とても焦(あせ)っています。

( 他走来走去，非常地焦急。 )

 

林(はやし)さんが授業中(じゅぎょうちゅう)に寝(ね)たり、話(はな)したり、お菓子(かし)を食(た)べたりしているから、先生(せんせい)は怒(おこ)っています。

( 林同学在上课时一下子睡觉、一下子讲话、一下子吃饭，因此老师生气了。 )

 

 

弟(おとうと)は溜(た)め息(いき)をついたり、頭(あたま)を掻(か)いたりして、とても悩(なや)んでいます。

( 弟弟一下子叹气、一下子抓头、非常烦恼。)

 

母(はは)は今日(きょう)スーパ(　)ーに行(い)ったり、郵便局(ゆうびんきょく)に行(い)ったり、電気屋(でんきや)に行(い)ったりして、かなり忙(いそが)しかったです。

( 母亲今天一下子去超市、一下子去邮局、)

 

妹(いもうと)は一日中(いちにちじゅう)イ(　)ンターネットをしたり、チャットしたり、オンラインゲームをしたりして、家(うち)でぶらぶらしていました。

( 妹妹一下子上网、一下子闲聊、一下子打在线游戏，一整天都在家里闲

晃。)

 

 

とか

 

「とか」同样也具有列举的意思，从许多动作或事情中，选出几项代表性的事物进行介绍，功能大致上和「たり」相同，不过「とか」属于较通俗的用法，因此只能使用在口语会话中，在文章或正式场合中不能使用。

 

不过也因为「とか」这个用法非常口语、非常通俗 ，因此，在「とか」之前可以接续任何词种，包括名词、形容词、动词等等，而且不用多做变化，直接加上去就可以了。在一句话中也可以连用好几个「とか」。

 

另外，「とか」不能当作句子的句尾，在「とか」之后，必须加上另一个句子，感觉会较自然。

 

基本用法: 名词／形容词／动词＋とか、～。

 

 

 

 

例：

 

バナナとか、リンゴとか、レモンとか、好(す)きな果物(くだもの)を選(えら)んでください。

( 香蕉啊、苹果啊、柠檬啊什么的，请选你喜欢的水果。)

 

医者(いしゃ)とか、弁護士(べんごし)とか、企業家(きぎょうか)とか、あなたは将来何(しょうらいなに)になりたい？

( 医生啊、律师啊、企业家啊什么的，你将来想当什么? )

 

中国(ちゅうごく)に行(い)くとか、日本(にほん)に行(い)くとか、今年(ことし)は沢山(たくさん)の国(くに)を旅行(りょこう)したいです。

( 中国啊、日本啊什么的，今年我想去许多国家。)

 

電車(でんしゃ)に乗(の)るとか、飛行機(ひこうき)に乗(の)るとか、台北(たいぺい)から高雄(たかお)まで行(い)く方法(ほうほう)はいろいろあるでしょう。

( 坐电车啊、坐飞机啊什么的，从台北到高雄有许多方法。)

 

写真(しゃしん)を撮(と)ったとか、おみやげを買(か)ったとか、美味しい物(もの)を食(た)べたとか、今回(こんかい)の海外旅行(かいがいりょこう)がとても楽(たの)しかったです。

( 拍照啊、买纪念品啊、吃美食啊什么的，这次的海外旅行非常愉快。)

 

 

やら

 

「やら」这个用法，简单来说，就是「とか」的正式版本。「やら」的功能和文法用法和「とか」大致相同，区别在于，「とか」为口头用语，「やら」是书面用语，「とか」用于口语会话时，而「やら」则用于文章或正式场合时。如果将原本使用「とか」的句子以「やら」替换，那么感觉起来就会像是写作文章、或是在正式场合和人讲话的感觉。

 

 

 

另外，「やら」之前的动词只能使用动词原形，这点也必须注意。

 

基本用法: 名词／形容词／动词＋やら

 

例：

 

バナナやら、リンゴやら、レモンやら、好(す)きな果物(くだもの)を選(えら)んでください。

( 香蕉、苹果、柠檬之类的，请选你喜欢的水果。)

 

医者(いしゃ)やら、弁護士(べんごし)やら、企業家(きぎょうか)やら、あなたは将来何(しょうらいなに)になりたい？

( 医生、律师、企业家之类的，你将来想成为什么? )

 

中国(ちゅうごく)に行(い)くやら、日本(にほん)に行(い)くやら、今年(ことし)は沢山(たくさん)の国(くに)を旅行(りょこう)したいです。

( 中国、日本之类的，今天想去许多国家旅行。)

 

電車(でんしゃ)に乗(の)るやら、飛行機(ひこうき)に乗(の)るやら、台北(たいぺい)から高雄(たかお)まで行(い)く方法(ほうほう)はいろいろあるでしょう。

( 电车、飞机之类的，从台北到高雄有很多方法。)

 

写真(しゃしん)を撮(と)るやら、おみやげを買(か)うやら、美味しい物(もの)を食(た)べるやら、今回(こんかい)の海外旅行(かいがいりょこう)でしたいことがいっぱいあります。

( 拍照、买纪念品、吃美食之类的，我想在这次的国外旅行中做许多事。)

 

 

 

 

 

 

 

し

 

「し」同样具有列举的功能，不过和前面几项用法最不同的地方，在于「し」不仅只是描述事情动作，而且还带有强烈表达自己主张的感觉。

 

因此，「し」多用于表达自己的意见、主张和不满等等和自己有关的事情，而较少用来单纯列举事物。多和助词「も(也)」「から(因此)」连用，因为这二个助词也有加强语气的意思。

 

例：

 

×　バナナし、リンゴし、レモンし、好きな果物を选んでください。

○　バナナも食べたし、リンゴも食べたし、レモンも食べたし、もう何も食べなくてもいいでしょう。

 

基本用法: 名词／な形容词＋だし

い形容词／动词原形＋し

 

例：

 

彼(かれ)は日本語(にほんご)の先生(せんせい)だし、日本語通訳(にほんごつうやく)だし、この仕事(しごと)をうまくやってくれるでしょう。

( 他既是日文老师、又是日文口译，想必一定能够顺利做好这个工作吧。)

 

この公園(こうえん)はきれいだし、静(しず)かだし、今度(こんど)からここで勉強(べんきょう)しましょうよ。

( 这座公园既漂亮、又安静，下次就来这里念书吧。 )

 

この店(みせ)の料理(りょうり)は美味しいし、安(やす)いし、かなり量(りょう)があるし、皆(みな)さんにオススメです。

( 这家店的料理既美味、便宜、量又多，推荐给大家。 )

 

宿題(しゅくだい)もしたし、予習(よしゅう)もしたし、目覚(めざ)まし時計(どけい)もかけたから、そろそろ寝(ね)ましょう。

( 作业写了、预习了、也设定闹钟了，差不多该睡了。 )

 

日本(にほん)にも行(い)ったし、アメリカとヨーロッパにも行(い)ったから、もう満足(まんぞく)しました。

( 既去过了日本、也去过了美国和欧洲，我已经心满意足了。 )

 

今日(きょう)は休(やす)みだし、親(おや)がいないし、コンビニもまだやっているから、友達(ともだち)と家(うち)でホームパーティーを開(ひら)きましょう。

( 今天放假、爸妈也不在、便利商店也还在营业，因此和朋友一起在家开

派对吧。 )

 

謝(あやま)ったし、お金(かね)も弁償(べんしょう)したし、反省書(はんせいしょ)も書(か)いたから、今回(こんかい)は許(ゆる)してあげる。

( 既然都道歉了、也赔钱了、也写下了反省书，因此这次就原谅你。)
