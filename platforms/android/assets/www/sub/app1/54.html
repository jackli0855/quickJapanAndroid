形容词名词化.动词化用法

学习目标

① 了解「形容词＋さ」的形容词名词化用法

② 了解「形容词＋み」的形容词名词化用法　
③ 了解「形容词＋がる」的形容词动词化用法

 

本单元的重点为「形容词的名词化和动词化」。

「形容词」这个词类，在日文当中(在其他语言中也相同)，是专门用来修饰名词，表示「当事者心里的感觉」，例如「湖好深!」「今天好快乐!」，「好冷」和「快乐」是形容词，表示「说话者当时心里的感觉」，并且用来形容「天气」「今天」等名词。

 

例：湖(みずうみ)は深(ふか)い。(湖水很深。)

今日(きょう)は楽(たの)しい。(今天很开心。)

 

 

但是，「深い」「楽しい」充其量只是形容词，相当于中文的「深的」「快乐的」，那么若是我们想说成名词的形式，例如「湖水的深度」「今天的乐趣」中的「深度」「乐趣」，那么该怎么说呢?  中文字汇众多，只要将形容词换成对应的名词就可以了，但是在日文当中，可没有那么多名词可以替身，而是将原本的形容词「深い」「楽しい」的语尾加以变化，修改成名词的形式。

 

形容词　→　名词

「深的」→「深度」    「快乐的」→「乐趣」

「深い」→「深さ」    「楽しい」→「楽しみ」

 

我们可以将形容词语尾的「い」去掉，加上「さ」或「み」，达到名词化的效果。「さ」或「み」的差别如下，后面我们会详细解释。

 

「さ」：表示程度，具体可以计算。

「み」：表示状态，抽象不可计算。

 

另外，我们也可以使用「がる」将形容词变化为动词，原本的形容词和之后的动词，差别在于 :

形容词： 形容心理的状态

形容词＋がる： 将心理状态表露于外表的样子。

以形容词「面白い」为例，

 

面白い   : 心里觉得很有趣。

面白がる:  外表表现出觉得很有趣的样子。

 

各位看到这里，可能还是觉得概念模模糊糊的，请参照接下来的详细用法及例句说明。

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

形容词＋さ?み

 

日文中的形容词，包含「い形容词」和「な形容词」，经由后方接续假名「さ」「み」，能够将原本的形容词名词化，创造出新的名词。

 

基本用法 : い形容词　－「い」＋「さ」

少部分い形容词－「い」＋「み」

　　　　　 な形容词　＋「さ」

 

例:

高(たか)い　→　高(たか)さ

長(なが)い　→　長(なが)さ

広(ひろ)い　→　広(ひろ)さ

太(ふと)い　→　太(ふと)さ

楽(たの)しい　→　楽(たの)しさ?楽(たの)しみ

悲(かな)しい　→　悲(かな)しさ?悲(かな)しみ

活発(かっぱつ)　→　活発(かっぱつ)さ

元気(げんき)　→　元気(げんき)さ

静(しず)か　→　静(しず)かさ

便利(べんり)　→　便利(べんり)さ

大変(たいへん)　→　大変(たいへん)さ

那么，「形容词＋さ」和「形容词＋み」，有什么地方不一样呢?

 

「さ」：表示程度，具体可以计算。

「み」：表示状态，抽象不可计算。

 

 

举例来说:

楽しさ:  表示快乐的程度，很快乐、普通快乐、不快乐?

楽しみ:  表示快乐的状态本身，可以说成「乐趣」。

 

深さ: 表示深的程度，100公尺深? 500公尺深? 1000公尺深?　

深み: 表示深的状态本身，像是「这本书很有深度」的「深度」。

 

悲しさ：表示悲伤的程度。

悲しみ：表示悲伤的状态本身，中文意为「悲伤、悲哀」。

 

例：

出発(しゅっぱつ)する前(まえ)の情報収集(じょうほうしゅうしゅう)によって旅行(りょこう)の楽(たの)しさが決(けっ)定(てい)される。

（出发前的信息收集工作，决定了旅行快乐的程度。）

台湾旅行(たいわんりょこう)の楽(たの)しみはいろんな屋(や)台(たい)料(りょう)理(り)が食(た)べられることだ。

（台湾旅行的乐趣，在于可以吃到各式各样的摊贩料理。）

湖(みずうみ)の深(ふか)さを測(はか)るには、大変精密(たいへんせいみつ)な機(き)械(かい)が必要(ひつよう)です。

（测量湖水的深度，需要非常精密的机械。）

台(たい)湾(わん)阿(あ)里(り)山(ざん)で採(と)れた高山茶(こうざんちゃ)は、深(ふか)みのあるお茶(ちゃ)です。

（台湾阿里山产的高山茶，味道十分具有深度。）

大(だい)学(がく)一(いっ)斉(せい)試(し)験(けん)から落(お)ちた悲(かな)しさから、やっと回復(かいふく)してきた。

（从大学联考落榜的强烈悲伤中，终于慢慢回复了。）

悲(かな)しみを力(ちから)に変(か)える方法(ほうほう)があれば、教(おし)えてください。

（如果有将悲伤变成力量的方法，请告诉我。）

 

 

另外，形容词名词化有一项规则，就是只有「程度较大」的形容词，才能使用「さ」「み」来名词化。以中文来思考就会很容易理解:

 

○高度  ×矮度　　→　　○高(たか)さ　×低(ひく)さ

○长度  ×短度　　→　　○長(なが)さ　×短(みじか)さ

○深度  ×浅度　　→　　○深(ふか)さ　×浅(あさ)さ

○重量  ×轻量　　→　　○重(おも)さ　×軽(かる)さ

 

 

当然，无论是い形容词或な形容词，都可以经由「さ」來名词化，少部分也可以经由「み」来名词化。

 

更多例子如下:

強(つよ)い　　　→　　強(つよ)さ?強(つよ)み  (强度、强项)

高(たか)い　　　→　　高(たか)さ  (高度)

長(なが)い　　　→　　長(なが)さ  (长度)

広(ひろ)い　　　→　　広(ひろ)さ  (宽度)

濃(こ)い　　　→　　濃(こ)さ   (浓度)

速(はや)い　　　→　　速(はや)さ  (速度)

大(おお)きい　　→　　大(おお)きさ  (大小)

重(おも)い　　　→　　重(おも)さ    (重量)

深(ふか)い　　　→　　深(ふか)さ?深(ふか)み   (深浅、深度)

熱(あつ)い　　　→　　熱(あつ)さ    (温度)

美(うつく)しい　→　　美(うつく)しさ  (美丽程度)

寒(さむ)い　　　→　　寒(さむ)さ     (寒冷程度)

きれい　　→　　きれいさ  (漂亮程度)

穏(おだ)やか　　→　　穏(おだ)やかさ  (稳定程度)

活発(かっぱつ)　　　→　　活発(かっぱつ)さ   (活泼程度)

元気(げんき)　　　→　 　元気(げんき)さ    (活力程度)

静(しず)か    　→   　静(しず)かさ   (安静程度)

便利(べんり)　   →    　便利(べんり)さ  (便利程度)

簡単(かんたん)　  →    　簡単(かんたん)さ  (简单程度)

大変(たいへん)　  →    　大変(たいへん)さ  (累人程度)

いい　   →     　よさ    (好坏)

 

 

形容词动词化

 

我们可以使用「がる」，将形容词转化为动词，将形容词所表示的内在意思和状态，具体以动词表示于外在的行为和态度上。

 

变化方法: い形容词－「い」＋「がる」

　　　　　な形容词＋「がる」

 

形容词： 形容心理的状态

形容词＋がる： 将心理状态表露于外表的样子。

 

例：　面白い   : 心里觉得很有趣。

面白がる:  外表表现出觉得很有趣的样子。

 

同时，因为形容词是「用以形容个人的感觉」，因此只能用在自己身上，不能用在别人身上。如果想要表示他人对某事物的感觉，就可以用「形容词＋がる」的动词形式。

面白(おもしろ)い : 只能用于第一人称。

面白(おもしろ)がる:  可以用于第二三人称。

 

例：

○    私(わたし)は言(こと)葉(ば)の研究(けんきゅう)が面白(おもしろ)いと思(おも)う。

    （我觉得研究语言很有趣。）

×　彼(かれ)は言(こと)葉(ば)の研究(けんきゅう)が面白(おもしろ)いと思(おも)う。

○    彼(かれ)は言(こと)葉(ば)の研究(けんきゅう)を面白(おもしろ)がる。

    （他看起来觉得研究语言很有趣。）

 

寂(さび)しい   : 只能用于第一人称。

寂(さび)しがる:  可以用于第二三人称。

 

例：

○    私(わたし)は寂(さび)しいと思(おも)う。

   （我觉得很寂寞。）

×　彼(かれ)は寂(さび)しいと思(おも)う。

○　彼(かれ)は寂(さび)しがる（寂(さび)しがっている）。

   （他看起来很寂寞、他怕寂寞。）

其他例子如下:

 

強い  : 很强、很厉害。 

強がる: 外表装出很强的样子。逞强

 

寒い：  很冷。

寒がる：外表表现出很冷的样子。怕冷。

寒がり：怕冷的人。

 

其他形容词动词化的例子如下:

 

い形容詞＋がる：

 

寒(さむ)い　　→　　寒(さむ)がる     (感到寒冷的样子)

恐(こわ)い　　→　　恐(こわ)がる     (感到恐惧的样子)

面白(おもしろ)い　→　　面白(おもしろ)がる  (感到有趣的样子)

痛(いた)い　　→　　痛(いた)がる     (感到痛苦的样子)

嬉(うれ)しい　　→　　嬉(うれ)しがる (感到开心的样子)

かゆい　　→　　かゆがる  (感到很痒的样子)

かわいい　　→　かわいがる (觉得很可爱的样子、爱护)

 

な形容动词＋がる：

 

重宝(ちょうほう)　　→　　重宝(ちょうほう)がる  (感到珍惜的样子)

いや　　  →　　いやがる    (感到讨厌的样子)

退屈(たいくつ)　　 →　　退屈(たいくつ)がる    (感到无聊的样子)

面倒(めんどう)　   →  　面倒(めんどう)がる    (感到麻烦的样子)
