 学习目标

①了解何谓「い形容词」及「な形容词」。

②了解如何区别「い形容词」及「な形容词」。

③形容词的基本用法。

④形容词的肯定及否定用法。

 

我们这回要介绍的是，日文中的形容词。如同我们在先前的讲义中提到的，形容词的功能是用来形容名词，如:「漂亮的花」「干净的房间」，其中「花」和「房间」是名词，「漂亮的」和「干净的」即是形容词。形容词的概念，对中文母语者来说，并不难理解，我们日常生活中无时无都会用到形容词，正确且广泛地使用形容词，会让你的话显得更为生动，同时更精准地传达自己的意思(例如比起说「这里有怪味道」，说「好臭!」更能快速且精准地表达自己想法)。因此，我们将形容词的学习放在初期阶段。

 

日文中的形容词，和中文略为不同，主要分为二大类: 「い形容词」及「な形容词」。关于名称的由来，之后会详细介绍。我们将依序学习如何区分这二种形容词，以及形容词的基本用法句型、形容词的否定形。

 

 

二大种类

 

日文形容词分为二类:い形容词和な形容词，区分方法很简单。

 

い形容词:以假名い结尾的形容词

 

例：長(なが)い(长的)。 短(みじか)い(短的)。 広(ひろ)い(宽的)。 狭(せま)い(窄的)。

深(ふか)い(深的)。 浅(あさ)い(浅的)。  おいしい(好吃的)。 まずい(难吃的)。

　　寂(さび)しい(寂寞的)。 忙(いそが)しい(忙碌的)。 楽(たの)しい(快乐的)。

暖(あたた)かい(温暖的)。

 

名称由来: 以い形容词来修饰名词时，

使用「い形容词＋名词」的方式。

 

例：長(なが)い傘(かさ) (很长的伞)。 短(みじか)い鉛(えん)筆(ぴつ) (很短的铅笔)。

広(ひろ)い部(へ)屋(や) (很宽的房间)。狭(せま)い部(へ)屋(や) (很窄的房间)。

深(ふか)い湖(みずうみ) (很深的湖)。 浅(あさ)い川(かわ) (很浅的河)。

おいしいデザート(很好吃的甜点)。まずい料(りょう)理(り) (很难吃的料理)。

寂(さび)しい人(ひと) (很寂寞的人)。 忙(いそが)しい会(かい)社(しゃ)員(いん) (很忙碌的上班族)。

楽(たの)しい一(いち)日(にち) (很快乐的一天)。暖(あたた)かい手(て) (很温暖的手)。

 

 

 

 

な形容词:  不以假名い结尾的形容词

 

例：静(しず)か (安静的)。 親(しん)切(せつ) (亲切的)。 賑(にぎ)やか (热闹的)。

便(べん)利(り) (方便的)。元(げん)気(き) (有精神的)。 簡(かん)単(たん)(简单的)。

上(じょう)手(ず) (拿手的)。 下(へ)手(た) (不拿手的)。

 

名称由来: 以な形容词来修饰名词时，

使用「な形容词＋な＋名词」的方式。

 

例：静(しず)かな教(きょう)室(しつ) (安静的教室)。 親(しん)切(せつ)な先(せん)生(せい) (亲切的老师)。

賑(にぎ)やかなスーパー (热闹的超市)。 便(べん)利(り)な所(ところ) (方便的地方)。

元(げん)気(き)な子(こ)供(ども) (朝气十足的小孩)。 簡(かん)単(たん)な質(しつ)問(もん) (简单的问题)。

上(じょう)手(ず)なコ(こ)ッ(っ)ク(く) (厉害的厨师)。下(へた)手な選(せん)手(しゅ) (憋脚的选手) 。

 

另外，有些形容词经常会让人产生混乱。

 

例：有名（ゆうめい；有名的）?綺麗（きれい；漂亮的）

嫌い（きらい；讨厌的）

 

前面二个形容词，虽然写成假名时是以「い」结尾，但是由于写成汉字时，后面并不会出现「い」，因此将它们归类在「な形容词」。最后的「嫌い」算是特例，虽然是「い」结尾，但仍属于「な形容词」。

 

例：有(ゆう)名(めい)な歌(か)手(しゅ) (有名的歌手)。きれいな公(こう)園(えん)(干净漂亮的公园)。

嫌(きら)いな店(てん)員(いん) (讨人厌的店员)。

 

形容词的肯定句型

 

以形容词来修饰名词时，除了使用「形容词＋名词」的形式外，我们也可以运用「～は～です」这个基本句型。二种都可以，意思上没有区分。

 

肯定句型: (名词)　は　(い形容词/な形容词)　です。

 

例：長(なが)い傘(かさ)　→　傘(かさ)は長(なが)いです。

短(みじか)い鉛(えん)筆(ぴつ)　→　鉛(えん)筆(ぴつ)は短(みじか)いです。

おいしいデザート　→　デザートはおいしいです。

静(しず)かな教(きょう)室(しつ)　→　教(きょう)室(しつ)は静(しず)かです。

親(しん)切(せつ)な先(せん)生(せい)　→　先(せん)生(せい)は親(しん)切(せつ)です。

賑(にぎ)やかなスーパー　→　スーパーは賑(にぎ)やかです。

 

形容词的否定句型

 

形容词的否定用法，各自不同。

 

い形容词：将字尾的「い」改成「く」后，加上「ありません」。

 

例：(伞不长)、  (铅笔不短)、  (甜点不好吃)

長(なが)い傘(かさ)　→　傘(かさ)は長(なが)いです →　傘(かさ)は長(なが)くありません。

短(みじか)い鉛(えん)筆(ぴつ)　→　鉛(えん)筆(ぴつ)は短(みじか)いです　→鉛(えん)筆(ぴつ)は短(みじか)くありません。

おいしいデザート　→　デザートはおいしいです　→

デザートはおいしくありません。

 

 

な形容词：直接加上「ではありません」。

 

例：(教室不安静)、 (老师不亲切)、 (超市不热闹)

静(しず)かな教(きょう)室(しつ)→教(きょう)室(しつ)は静(しず)かです→教(きょう)室(しつ)は静(しず)かではありません。

親切(しんせつ)な先生(せんせい)→先生(せんせい)は親切(しんせつ)です→先生(せんせい)は親切(しんせつ)ではありません。

賑(にぎ)やかなスーパー　→　スーパーは賑(にぎ)やかです。　→

 

スーパーは賑(にぎ)やかではありません。
