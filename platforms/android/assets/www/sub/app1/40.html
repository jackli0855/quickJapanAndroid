单元40    日本语の限定用法

             （日文的限定用法）

学习目标

 ① 学习限定用法「だけ」

② 学习限定用法「しか」

 

本单元要介绍的是，日文中的「限定用法」，也就是用来「限定说话或动作对象」的文法，简单来说，就是中文当中的「只有…」「除了…都不…」句型，例如「家里只有弟弟一个人」「我除了零食之外什么都不吃」等等。

 

あなただけ见つめてる～～　　BY　スラムダンク

 

以上是动画「灌篮高手」片尾曲的第一句歌词，如果你是七年级生，应该立刻就能回忆起歌曲的旋律和片尾动画吧，没错，就是流川枫和樱木花道抢篮板的定格画面!

 

我们本单元的主题，其实就在这一句歌词中。「あなただけ见つめてる」中文为「我只凝视着你」，当中的「だけ」，是「只有…」的意思，这就是本单元将介绍的限定用法之一。中文当中的「只有」「只是」「只不过」等等用法，几乎都可以套用日文中的「だけ」文法来使用。就这点来说，日文方便多了。

 

日文中还有另一种常见的限定用法: 「しか」。「しか」的语气比「だけ」更为强烈，相当于中文「除了…之外，都不…」，例如「我只嫁给你（私はあなたとだけ结婚する）」「我除了你以外谁都不嫁（私はあなたとしか结婚しない）」，后句的语气就明显强烈多了。

 

在本单元中，我们将依序介绍「だけ」和「しか」的使用方法、以及如何区别二者的使用时机。

 

 

だけ

 

「だけ」相当于中文的「只有…」，用于限定动作的对象。

就文法性质来说，「だけ」经常接在名词之后。

 

基本用法:  名词　＋　だけ

( 只有… )

　名词　＋　だけは＋否定

( 只有… 不行)  加强语气

 

 

例：

 

私(わたし)の全財産(ぜんざいさん)は100元(げん)だけです。

（我全部的财产只有100元。）

あなただけは許(ゆる)せない！

（只有你我无法原谅!）

家(うち)にいる人(ひと)は弟(おとうと)だけです。

（只有弟弟一人在家。）

 

この提案(ていあん)に賛成(さんせい)した人(ひと)は二人(ふたり)だけです。

（赞成这则提案的只有二人。）

通訳(つうやく)の仕事(しごと)ができる人(ひと)は、クラスの中(なか)で王(おう)さんだけです。

（能够胜任口译工作的人，班上只有王同学而已。）

私(わたし)は刺身(さしみ)だけは食(た)べない。

（我只有生鱼片不吃。）

タバコだけは吸(す)わないでください。

（只有香烟别去碰。）

冷蔵庫(れいぞうこ)にある食材(しょくざい)はジャガイモだけだ。

（冰箱里的食材只有马铃薯而已。）

彼女(かのじょ)は見(み)た目(め)だけが綺麗(きれい)で、性格(せいかく)が悪(わる)い人(ひと)だ。

（她只有外表漂亮，其实个性不太好。）

 

 

しか

 

「しか」和「だけ」相似，也是表示对象的限定，相当于中文的「只有…」，但是较为不同的是，「しか」后面必须接否定形，语气比「だけ」更为强烈，听起来有「除了…之外什么都没有」的感觉。

 

基本用法：しか　＋　否定形

            ( 除了…都不… )  语气较强烈

 

例：

 

 

 

 

私(わたし)の全財産(ぜんざいさん)は100元(げん)しかないです。

（除了100元之外，没有其他财产了。）

家(うち)にいる人(ひと)は弟(おとうと)しかいないです。

（除了弟弟之外，没有人在家。）

この提案(ていあん)に賛成(さんせい)した人(ひと)は二人(ふたり)しかいないです。

（除了二个人之外，没有人赞成这个提案。）

私(わたし)は刺身(さしみ)しか食(た)べない。

（我除了生鱼片之外，什么都不吃。）

冷蔵庫(れいぞうこ)にある食材(しょくざい)はジャガイモしかない。

（除了马铃薯之外，冰箱里没有其他食材。）

私(わたし)は現在携帯(げんざいけいたい)と財布(さいふ)しか持(も)っていない。

（除了手机和钱包之外，我现在没有带其他东西。）

私(わたし)は日本語(にほんご)と英語(えいご)しかできない。

（我除了日语和英语外，什么都不会。）

木下(きのした)さんは朝味噌汁(あさみそしる)とご飯(はん)しか食(た)べない。

（木下先生除了味噌汤和白饭外，早上不吃其他东西。）

沢尻(さわじり)さんは高(たか)い服(ふく)しか買(か)わない。

（除了贵的衣服外，泽尻小姐不买其他衣服。）

私(わたし)は明(あか)るい人(ひと)としか友達(ともだち)にならない。

（除了开朗的人之外，我不和其他人做朋友。）

私(わたし)の進(すす)める道(みち)はこれしかない。

（除了这条路之外，我没有其他可以选择了。）

 

 

 

 

「だけ」と「しか」の区别

 

 

「だけ」：  「 只有… 」单纯限定某对象，语气较弱。

「しか」：　「除了…之外、都不…」藉由否定其他所有东西，来间接限

定某对象，语气较强。

 

例：

 

私(わたし)は寿司(すし)だけ食(た)べる。（我只吃寿司。）

私(わたし)は寿司(すし)しか食(た)べない。（除了寿司外什么都不吃。）

 

あなただけ見(み)つめている。（我只注视着妳。）

あなたしか見(み)つめていない。（除了妳之外，我谁都不看。）

 

但是，如果是要表现出「悲壮」或「下定决心」的语感的话，就只能用「しか」这个文法了。

 

例：

 

俺(おれ)にはあなたしかない！

（我除了妳之外什么都没有了!）

俺(おれ)は一生懸命頑張(いっしょうけんめいがんば)るしかない！

（我除了拼命努力，没有其他方法了!）

俺(おれ)は今前(いままえ)に進(すす)むしかない！

（我除了向前迈进之外，没有其他路可走了!）

俺(おれ)が持(も)っているものはこの命(いのち)しかない！

（我除了这条命之外，什么东西都没有了!）
