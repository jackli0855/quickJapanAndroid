单元42    自动词と他动词

              （日文的自动词和他动词）

学习目标

①　了解何谓「自动词」和「他动词」

②　了解「自动词」和「他动词」的使用方法

③　了解「自动词」和「他动词」的区别

 

这次要介绍的概念比较抽象，为日文中的「自动词」和「他动词」。日文的动词，依使用方法和意思，分为「自动词」和「他动词」二大类，事不宜迟，我们立刻进入学习内容吧!

自动词？他动词？

 

「自动词」和「他动词」到底是什么意思呢?

 

自动词 : 由自己进行该动作的动词。

        ＝「～が～になる」（～自然变成～）

 

他动词 : 藉由他人力量(外力) 进行该动作的动词。

　　　　＝「～を～にする」（～变成～）

 

例：

 

入(はい)る  →　自己进去某场所　  自动词

入(い)れる　→ 将…放进去        他动词

 

出(で)る　→  自己从某场所出去   自动词

出(だ)す→　将…拿出来           他动词

 

壊(こわ)れる　→ 东西坏掉了        自动词

壊(こわ)す →  将东西弄坏          他动词

 

消(き)える　→  东西自己消失了    自动词

消(け)す  →  将…消除掉          他动词

 

倒(たお)れる　→  某物自己倒下      自动词

倒(たお)す  → 将某物弄倒           他动词

 

 

如果以文法规则的角度解释，则如下所示，     

 

 

自动词 : 前面不需要加上表示动作对象的受词。

他动词 : 前面要加上表示动作对象的受词，意思才完整。

 

 

以中文为例子， 

 

 

①　「我跑」  「我走路」  「我生气」  「我哭」

 

这类动词，只要「名词＋动词」，意思就可以通。

 

②　但是像  我「看」   我「写」  我「参加」  我「用」

这类动词，如果只有「名词＋动词」的话，意思没办法清楚传达。

 

我看什么?   我写什么?  我参加什么?  我用什么?

 

少了后面的字，我们很难弄清楚那是什么意思。在动词后面要加上受词，意思才完整。

 

例如: 我看小说   我写字  我参加话剧  我用计算机

 

 

第一种动词，就是自动词；第二种动词，就是他动词。

 

自动词 : 前面不需要加上其他词汇。

他动词 : 前面要加上其他词汇，意思才完整。

 

如果你的英文能力不错的话，也可以用以下方式进行理解:

 

自动词 : 英文的不及物动词。意思就是动词前面不用有东西。

他动词 : 英文的及物动词。意思就是动词前面要有东西在才行。

 

将刚才的中文举例，翻译成日文的话，就会变成 :

 

自動詞(じどうし): 「私(わたし)は走(はし)る」、「私(わたし)は歩(ある)く」、「私(わたし)は怒(おこ)る」、「私(わたし)は泣(な)く」。

他動詞(たどうし)：「私(わたし)は小説(しょうせつ)を読(よ)む」「私(わたし)は字(じ)を書(か)く」、

「私(わたし)は芝居(しばい)に参加(さんか)する」、「私(わたし)はパソコンを使(つか)う」。

 

 

 

 

 

自他动词用法

 

 

基本用法 : 　を　＋　他动词

其他助词　＋　自动词

 

例：

 

私(わたし)は朝(あさ)起(お)きる。

（我早上起床。） 自动词

目(め)が覚(さ)める。

（眼睛睁开。） 自动词

朝食(ちょうしょく)を食(た)べる。

（吃早饭。） 他动词

靴(くつ)を履(は)く。

（穿鞋子。） 他动词

学校(がっこう)に行(い)く。

（去学校） 自动词

バスに乗(の)る。

（坐公交车。） 自动词

人(ひと)が多(おお)くて困(こま)る。

（人很多，很困扰。） 自动词

学校(がっこう)に到着(とうちゃく)する。

（到了学校。） 自动词

 

 

先生(せんせい)に会(あ)う。

（和老师见面。） 自动词

授業(じゅぎょう)を受(う)ける。

（上课。） 他动词

先生(せんせい)の話(はなし)を聞(き)く。

（聆听老师的话。） 他动词

宿題(しゅくだい)を提出(ていしゅつ)する。

（交作课。） 他动词

先生(せんせい)が先週(せんしゅう)のテストを配(くば)る。

（老师发回上周的考卷。） 他动词

クラスメイトがすごく心配(しんぱい)する。

（班上同学都非常担心。） 自动词

 

 

区别自他动词的方法

 

一般情况下，同一种意思的动词，可能会同时有「自动词」「他动词」二种型态，

 

例如:

転(ころ)げる／転(ころ)がる　　流(なが)れる／流(なが)す　　外(はず)れる／外(はず)す

照(て)る／照(て)らす　　動(うご)く／動(うご)かす　　回(まわ)る／回(まわ)す

 

这时，如何区分其意思，就变得十分关键。

 

区别自他动词的方法，有三种:

 

 

 

①　依据经验

②　依据前面的助词

③　依据固定的音韵变化

 

 

①　待日文学习到一定程度，累积到一定语汇量后，就会知道哪些是自动

词、哪些是他动词。不过这是废话，说了等于没说，必须得以时间慢

慢磨，没有任何技巧性可言。

 

 

②　以动词前面出现的助词来区分。

 

他动词 : 前方一定会出现助词「を」。

自动词 : 前方会出现其他助词(例：「に」「が」)。

 

这个方法容易理解，准确率也很高。但是有一个缺点，如果动词单独出现，前面没有助词可供参考的话，那么就无从判断起了。

 

 

③　自动词和他动词之间的音韵变化，具有一定的规律性，可以从其发音

的变化，来判断及记忆自动词及他动词的差别。
