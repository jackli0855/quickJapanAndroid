单元43    动词意志形と愿望の言い方

             （动词邀请形(意志形)和愿望的说法）

 

学习目标

①　学会表示邀请的「动词邀请形」

②  学会表示愿望的句型「动词邀请形＋と思う」

③　学会表示计划的句型「～つもり」

④　学会表示计划的句型「～予定」

本单元的学习重点有二个，分别是「动词邀请形」的用法，以及表示愿望的「～つもり」和表示计划「～予定」用法，这二种用法经常容易搞混。

 

首先是「动词邀请形」。我想应该没有人听过这个专有名词，因为「动词邀请形」这个名字是我自己取的，原名是「动词意志形」或「动词意向形」，功能是表示自己强烈的个人意志……不过这种叙述谁看得懂呢? 

 

简单来说 :

「动词邀请形」＝「～ましょう」的常体形态

 

「～ましょう」这个用法相信大家都很熟悉，相当于中文的「 一起….吧」，例如「ご饭を食べましょう」「日本に行きましょう」，常用来邀请别人一起做什么样的事情，但是这个「～ましょう」是由「～ます」变来的，属于所谓的「敬体」，可以用于老师或公司上司，但是却不能用于你的家人或宠物，因此我们需要「常体形态」的「～ましょう」，这个东西就是「动词邀请形」，意思完全相同，只是正式程度不同。

 

另外，我们也将介绍「动词邀请形＋と思う」句型，这个句型基本上和「～たい」相同，表达个人的希望和愿望，只是听起来比较委婉。

 

本单元的另一个重点，则是学习「计划」的说法，例如「我计划周末去玩」「明天预定和部长吃饭」等等语句。日文当中，则是使用「つもり」和「予定」这二项句型表示。

 

「つもり」: 表示仅止于自己心里计划，并未确定100%会执行。

            相当于中文的「我计划…」。

 

「予定(よてい)」:  表示已经确定会进行的计划。

            相当于中文的「我预定…」。

 

以下将进行更详细的解说。

 

动词邀请形

 

「动词邀请形」和「ましょう」的意思相同，都是表示「一起…吧」的意思，唯一不同的地方在于，「ましょう」属于敬体，只能用于师长上司等地位比自己高的人，「动词邀请形」属于常体，可以用于家人朋友同事等等上面。

 

例：

ご飯(はん)を食(た)べましょう。ご飯(はん)を食(た)べよう。

海外旅行(かいがいりょこう)に行(い)きましょう。海外旅行(かいがいりょこう)に行(い)こう。

 

 

动词变化方法

 

第一类动词

 

 

将动词字尾的「U」改成「O」 之后，加上「う」。

 

例：

 

貸(か)す　Kasu →　貸(か)そ  Kaso　→　貸(か)そう　　（借吧）

書(か)く　Kaku →　書(か)こ  Kako　→　書(か)こう     （写吧）

泳(およ)ぐ　Oyogu →　泳(およ)ご  Oyogo　→　泳(およ)ごう  （游泳吧）

言(い)う　Iu →　言(い)お  Io　→　言(い)おう          （说吧）

待(ま)つ　Matsu →　待(ま)と  mato　→　待(ま)とう     （等吧）

帰(かえ)る　Kaeru →　帰(かえ)ろ  Kaero　→　帰(かえ)ろう   （回家吧）

読(よ)む　Yomu →　読(よ)も  Yomo　→　読(よ)もう       （读书吧）

 

第二类动词

 

将动词字尾的「る」改成「よう」即可。

 

食(た)べる　→　食(た)べよう      （吃吧）

着(き)る　→　着(き)よう           （穿吧）

忘(わす)れる　→　忘(わす)れよう      （忘记吧）

寝(ね)る　→　寝(ね)よう           （睡吧）

開(あ)ける　→　開(あ)けよう       （打开吧）

見(み)る　→　見(み)よう　         （看吧）

 

 

不规则动词

 

する　→　しよう               （做…吧）

勉強(べんきょう)する　→　勉強(べんきょう)しよう   （读书吧）

来(く)る　→　来(こ)よう               （来吧）

 

 

动词邀请形＋と思う

 

「动词邀请形＋と思う」为「一起…」加上「我觉得」，直译是「我觉得要一起…」，其实就是「我想要…」的意思。

 

「动词邀请形＋と思う」和我们之前学过的「～たい」意思相似，都是用来表示自己的想法和愿望。不同的地方在于，「～たい」给人感觉非常的直接，「动词邀请形＋と思う」则比较委婉，适合用在正式场合中。

 

「～たい」:  我想要… 。语气较直接。

「动词邀请形＋と思う」：我打算要…。语气较间接。

 

例：

 

私は日本に行きたい。          （我想去日本。）

私は日本に行こうと思う。      （我打算去日本。）

私は新しい部屋を买いたい。    （我想买新房子。）

私は新しい部屋を买おうと思う。 （我打算买新房子。）

 

 

基本用法 :  动词邀请形＋と思う 。

( 我打算要… )

 

 

例：

 

あのことを忘(わす)れようと思(おも)う。

（打算忘记那件事。）

新(あたら)しい服(ふく)を着(き)ようと思(おも)う。

（打算穿新衣服。）

小説(しょうせつ)を書(か)こうと思(おも)う。

（打算写小说。）

ノートパソコンを買(か)おうと思(おも)う。

（打算买计算机。）

朝(あさ)まで仕事(しごと)しようと思(おも)う。

（打算工作到早上。）

道子(みちこ)さんと一緒(いっしょ)に来(こ)ようと思(おも)った。

（本来打算和道子小姐一起来。）

 

 

 

 

 

 

 

 

 

 

つもり 　予定（よてい）

 

「つもり」和「予定（よてい）」，都是用来表示「计划」的说法，描述今后想要进行的动作或事情。区别在于 :

 

「つもり」: 表示仅止于自己心里计划，并未确定100%会执行。

            相当于中文的「我计划…」。

　　　　　　由于是表示自己的心里想法，因此只能用于第一人称「我」。　

 

「予定(よてい)」:  表示已经确定会进行的计划。

            相当于中文的「我预定…」。

 

 

基本用法 : 动词原形?ない形　＋　つもり／予定

 

例：

 

私(わたし)は今晩(こんばん)カレーを食(た)べるつもり／予定(よてい)だ。

（我计划／预定今晚吃咖哩。）

 

私(わたし)は先生(せんせい)と食事(しょくじ)に行(い)くつもり／予定(よてい)だ。

（我计划／预定和老师去吃饭。）

 

徹夜(てつや)で仕事(しごと)をするつもり／予定(よてい)だ。

（我计划／预定熬夜工作。）

 

朝(あさ)６時(じ)に起(お)きるつもり／予定(よてい)だ。

（我计划／预定早上六点起来。）

 

 

 

X　父(ちち)は新(あたら)しいノートパソコンを購入(こうにゅう)するつもりだ。

○　父(ちち)は新(あたら)しいノートパソコンを购入する予定だ。

（爸爸预定购买新的笔电。） つもり只能用于第一人称

 

X　手嶋(てしま)さんは７月(がつ)に日本語検定(にほんごけんてい)に参加(さんか)するつもりだ。

○　手嶋(てしま)さんは７月(がつ)に日本語検定(にほんごけんてい)に参加(さんか)する予定だ。

（手嶋先生预定七月参加日语检定。） つもり只能用于第一人称

 

 

比较需要注意的是，「つもり」和「予定」的否定用法。

 

否定用法 : ①　动词否定形　＋　つもり／予定

( 我计划／预定不要……)

　　　　　 ②　动词常体　＋　つもり／予定　＋はない。

( 我没有……的计划／预定)

 

不同之处在于，第一句的否定语气较弱，第二句的否定语气较强烈。

请对照下列例句的中文翻译，相信并不难理解。

 

例：

 

私(わたし)はパーティーに行(い)かないつもりだ。

（我打算不去参加派对。）

私(わたし)はパーティーに行(い)かない予定(よてい)だ。

（我预计不去参加派对。）   语气较强烈

 

私(わたし)はパーティーに行(い)くつもりはない。

（我没有去参加派对的打算。）

私(わたし)はパーティーに行(い)く予定(よてい)はない。

（我没有预定要去参加派对。）  语气较强烈

 

徹夜(てつや)で仕事(しごと)をしないつもりだ。

（我打算不熬夜工作。）

徹夜(てつや)で仕事(しごと)をしない予定(よてい)だ。

（我预计不熬夜工作。）   语气较强烈

 

徹夜(てつや)で仕事(しごと)をするつもりはない。

（我没有熬夜工作的打算。）

徹夜(てつや)で仕事(しごと)をする予定(よてい)はない。

（我没有预定要熬夜工作。）   语气较强烈
