复合语用法

学习目标

① 学会表示过度的「～すぎる」复合语用法

② 学会表示难易度的「～やすい」「～にくい」复合语用法　
③ 学会表示开始和结束的「～はじめる」「～おわる」复合语用法

④ 学会表示持续的「～つづける」复合语用法　

本单元将介绍一个全新的概念，称作「复合语」。顾名思义，复合语就是「将复数的语汇合起来」的意思，一般在日文当中，有些时候光凭一个动词、或是一个形容词，很难表达我们想表达出来的东西，这时就会将二个不同的字合起来变成一个语汇，称作「复合语」。

 

例:

讀書　　　本を読む

開始讀書　本を読む？   (如何表达「开始」? 意思不明 )

讀完書　　本を読む？読んだ？ (如何表达「结束」的意思? )

 

这时，就可以在原本的动词后面，再加上表示「开始、结束」的动词，让意思更为完整。

 

「読む」＋「始める」→　読(よ)み始(はじ)める

「読む」＋「終わる」→　読(よ)み終(お)わる

 

如上述例句般，将二个字(例如:「読む」「始める」) 合起来当作一个字，意思表现也会更加完整。这就是复合语的功能: 传达更完整的意思。

各位可能没注意到，其实中文当中也有许多复合语存在。

例: 进来  进去  拿起来  做完

 

如果你询问任何一位学习中文的外国人，这些复合语都是非常难以理解和学习的部分，「拿就是拿啊，为什么还要加上『起』和『来』二个字?」常常会听到类似的抱怨声音。不过正因为中文也有复合语的概念，因此我们在理解日文中的复合语时，也会比较容易。

 

本单元将介绍六项日文会话中常见的复合语用法，分别为表示过度的「～すぎる」、学会表示难易度的「～やすい」「～にくい」、学会表示开始和结束的「～はじめる」「～おわる」、学会表示持续的「～つづける」。

～すぎる

 

「すぎる」的汉字写法为「過ぎる」，意思为「经过、渡过」，例如「長(なが)い年月(ねんげつ)が過(す)ぎた」「中旬(ちゅうじゅん)を過(す)ぎると桜(さくら)が咲(さ)く」等等，不过，如果「すぎる」接在其他字后面，就会变成「过于…、太…」的抽象意思。

 

例如「このクラスは静(しず)かすぎる（這個班級太安靜了。）」「隣(となり)の人(ひと)がうるさすぎる（隔壁的人太吵了。）」等等，相当于中文当中「过于、太…、…到不行」的意思。

 

基本用法: 　动词ます形－「ます」＋「すぎる」

　　　　　　い形容词－「い」＋「すぎる」

　　　　　　な形容词＋「すぎる」

 

例：

頑(がん)張(ば)る　　   →　　  頑(がん)張(ば)りすぎる　　（太努力了。）

見(み)る　　　   →　　  見(み)すぎる        （看太多。）

勉強(べんきょう)する　　→　　 勉強(べんきょう)しすぎる   （读太多书。）

 

おいしい　　　→　 　 おいしすぎる   （太好吃了。）

長(なが)い　　　　  →　　　長(なが)すぎる       （太长了。）

広(ひろ)い　　　　  →　　　広(ひろ)すぎる       （太宽大了。）

強(つよ)い　　　　→　　　強(つよ)すぎる      （太强了。）

近(ちか)い　　　　→　　　近(ちか)すぎる      （太近了。）

 

必死(ひっし)　　　　→　　　必死(ひっし)すぎる     （太拼命了。）

静(しず)か　　　　→　　　静(しず)かすぎる    （太安静了。）

有名(ゆうめい)　　　　→　　　有名(ゆうめい)すぎる   （太有名了。）

危(き)険(けん)　　　　→　　　危(き)険(けん)すぎる     （太危险了。）

簡単(かんたん)　　　→　　　　簡単(かんたん)すぎる   （太简单了。）

衝(しょう)撃(げき)　　→　　　　衝(しょう)撃(げき)すぎる   （太过冲击了。）

 

 

～やすい?～にくい

 

「やすい」「にくい」的汉字写法为「易い」「難い」，意思为「简单」和「困难」，如果「やすい」「にくい」接在其他字后面，就会变成「容易从事那件事」「很难从事那件事」等抽象意思。

 

例如「この小説は読みやすい（这本小说很容易阅读）」「教科書は読みにくい( 教科书很难阅读；很难理解)」等等，相当于中文当中「很容易…、很方便…」「不容易…、不方便…」的意思。

 

 

 

基本用法: 　动词ます形－「ます」＋「やすい／にくい」

　　　　　　

例：

食(た)べる　　→　　食(た)べやすい?食(た)べにくい   （方便吃、不方便吃）

作(つく)る　　　→　　作(つく)りやすい?作(つく)りにくい  （容易做、不容易做）

歩(ある)く　　　→　　歩(ある)きやすい?歩(ある)きにくい  （好走、不好走）

入(はい)る　　　→　　入(はい)りやすい?入(はい)りにくい  （好进去、不好进去）

書(か)く　　　→　　書(か)きやすい?書(か)きにくい    （好写、不好写）

見(み)る　　　→　　見(み)やすい?見(み)にくい        （容易看、不容易看）

発音(はつおん)する　→　　発音(はつおん)しやすい?発音(はつおん)しにくい

（容易发音、不容易发音）

使(つか)う　　　→　　使(つか)いやすい?使(つか)いにくい   （好用、不好用）

運(はこ)ぶ　　　→　　運(はこ)びやすい?運(はこ)びにくい   （好搬、不好搬）

覚(おぼ)える　　→　　覚(おぼ)えやすい?覚(おぼ)えにくい  

（容易记住、不容易记住）

忘(わす)れる　　→　　忘(わす)れやすい?忘(わす)れにくい  

（容易忘记、不容易忘记）

伝(つた)える　　→　　伝(つた)えやすい?伝(つた)えにくい 

（容易传达、不容易传达）

 

～はじめる／～おわる

 

「はじめる」「おわる」这二个动词，相信大家都已经很熟悉了吧，汉字写法为「始める」「終わる」，意思为「开始」和「结束」，例如「授業(じゅぎょう)を始(はじ)める」「授業(じゅぎょう)を終(お)わる（が終(お)わる）」等等。

 

如果「はじめる」「おわる」接在其他字后面，形成「复合语」，就会变成「开始做那件事」「做完那件事」等抽象意思。相当于中文的「开始…」「结束…」。

 

例如「この小説(しょうせつ)を読(よ)みはじめる（开始读这本小说）」「教(きょう)科(か)書(しょ)を読(よ)み終(お)わる( 读完教科书)」等等。

 

基本用法: 　动词ます形－「ます」＋「はじめる／おわる」

 

例：

食(た)べる　　→　食(た)べはじめる?食(た)べおわる    （开始吃、吃完）

飲(の)む　　  →　飲(の)みはじめる?飲(の)みおわる    （开始喝、喝完）

作(つく)る　　  →　作(つく)りはじめる?作(つく)りおわる   （开始做、做完）

習(なら)う　　  →　習(なら)いはじめる?習(なら)いおわる   （开始学、学完了）

使(つか)う　　  →　使(つか)いはじめる?使(つか)いおわる   （开始用、用完了）

書(か)く　　  →　書(か)きはじめる?書(か)きおわる    （开始写、写完了）

見(み)る　　  →　見(み)はじめる?見(み)おわる         （开始看、看完了）

 

する　　  →　しはじめる?しおわる         

（开始做某件事、做完某件事）

洗濯(せんたく)する　　→　洗濯(せんたく)しはじめる?洗濯(せんたく)しおわる

（开始清洗、清洗完了）

演奏(えんそう)する　　→　演奏(えんそう)しはじめる?演奏(えんそう)しおわる

（开始演奏、演奏完了）

 

 

～つづける

 

我们刚才介绍过「开始」和「结束」的复合语说法，有开始、有结束，就有「持续」。

 

「つづける」的汉字写法为「続ける」，意思为「持续、继续」，例如「仕事を続ける」「学業を続ける」等等，不过，如果「つづける」接在其他字后面，形成「复合语」，就会变成「持续做某件事」的抽象意思。

相当于中文当中「持续…」的意思。例如「映画を撮りつづける」「日本語を教えつづける」等等，

基本用法: 　动词ます形－「ます」＋「つづける」

　　　　　　

例：

昨(さく)夜(や)から雨(あめ)が降(ふ)り続(つづ)けている。

（从昨晚开始不停地下雨。）

毎日新聞(まいにちしんぶん)を読(よ)み続(つづ)けると、世(よ)の中(なか)の動(うご)きをより分(わ)かる。

（每天持续阅读报纸的话，就能比较了解世界的动向。）

大学(だいがく)を卒業(そつぎょう)するまで、ブログを書(か)き続(つづ)けると思(おも)う。

（打算到大学毕业为止，持续写网志。）

長(ちょう)時(じ)間(かん)にわたりパソコンの画(が)面(めん)を見(み)続(つづ)けるのは目(め)によくない。

（长时间持续盯着计算机画面的话，对眼睛不好。）

彼(かれ)は２４時(じ)間(かん)寝(ね)ずに仕(し)事(ごと)し続(つづ)けたという。

（听说他二十四小时不睡觉一直持续工作。）

諦(あきら)めないで、行動(こうどう)し続(つづ)けると、いつか夢(ゆめ)がかなうはずだ。

（不放弃、持续尝试的话，总有一天梦想应该会实现。）
