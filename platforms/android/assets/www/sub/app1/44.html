单元44    アドバイスと推测の言い方

           (建议和推测的说法)

学习目标

①  学习表示建议的「动词た形＋ほうがいい」句型

②　学习表示推测的「かもしれない」「～だろう」「だろうと思う」句型

 

本单元将介绍日文会话中常见的「建议」和「推测」的说法。

 

所谓「建议」，指的是针对他人的疑问提出自己的看法、意见，像是「你应该怎样怎样比较好喔! 」「我觉得应该要怎样怎样」之类的句子。日文会话中，则是习惯使用「动词た形＋ほうがいい」句型来表示「给予人的建议或评论」。「ほう」的汉字写做「方」，为「方面」的意思，因此「ほう が いい」直译中文就是「这一方面很好」，也就是「这么做比较好」这样的建议语气。

 

至于推测用法，在日文中有很多，这可能和日本的民族性格有关系，日本人认为太过主观、武断的说法会显得不客气、对听话者不礼貌、同时太过武断会有风险，因此很多时候，即使是已经非常肯定的事情，也会使用「推测语气」来讲。例如，即使降雨机率100%，气象主播仍会说成「明日は雨が降るでしょう(明天应该会下雨吧)」。

 

 

 

 

本单元中，将介绍三种基本的「推测用法」，分别为「かもしれない」「だろう」「だろうと思う」。「かもしれない」相当于中文的「或许、有可能」之意，「だろう」则和「でしょう」相同，用在表示不确定的情况。「だろうと思う」则是比「だろう」更为间接的讲法，「だろう(应该)」加上「と思う(我觉得)」，等于是具有双重推测的功能，语气显得更为委婉。

 

详细介绍请参照以下的说明。

 

～たほうがいい

 

「动词た形＋ほうがいい」是日文当中经常用来表示建议的句型用法，用于向对方表达自己的主张或是评论。「动词た形＋ほうがいい」由几个部分组成:

 

「动词た形」＋「ほう」＋「が」＋「いい」

   动作         方面      是       好的

 

相当于中文的「这样做比较好～」。不过必须注意的是，前面的动词要使用「动词た形」，听起来才会有「建议」的感觉。

 

基本用法 : 动词た形＋ほう　が　いい  だ／です

(  这样做比较好  )

 

例:

 

寝(ね)る前(まえ)に宿題(しゅくだい)をしたほうがいいですよ。

（睡觉前最好先写完作业喔。）

大学(だいがく)でいろんなサークルに入(はい)ってみたほうがいいですよ。

（在大学中最好多去参加看看不同的社团喔。）

あまりテレビを見(み)ないほうがいいですよ。

（最好别看太多电视喔。）

ファストフードより、健康志向(けんこうしこう)の食(た)べ物(もの)を食(た)べたほうがいいです。

（比起快餐，多吃点对身体有益的食物比较好喔。）

アルバイトを沢山(たくさん)するより、勉強(べんきょう)に専念(せんねん)したほうがいいです。

（比起打很多任务，最好专心念书比较好喔。）

部屋(へや)は一週間(いちしゅうかん)に一回(いっかい)ぐらい掃除(そうじ)したほうがいいですよ。

（最好一星期打扫一次房间喔。）

 

推测的说法

 

かもしれない

 

「かもしれない」为日文中典型表示推测的说法，相当于中文的「或许、应该」，用于句尾，使用范围很广泛，可以使用于大多数带有「或许、应该」语气的情况下。

 

基本用法: 动词原形　＋　かもしれない／かもしれません

　　　　　　( 或许、应该 )

 

例：

 

私(わたし)は明日(あした)の打ち上(あ)げパーティーに行(い)かないかもしれない。

（我或许会去明天的庆功宴。）

今回(こんかい)は内定(ないてい)が取(と)れるかもしれない。

（这次或许能顺利录取。）

速(はや)ければ、三日以内(みっかいない)に商品(しょうひん)を手に入(い)れるかもしれない。

（快的话，或许在三天之内能够收到商品。）

 

 

彼(かれ)は将来(しょうらい)もしかして総統(そうとう)になれるかもしれない。

（他将来或许能够成为总统。）

沢山(たくさん)レシートを集(あつ)めると、一等賞(いっとうしょう)に当(あ)たるかもしれない。

（收集很多发票的话，说不定就能中头奖。）

今日(きょう)は天気(てんき)いいから、海水浴場(かいすいよくじょう)に遊(あそ)びに行(い)く人(ひと)は多(おお)いかもしれない。

（今天天气很好，去海水浴场玩的人应该很多。）

 

 

だろう／でしょう

 

「だろう／でしょう」常用在「不是很确定」「不是很肯定」的情况之下，相较于肯定语气的「だ／です」，「だろう／でしょう」显得较为客观、委婉、有礼貌，因此有时候为了听起来不那么主观，也会使用「だろう／でしょう」。

 

例：

 

彼(かれ)は来(き)ます。彼(かれ)は来(く)るでしょう。

（他会来／他应该来会吧） 后者较客观

先生(せんせい)、大丈夫(だいじょうぶ)ですか。先生(せんせい)、大丈夫(だいじょうぶ)でしょうか。

（老师，没事吧?／老师，您没事吧? ） 后者较礼貌

明日(あした)、空(あ)いていますか。明日(あした)、空(あ)いているでしょうか。

（明天有空吗?／明天是否有空呢?） 后者较委婉

 

 

基本用法 : ( 用于句尾  )  だろう／でしょう。

(  …吧  )

 

 

例 :

 

この映画(えいが)を見(み)た人(ひと)は沢山(たくさん)いるだろう。

（看过这部电影的人应该很多吧。）

有名(ゆうめい)なコックさんだから、作(つく)る料理(りょうり)は美味しいだろう。

（因为是有名的厨师，所以做的菜应该很好吃吧。）

一番大事(いちばんだいじ)なものは何(なん)だろう。

（最重要的东西是什么呢?）

休日(きゅうじつ)の学校図書館(がっこうとしょかん)は普段(ふだん)より静(しず)かだろう。

（假日的学校图书馆应该比平时安静吧。）

もう少(すこ)しで家(うち)に帰(かえ)れるだろう。

（再过一下就能回家了吧。）

彼(かれ)は確(たし)かにそういうことを言(い)っただろう。

（他的确说了那样的事情吧。）

 

 

だろうと思う

 

「だろうと思う」由刚才介绍过的「だろう」和「と思う」组合而成。

 

「と思う」本身就有一点点推测的意思，

 

例:

明日学校に行く。明日学校に行くと思う。（我认为明天要去学校。）

この服を买う。この服を买うと思う。（我认为会买这件衣服。）

 

因此「だろう」加上「と思う」，等于是使用二种具有推测功能的字汇，有双重推测的感觉，比起单独使用「だろう」语气较为和缓、委婉且客气。

另外，由于「と思う」前面只能接动词常体，因此在这个句型中，只能使用「だろう」，而不能使用「でしょう」。

 

基本用法 : ～　だろう　と思う／思います。

( 我认为应该是… )

 

例：

 

彼(かれ)はもうすぐ帰(かえ)るだろうと思(おも)います。

（我认为应该马上能回家了。）

カレーに蜂蜜(はちみつ)を入(い)れたらうまいだろうと思(おも)います。

（我觉得咖哩中加入蜂蜜应该会很好吃。）

もっと安(やす)い値段(ねだん)で買(か)えるだろうと思(おも)います。

（我认为应该能用更便宜的价钱买到。）

社長(しゃちょう)は今中国(いまちゅうごく)を旅行(りょこう)しているだろうと思(おも)います。

（我认为社长现在应该在中国旅行。）

これから物価(ぶっか)が上(あ)がるだろうと思(おも)います。

（我觉得今后物价应该会上涨。）

雪国(ゆきぐに)の生活(せいかつ)は厳(きび)しいだろうと思(おも)います。

（我认为雪国的生活应该很辛苦。）

明日会議(あしたかいぎ)に出(で)なければならないだろうと思(おも)います。

（我认为明天应该去开会。）
