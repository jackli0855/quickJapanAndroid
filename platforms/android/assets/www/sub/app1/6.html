①了解助词「と?や?か?など」的用法

 

 

这回开始进入助词的范围。助词是什么？简单来说，助词就是单独不具有意思，只能接在名词、动词或其他词类后方的词，举例来说:「我和朋友去吃饭」这句话当中，「和」就是助词，你无法将「和」单独当作一句话来说，必须要配合「我」「朋友」才具有意思。我们学过的「の」也是助词。

 

助词虽然不具单独意思，但是就像空气一样，虽然没什么存在感，但是在会话时却扮演着很重要的角色，随着助词的不同，句子的意思也会改变，例如「私は刺身がいいです」是「我想吃生鱼片」，但是如果改成「私は刺身はいいです」，就变成了「我不要生鱼片」，意思完全不同。因此，在日文学习的过程中，助词的学习是相当重要的部分。

 

 

这次我们要介绍「と?や?か?など」四个助词，这些助词的使用频率非常之高，概念也很好理解，很快就能够上手。

 

と

 

「と」: 连接名词，表示并列。相当于中文的「和」。

 

基本用法: (名词)＋ と ＋(名词)

 

例：

私(わたし) と 妹(いもうと) と 弟(おとうと) (我和妹妹和弟弟)

犬(いぬ) と 猫(ねこ) (狗和猫)

テーブル と 椅(い)子(す) (桌子和椅子)

梅(うめ) と 桜(さくら) (梅花和樱花)

 

ハンバーガー と コーラ (汉堡和可乐)

 

 

や?など

 

「や」: 连接名词，表示从众多物品中选择几样当作例子。相当于中文的「像是…」。

「など」: 和「や」一起用，接在句子的最后面。相当于中文的「…等等」。

 

基本用法:  (名词) ＋ や ＋ (名词) ＋ など

 

例：和(わ)食(しょく)と言(い)えば、寿(す)司(し) や 味(み)噌(そ)汁(しる) や てんぷら など です。

    (说到和食，有寿司、味噌汤、天妇罗等等。)

　　ペットと言(い)えば、犬(いぬ) や 猫(ねこ) や 兎(うさぎ) や 鼠(ねずみ) など です。

(说到宠物，有狗、猫、兔子、老鼠等等。)

 

　　飲(の)み物(もの)と言(い)えば、コーラ や ジュース や お茶(ちゃ) など です。

    (说到饮料，有可乐、果汁、茶等等。)

 

 

か

 

「か」: 连接名词，表示从中择一。相当于中文的「或者」。

 

基本用法: (名词)＋ か ＋ (名词)

 

例：

①A：いつが暇(ひま)ですか。 (什么时候有空呢?)

B：土(ど)曜(よう)日(び) か 日(にち)曜(よう)日(び) です。(星期六或是星期日。)

②A：旅(りょ)行(こう)はいつですか。(旅行什么时候呢?)

B：明(あ)日(した) か あさって です。(明天或是后天。)

③A：何(なに)を食(た)べますか。(要吃什么呢?)

B：定(てい)食(しょく) か うどん か ラーメン です。(定食或乌龙面或拉面。)

④A：何(なに)を買(か)いますか。(要买什么呢?)

B：かばん か 服(ふく) か 化(け)粧(しょう)品(ひん)です。(包包或衣服或化妆品)

 

 

在会话中，同样的单字句型，使用不同助词的话，会有完全不同的意思喔！请参照以下举例。

 

会話例 ①：

 

A：何(なに)を食(た)べますか。(要吃什么呢?)

B：①定(てい)食(しょく) や うどん や ラーメン など ですね。

(像是定食、乌龙面、拉面等等的吧) 没有明确指定吃哪一样

②定(てい)食(しょく) か うどんか ラーメン ですね。

(定食、乌龙面、或是拉面吧) 明确表示从三样中挑选其中一样

③定(てい)食(しょく) と うどん と ラーメンです。

   (定食和乌龙面和拉面吧) 明确表示三样都要

 

会話例 ②：

 

A：テストは いつ ですか。(考试是什么时候呢?)

B：①明(あした)日(した) や 明(あ)後(さっ)日(て) や 来(らい)週(しゅう) など ですね。

(明天、后天、下周等等吧) 表示考试很多，不只明天、后天、下周。

②明(あ)日(した) か 明(あ)後(さっ)日(て) か 来(らい)週(しゅう) ですね。

 (明天、后天、或是下周吧) 表示考试是明天、后天、下周其中一天。

③明(あ)日(した) と 明(あ)後(さっ)日(て) と 来(らい)週(しゅう) ですね。

(明天和后天和下周吧) 表示考试是在明天、后天、以及下周，三次。
