形容詞的名詞．動詞化

.

學習目標

① 了解「い形容詞＋さ」的形容詞名詞化用法

② 了解「い形容詞＋み」的形容詞名詞化用法
③ 了解「形容詞＋がる」的形容易動詞化用法

本單元的重點為「形容詞的名詞化和動詞化」。

「形容詞」這個詞類，在日文當中(在其他語言中也相同)，是專門用來修飾名詞，表示「當事者心裡的感覺」，例如「湖好深!」「今天好快樂!」，「好冷」和「快樂」是形容詞，表示「說話者當時心裡的感覺」，並且用來形容「天氣」「今天」等名詞。

例：湖は深い。(湖水很深。)

今日は楽しい。(今天很開心。)

但是，「深い」「楽しい」充其量只是形容詞，相當於中文的「深的」「快樂的」，那麼若是我們想說成名詞的形式，例如「湖水的深度」「今天的樂趣」中的「深度」「樂趣」，那麼該怎麼說呢?  中文字彙眾多，只要將形容詞換成對應的名詞就可以了，但是在日文當中，可沒有那麼多名詞可以替身，而是將原本的形容詞「深い」「楽しい」的語尾加以變化，修改成名詞的形式。

形容詞　→　名詞

「深的」→「深度」    「快樂的」→「樂趣」

「深い」→「深さ」    「楽しい」→「楽しみ」

我們可以將形容詞語尾的「い」去掉，加上「さ」或「み」，達到名詞化的效果。「さ」或「み」的差別如下，後面我們會詳細解釋。

「さ」：表示程度，具體可以計算。

「み」：表示狀態，抽象不可計算。

另外，我們也可以使用「がる」將形容詞變化為動詞，原本的形容詞和之後的動詞，差別在於 :

形容詞： 形容心理的狀態

形容詞＋がる： 將心理狀態表露於外表的樣子。

以形容詞「面白い」為例，

面白い   : 心裡覺得很有趣。

面白がる:  外表表現出覺得很有趣的樣子。

各位看到這裡，可能還是覺得概念模模糊糊的，請參照接下來的詳細用法及例句說明。

形容詞＋さ・み

日文中的形容詞，包含「い形容詞」和「な形容詞」，經由後方接續假名「さ」「み」，能夠將原本的形容詞名詞化，創造出新的名詞。

基本用法 : い形容詞　－「い」＋「さ」

少部分い形容詞－「い」＋「み」

な形容詞　＋「さ」

例:

高(たか)い　→　高(たか)さ

長(なが)い　→　長(なが)さ

広(ひろ)い　→　広(ひろ)さ

太(ふと)い　→　太(ふと)さ

楽(たの)しい　→　楽(たの)しさ・楽(たの)しみ

悲(かな)しい　→　悲(かな)しさ・悲(かな)しみ

活発(かっぱつ)　→　活発(かっぱつ)さ

元気(げんき)　→　元気(げんき)さ

静(しず)か　→　静(しず)かさ

便利(べんり)　→　便利(べんり)さ

大変(たいへん)　→　大変(たいへん)さ

那麼，「形容詞＋さ」和「形容詞＋み」，有什麼地方不一樣呢?

「さ」：表示程度，具體可以計算。

「み」：表示狀態，抽象不可計算。

舉例來說:

楽しさ:  表示快樂的程度，很快樂、普通快樂、不快樂?

楽しみ:  表示快樂的状態本身，可以説成「樂趣」。

深さ: 表示深的程度，100公尺深? 500公尺深? 1000公尺深?

深み: 表示深的状態本身，像是「這本書很有深度」的「深度」。

悲しさ：表示悲傷的程度。

悲しみ：表示悲傷的状態本身，中文意為「悲傷、悲哀」。

例：

出発(しゅっぱつ)する前(まえ)の情報収集(じょうほうしゅうしゅう)によって旅行(りょこう)の楽(たの)しさが決定(けってい)する。

（出發前的資訊收集工作，決定了旅行快樂的程度。）

台湾旅行(たいわんりょこう)の楽(たの)しみはいろんな屋台料理(やたいりょうり)が食(た)べられることだ。

（台灣旅行的樂趣，在於可以吃到各式各樣的攤販料理。）

湖(みずうみ)の深(ふか)さを測(はか)るには、大変精密(たいへんせいみつ)が機械が必要です。

（測量湖水的深度，需要非常精密的機械。）

台湾阿里山(たいわんありざん)で採(と)れた高山茶(こうざんちゃ)は、深(ふか)みのあるお茶(ちゃ)です。

（台灣阿里山產的高山茶，味道十分具有深度。）

大学一斉試験(だいがくいっせいしけん)から落(お)ちた悲(かな)しさから、やっと回復(かいふく)してきた。

（從大學聯考落榜的強烈悲傷中，終於慢慢回復了。）

悲(かな)しみを力(ちから)に変(か)える方法(ほうほう)があれば、教(おし)えてください。

（如果有將悲傷變成力量的方法，請告訴我。）

另外，形容詞名詞化有一項規則，就是只有「程度較大」的形容詞，才能使用「さ」「み」來名詞化。以中文來思考就會很容易理解:

○高度  ×矮度　　→　　○高(たか)さ　×低(ひく)さ

○長度  ×短度　　→　　○長(なが)さ　×短(みじか)さ

○深度  ×淺度　　→　　○深(ふか)さ　×浅(あさ)さ

○重量  ×輕量　　→　　○重(おも)さ　×軽(かる)さ

當然，無論是い形容詞或な形容詞，都可以經由「さ」來名詞化，少部分也可以經由「み」來名詞化。

更多例子如下:

強(つよ)い　　　→　　強(つよ)さ・強(つよ)み  (強度、強項)

高(たか)い　　　→　　高(たか)さ  (高度)

長(なが)い　　　→　　長(なが)さ  (長度)

広(ひろ)い　　　→　　広(ひろ)さ  (寬度)

濃(こ)い　　　→　　濃(こ)さ   (濃度)

速(はや)い　　　→　　速(はや)さ  (速度)

大(おお)きい　　→　　大(おお)きさ  (大小)

重(おも)い　　　→　　重(おも)さ    (重量)

深(ふか)い　　　→　　深(ふか)さ・深(ふか)み   (深淺、深度)

熱(あつ)い　　　→　　熱(あつ)さ    (溫度)

美(うつく)しい　→　　美(うつく)しさ  (美麗程度)

寒(さむ)い　　　→　　寒(さむ)さ     (寒冷程度)

きれい　　→　　きれいさ  (漂亮程度)

穏(おだ)やか　　→　　穏(おだ)やかさ  (穩定程度)

活発(かっぱつ)　　　→　　活発(かっぱつ)さ   (活潑程度)

元気(げんき)　　　→　 　元気(げんき)さ    (活力程度)

静(しず)か    　→   　静(しず)かさ   (安靜程度)

便利(べんり)　   →    　便利(べんり)さ  (便利程度)

簡単(かんたん)　  →    　簡単(かんたん)さ  (簡單程度)

大変(たいへん)　  →    　大変(たいへん)さ  (累人程度)

いい　   →     　よさ    (好壞)

形容詞動詞化

我們可以使用「がる」，將形容詞轉化為動詞，將形容詞所表示的內在意思和狀態，具體以動詞表示於外在的行為和態度上。

變化方法: い形容詞－「い」＋「がる」

な形容詞＋「がる」

形容詞： 形容心理的状態

形容詞＋がる： 將心理状態表露於外表的樣子。

例：　面白い   : 心裡覺得很有趣。

面白がる:  外表表現出覺得很有趣的樣子。

同時，因為形容詞是「用以形容個人的感覺」，因此只能用在自己身上，不能用在別人身上。如果想要表示他人對某事物的感覺，就可以用「形容詞＋がる」的動詞形式。

面白い   : 只能用於第一人稱。

面白がる:  可以用於第二三人稱。

○　私は言葉の研究が面白いと思う

×　彼は言葉の研究が面白いと思う

○　彼は言葉の研究を面白がる。

寂しい   : 只能用於第一人稱。

寂しがる:  可以用於第二三人稱。

○　私は寂しいと思う

×　彼は寂しいと思う

○　彼は寂しがる（寂しがっている）。

其他例子如下:

強い  : 很強、很厲害。

強がる: 外表裝出很強的樣子。逞強

寒い：  很冷。

寒がる：外表表現出很冷的樣子。怕冷。

寒がり：怕冷的人。

其他形容詞動詞化的例子如下:

い形容詞＋がる：

寒(さむ)い　　→　　寒(さむ)がる     (感到寒冷的樣子)

恐(こわ)い　　→　　恐(こわ)がる     (感到恐懼的樣子)

面白(おもしろ)い　→　　面白(おもしろ)がる  (感到有趣的樣子)

痛(いた)い　　→　　痛(いた)がる     (感到痛苦的樣子)

嬉(うれ)しい　　→　　嬉(うれ)しがる (感到開心的樣子)

かゆい　　→　　かゆがる  (感到很癢的樣子)

かわいい　　→　かわいがる (覺得很可愛的樣子、愛護)

な形容動詞＋がる：

重宝(ちょうほう)　　→　　重宝(ちょうほう)がる  (感到珍惜的樣子)

いや　　  →　　いやがる    (感到討厭的樣子)

退屈(たいくつ)　　 →　　退屈(たいくつ)がる    (感到無聊的樣子)

面倒(めんどう)　   →  　面倒(めんどう)がる    (感到麻煩的樣子)
