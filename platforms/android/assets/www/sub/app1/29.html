单元29    义务と许可の言い方

            (义务和许可的说法)

 

学习目标

 ①  学习表示义务的句型「～なければなりません」

②　学习表示许可的句型「～なくてもいいです」

③　学习表示许可的句型

「～てもかまいません」「～なくてもかまいません」

 

我们在上一单元学到了「动词ない形」，也就是「动词否定形」的用法，在学习下一项动词变化、进入下一单元前，我们打铁趁热，趁着各位对于

「动词ない形」记忆犹新的时候，先介绍几项和「动词ない形」相关的句型用法。

 

首先是表示责任、义务的「～なければなりません」，相当于中文的「必须…；不得不…」，例如「不得不去上学」「不得不写作业」「不得不12点前睡觉」等等。

 

其次为表示许可、允许的句型。我们先前学过「～てもいいです」，意思为「可以做什么什么事」，例如「见てもいいです(你可以看)」「使ってもいいです(你可以使用)」。这次加上「动词ない形」，变成「～なくてもいいです」的句型，多了否定的意思，意思是「不做什么什么事也可以」，

 

 

 

第二项表示许可的句型，是「～てもかまいません」，句型架构和「～てもいいです」大致相同，「かまいません（かまわない）」的原义为「没有关系」，因此整句意思为「即使做什么什么事也没关系」。最后加上「动词ない形」，多了否定的意思，变成「～なくてもかまいません」，意为「即使不做什么什么事也没关系」。

 

简单说明如上，我们会在后面的篇幅中详细介绍用法，以及附上例句，让这些句型更容易理解。这些句型皆属于「会话句型」，也就是说，我们所学到的，都是在日常会话中能够立刻使用的文法句型，非常实用，学起来也比较不会枯燥乏味。

 

 

～なければなりません

 

别被这么一长串假名吓到，其实它表达的意思很单纯，为「必须…；不得不…」的意思，例如「必须上学」「必须写作业」等等。

 

基本用法: 「动词ない形」－「い」＋「ければなりません」

 

例：

 

学校(がっこう)に行(い)く　学校(がっこう)に行(い)かない　

→　学校(がっこう)に行(い)かなければなりません　( 不去学校不行。)

仕事(しごと)をする　仕事(しごと)をしない　

→　仕事(しごと)をしなければなりません  ( 不工作不行。)

先生(せんせい)に電話(でんわ)する　先生(せんせい)に電話(でんわ)しない　

→　先生(せんせい)に電話(でんわ)しなければなりません  ( 不打电话给老师不行。)

 

 

家(うち)に帰(かえ)る　家(うち)に帰(かえ)らない　

→　家(うち)に帰(かえ)らなければなりません  ( 不回家不行。)

単語(たんご)を覚(おぼ)える　単語(たんご)を覚(おぼ)えない　

→　単語(たんご)を覚(おぼ)えなければなりません  ( 不记单字不行。)

12時(じ)に寝(ね)る　12時(じ)に寝(ね)ない　

→　12時(じ)に寝(ね)なければなりません   ( 不十二点时睡觉不行。)

 

 

也可以和我们之前学过的「から」一起使用。

 

例:

 

仕事(しごと)をする

→　会社(かいしゃ)に入(はい)ったから、仕事(しごと)をしなければなりません。

( 因为已经进了公司，因此不工作不行。)

 

先生(せんせい)に電話(でんわ)する

→　明日用事(あしたようじ)があるから、先生(せんせい)に電話(でんわ)しなければなりません。

( 因为明天有事，因此不打电话给老师不行。)

 

家(うち)に帰(かえ)る

→　もう11時(じ)だから、家(うち)に帰(かえ)らなければなりません。

( 因为已经十一点了，因此不回家不行。)

 

 

単語(たんご)を覚(おぼ)える

→　英語(えいご)のテストがあるから、単語(たんご)を覚(おぼ)えなければなりません。

( 因为有英文考试，因此不记单字不行。)

12時(じ)に寝(ね)る

→　明日(あした)は卒業旅行(そつぎょうりょこう)だから、12時(じ)に寝(ね)なければなりません。

( 因为明天是毕业旅行，因此十二点不睡觉不行。)

 

 

なくてもいいです

 

「～なくてもいいです」为先前学过的「～てもいいです」句型，加上表示否定的「动词ない形」。

 

基本句型:  「动词ない形」－「い」＋「てもいいです」

( 不….也可以 )

 

例：
読(よ)む　  →　読(よ)まない　 →　読(よ)まなくてもいい  (不读也可以。)

言(い)う　  →　言(い)わない　 →　言(い)わなくてもいい  (不说也可以。)

食(た)べる　→　食(た)べない　 →　食(た)べなくてもいい  (不吃也可以。)

覚(おぼ)える　→　覚(おぼ)えない　→　覚(おぼ)えなくてもいい  (不记也可以。)

する　  →　しない　   →　しなくてもいい    (不做也可以。)

来(く)る　  →　来(こ)ない　   →　来(こ)なくてもいい    (不来也可以。)

 

てもかまいません

 

「～てもかまいません」和「～てもいいです」句型的差别，只有在后面的部分，「いいです」是「可以」的意思；「かまいません」的动词原形是「かまわない」，是「没关系」的意思，因此整句意为「即使…也没关系」。

意思和「～てもいいです( ….也可以 )」很类似，但是感觉起来较为委婉有礼貌。

 

 

基本用法: 「动词て形」＋「も」＋「かまいません」

( 即使…也没关系 )

 

例：

 

読(よ)む　　→　読(よ)んで　→　読(よ)んでもかまいません  (读了也没关系。)

言(い)う　　→　言(い)って　→　言(い)ってもかまいません  (说了也没关系。)

食(た)べる　→　食(た)べて　→　食(た)べてもかまいません  (吃了也没关系。)

覚(おぼ)える　→　覚(おぼ)えて　→　覚(おぼ)えてもかまいません (记住也没关系。)

する　　→　　して　 →　してもかまいません   (做了也没关系。)

来(く)る　　→　　来(き)て　 →　来(き)てもかまいません   (来也没关系。)

 

～なくてもかまいません

 

「～てもかまいません」＋「动词ない形」＝「～なくてもかまいません」。

「～てもかまいません」:  即使…也没关系

「～なくてもかまいません」:  即使不…也没关系

 

基本用法:

「动词ない形」－「い」＋「くて」＋「も」＋「かまいません」

 

例：

 

読(よ)む　　→　読(よ)まない　

→　読(よ)まなくてもかまいません　(不读也没关系。)

行(い)く　　→　行(い)かない　

→　行(い)かなくてもかまいません  (不去也没关系。)

食(た)べる　→　食(た)べない　

→　食(た)べなくてもかまいません  (不吃也没关系。)

覚(おぼ)える　→　覚(おぼ)えない　

→　覚(おぼ)えなくてもかまいません  (不记住也没关系。)

する　　→　　しない　　

→　しなくてもかまいません    (不做也没关系。)

来(く)る　　→　　来(こ)ない　　

→　　来(こ)なくてもかまいません  (不来也没关系。)

 

 

问句说法

 

本单元介绍的句型，只要在句尾加上表示疑问的「か」，就能够直接当成疑问句使用，如下:

 

①　～なければなりませんか

②　～てもいいですか。～なくてもいいですか

③　～てもかまいませんか。～なくてもかまいませんか。

 

至于肯定和否定的回答方式，请参考以下例句，肯定句的问答方式很单纯，只要重复疑问句即可，不过否定句的回答方式，就得转一圈，使用其他句型了。

 

例：

 

宿题をしなければなりませんか。

はい、宿题をしなければなりません。

いいえ、宿题をしなくてもいいです。

不写作业不行吗?

是的，不写作业不行。

不，不写作业也可以。

 

ピーマンを食べなければなりませんか。

はい、ピーマンを食べなければなりません。

いいえ、ピーマンを食べなくてもいいです。

不吃青椒不行吗?

是的，不吃青椒不行。

不，不吃青椒也可以。

 

テレビを见てもいいですか。

はい、テレビを见てもいいですよ。

いいえ、テレビを见てはいけません。

可以看电视吗?

是的，可以看电视。

不，不行看电视。

 

 

宿题をしなくてもいいですか。

はい、しなくてもいいです。

いいえ、宿题をしなければなりません。

不写作业也可以吗?

是的，不写作业也可以。

不，不写作业不行。

 

テレビを见てもかまいませんか。

はい、テレビを见てもかまいません。

いいえ、テレビを见てはいけません。

看电视也没关系吗?

是的，看电视也没关系。

不，不行看电视。

 

宿题をしなくてもかまいませんか。

はい、しなくてもかまいません。

いいえ、宿题をしなければなりません。

不写作业也没关系吗?

是的，不写作业也没关系。

不，不写作业不行。
