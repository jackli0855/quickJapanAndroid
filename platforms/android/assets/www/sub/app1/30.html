另外，我们也会一并介绍表示经验的「动词た形＋ことがある」句型。我们学过表示能力的「～ことができる」句型、以及名词化的「こと」，因此对于这个句型应该很快就能上手。「动词た形＋ことがある」主要功能为叙述过去经验，相当于中文的「我曾经…」，更多说明及例句请参考以下讲解。

 

动词た形

 

「动词た形」主要用法和「～ました」相同，表示「过去式」，也就是过去某个时间点所发生的事情和动作。动词变化方法，和「动词て形」完全相同，将「动词て形」的「て」代换成「た」即可。

 

变化方法: 　「动词て形」－「て」＋「た」

 

注意: 原本在「动词て形」中会变成浊音「で」的字，在「动词た形」中也要变成浊音的形式「だ」。例：「泳ぐ→泳いで→泳いだ」

 

第一类动词

 

例：

 

す　→　した

 

く?ぐ　→　いた

 

う?つ?る　→　った

 

ぶ?む?ぬ　→　んだ

 

例：

 

貸(か)す　→　貸(か)して　→　貸(か)した

刺(さ)す　→　刺(さ)して　→　刺(さ)した

書(か)く　→　書(か)いて　→　書(か)いた

泳(およ)ぐ　→　泳(およ)いで　→　泳(およ)いだ

言(い)う　→　言(い)って　→　言(い)った

待(ま)つ　→　待(ま)って　→　待(ま)った

帰(かえ)る　→　帰(かえ)って　→　帰(かえ)った

飛(と)ぶ　→　飛(と)んで　→　飛(と)んだ

読(よ)む　→　読(よ)んで　→　読(よ)んだ

　

第二类动词

 

例：

 

食(た)べる　→　食(た)べた

着(き)る　→　着(き)た

忘(わす)れる　→　忘(わす)れた

寝(ね)る　→　寝(ね)た

開(あ)ける　→　開(あ)けた

見(み)る　→　見(み)た　

不规则动词

 

する　→　した

勉強(べんきょう)する　→　勉強(べんきょう)した

買物(かいもの)する　→　買物(かいもの)した

来(く)る　→　来(き)た

 

 

～ことがある

 

「动词た形＋ことがある」的主要功能为表示过去的经验，若照字面直接翻译的话，「动词た形」为「过去式」，「ことがある」是「有这样的事」，因此整句话直译就是: 「过去有这样的事」，也就是「曾经做过什么事」的意思。

 

基本用法: 「动词た形」＋こと　が　ある／あります

           ( 曾经…  )

 

例：

 

私(わたし)は日本(にほん)に行(い)ったことがあります。

( 我曾经去过日本。 )

私(わたし)は市立(しりつ)プールで泳(およ)いだことがあります。

( 我曾经去市市游泳池游过泳。 )

私(わたし)は塾(じゅく)で日本語(にほんご)を学(まな)んだことがあります。

( 我曾经在补习班学过日文。 )

 

私(わたし)はパンダを见たことがあります。

( 我曾经看过猫熊。 )

私(わたし)は懐石料理(かいせきりょうり)を食(た)べたことがあります。

( 我曾经吃过怀石料理。 )

私(わたし)は徹夜(てつや)でテストの勉強(べんきょう)をしたことがあります。

( 我曾经熬夜读书准备考试。 )

私(わたし)はこの場所(ばしょ)に来(き)たことがあります。

( 我曾经来过这个地方。 )

 

「动词た形＋ことがある」「动词た形」的区别

 

本单元介绍的二个用法，都是表示「过去做过的事」，那么究竟该如何区别这二种句型用法呢?  请看以下解说 :

 

「动词た形」: 代表短时间内做过的事

「动词た形＋ことがある」: 代表做过的事距今已经有段时间了

 

不过，用中文的语感来想就很容易理解了:

 

「动词た形」: 「…了」

「动词た形＋ことがある」:「曾经…」

 

彼女は学校に行きました。

(她去学校了。)  可能十分钟前才出门

 

彼女は学校に行ったことがあります。

(她曾经去过学校。)  可能读过小学，但是之后因故失学，没有再去学校

 

 

私は映画を见ました。

(我去看了电影。)  可能是上周才看。

 

私は映画を见たことがあります。

(我曾经看过电影。)  可能很久之前看过，但是很久没有看了
