单元26    动词て形＋授受动词

 

学习目标

 ①学习「动词て形」＋「くれる」「あげる」「もらう」等授受动词的用法。

 

我们在前面的单元中，介绍过日文的授受动词，各位还记得吗?

 

例：

私は先生から本をもらいます。

先生は私に小说をくれます。

先生は彼にプレゼントをあげます。

 

所谓的「授受动词」，即是表示「给予、得到」的动词。如上述例句，我们先前学到的，是关于「名词+授受动词」的讲法，像是书、小说、礼物等等具体的东西。但是，授受动词的另一大用法，则是用来连接「动词」。啊? 什么意思??

 

例句①：先生は私に日本语を教えます。

( 老师教我日文。)

 

将最后的动词改为て形后，加上「くれる」

 

例句②：先生は私に日本语を教えてくれます。

( 老师教我日文。)

 

咦? 动词「教」这件事不是「具体的东西」，而是「抽象的东西」，要怎么给予呢? 这二句有什么地方不一样呢?  第一句的意思为「老师教我日文」；第二句的意思，直译是「老师给我『教日文』这件事」，也是「老师教我日文」的意思。但是不一样的地方是，由于例句二使用了授受动词「くれる」，「くれる」是「给予」的意思，给予什么呢? 在日本人的想法中，给予了「恩惠」。

 

例句①：先生は私に日本语を教えます。

(单纯描述老师教我日文这件事。)

 

例句②：先生は私に日本语を教えてくれます。

(除了描述老师教我日文之外，另外还有接受老师的恩惠，表达感谢之意。)

 

日本是非常重礼节的国家，因此如何表示「恩惠」「感谢」是很重要的，例句一和例句二虽然在字义上几乎相同，但是在对方听起来，却是完全不同的感觉。例句二除了单纯描述之外，还多了受到对方恩惠、表达感谢之意，是较为礼貌的说法，因此在口语会话时经常使用。

 

简单来说，当「名词+授受动词」时，意思为给予、得到一项具体的东西，例如：「本をくれます」，当「动词+授受动词」时，由于是「抽象的动作」，因此意思为给予、得到「恩惠」。

 

关于详细介绍，请继续往下看。

 

 

动词て形＋くれる

 

「くれる」为「对方给予我….」之意。

 

基本用法:  「主词」は 「对象」に「动词て形」＋くれる

 

可以试着比较下面二句的差别。

 

 

①　先生が私に日本语の文法を说明します。

②　先生が私に日本语の文法を说明してくれます。

 

 

当我们使用第一句时，只是单纯在描述一个事件，不带有任何感情。但是如果今天我们想向别人表达「老师教我日文文法耶! 真是太感谢了!」时，就必须使用第二句的型式，因为第二句中多了「老师给我恩惠」的感觉，适合用来表达感谢之意。

 

 

图解如下:

 


例：

 

母(はは)は私(わたし)に飴(あめ)を買(か)ってくれます。

( 母亲买糖果给我。 )

高野(たかの)さんは私(わたし)に英語(えいご)を教(おし)えてくれます。

(  高野先生教我英文。 )

電気屋(でんきや)の人(ひと)は私(わたし)に新(あたら)しいテレビを運(はこ)んでくれます。

(  电气行的人搬新的电视来给我。 )

役所(やくしょ)の人(ひと)は私(わたし)だちに道(みち)を案内(あんない)してくれます。

(  区公所的人告诉我们路怎么走。 )

あのおじいさんは妹(いもうと)に靴(くつ)を直(なお)してくれます。

(  那位老爷爷替妹妹修好鞋子。 )

旅行会社(りょこうがいしゃ)の人(ひと)は家族(かぞく)に海外旅行(かいがいりょこう)のビザを申(もう)し込(こ)んでくれます。

(  旅行社的人替我的家人办了国外旅行的签证。 )

 

 

动词て形＋あげる

 

「あげる」为「我给对方」或是「对方给予第三者」之意。

 

基本用法:  「主词」は 「对象」に「动词て形」＋あげる

 

例:

 

王(おう)さんは竹内(たけうち)さんに仕事(しごと)の方法(ほうほう)を教(おし)えます。

　　　　　　　↓

王(おう)さんは竹内(たけうち)さんに仕事(しごと)の方法(ほうほう)を教(おし)えてあげます。

 

 

如果「くれる」为「接受恩惠」，那么「あげる」就是「给予恩惠」的意思。因此同样地，第一句话为单纯描述某个事件，第二句话则带有「王さん给予竹内さん恩惠」的感觉，听起来会像是「王さん明明很忙，却拨出时间来教」「王さん特地去教」的感觉。

 

图解如下:

 

 


例：

 

私(わたし)はあなたに宿題(しゅくだい)の答(こた)えを教(おし)えてあげます。

( 我告诉你作业的答案。)

母(はは)は弟(おとうと)にオモチャを買(か)ってあげます。

( 母亲买玩具给弟弟。)

王(おう)さんは彼女(かのじょ)にギターを弾(ひ)いてあげます。

( 王先生弹吉他给女朋友听。)

先生(せんせい)はクラスメートにきれいな絵本(えほん)を貸(か)してあげます。

( 老师借给班上同学漂亮的绘本。)

寮(りょう)の管理員(かんりいん)さんは郵便物(ゆうびんぶつ)をみんなのメールボックスに入れてあげます。

( 宿舍管理员将邮件放到大家的信箱当中。)

 

动词て形＋もらう

 

「もらう」为「得到…」之意。

 

基本用法:  「主词」は 「对象」に「动词て形」＋もらう

　　　　　　( 某人让某人做…)

 

例:

 

私(わたし)は岸部(きしべ)さんにギターの弾(ひ)き方(かた)を教(おし)えます。

　　　　　　　　　　↓

私は岸部さんにギターの弾き方を教えてもらいます。

 

以上二句例句，虽然差别只有在句尾的「もらう」，但是整句的意思完全不一样。第一句是「我教岸部先生吉他的演奏方法」，第二句则是「我从岸部先生得到了『教吉他』这件事」，也就是「岸部先生教我吉他」。请特别注意这种给予者和被给予者的关系。

 

图解如下:

 


例：

 

私(わたし)は友達(ともだち)に本(ほん)を買(か)ってもらいます。

( 朋友帮我买了书。 )

私(わたし)は弟(おとうと)に部屋(へや)を掃除(そうじ)してもらいます。

( 弟弟帮我打扫房间。)

私(わたし)は旅行会社(りょこうがいしゃ)の人(ひと)にホテルを予約(よやく)してもらいます。

( 旅行社的人帮我预约旅馆。 )

彼(かれ)は妻に昼(ひる)ごはんの弁当(べんとう)を作(つく)ってもらいます。

( 他的妻子为他做中午的便当。 )

彼女(かのじょ)は彼氏(かれし)に食事代(しょくじだい)を払(はら)ってもらいます。

( 她的男朋友帮她付吃饭钱。 )

田村(たむら)さんは林(はやし)さんに中国語(ちゅうごくご)を教(おし)えてもらいます。

( 林先生教田村先生中文。 )

あの塾(じゅく)はパートにチラシを配(くば)ってもらいます。

( 工读生帮那间补习班发传单。 )
