单元18    好き嫌いと程度副词

           （喜好和程度副词）



学习目标

①学会喜好的说法「好き」「嫌い」

②学会使用程度副词「一番」「すごく」「非常に」「とても」「かなり」

 

 

还记得我们之前学过的形容词和形容词句型吗? 我们将利用之前学过的二个形容词「好き(喜欢)」以及「嫌い(讨厌)」，配合形容词句型，介绍如何以日文表达「个人喜好」。所谓的个人喜好，就是「我喜欢什么」「我讨厌什么」等等的说法。

 

至于为什么要学习喜欢和讨厌的讲法呢? 首先，喜欢和讨厌是构成个人情绪的基本部分，你一定有一些喜欢和讨厌的东西，而且经常在日常会话中出现，例如「我喜欢赖床」和「我讨厌吃青椒」等等，使用频率不低。其次，也是最主要的功能，就是藉由这个句型，让你学会所谓的「程度副词」。

 

什么是「程度副词」? 「程度副词」就是表达程度的副词。好吧，你一定认为这是废话，那么让我们举个实例来解说吧。

 

A: 我讨厌吃青椒!

B: 讨厌到什么程度?

A: 相当讨厌。不，应该说非常讨厌!

 

句中「相当讨厌」的「相当」，以及「非常讨厌」的「非常」，都是用来表示「讨厌」的程度有多大，这些词就是「程度副词」。不明白也没关系，稍后你就会明白是怎么回事了。

 

这一单元有二个学习重点: ①学会喜好的说法。②学会使用程度副词。我们将依序介绍。

 

喜欢和讨厌

 

我们会用到以下二个形容词。

 

「好き」：な形容词。等于中文的「喜欢」。

「嫌い」：な形容词。等于中文的「讨厌」。

 

我们之前学习过的形容词句型如下:

 

形容词句型 ：名词　は　形容词　です。

 

然而，在表示个人喜好时，会使用下面这个句型:

 

句型 ：　私　は　名词　が　好き／嫌い　です。

(我喜欢／讨厌…)

　例：私(わたし)は寿司(すし)が好(す)きです。     ( 我喜欢寿司。 )

　　　私(わたし)は日本料理(にほんりょうり)が好きです。( 我喜欢日本料理。)

　　　私(わたし)はゲームが好(す)きです。   ( 我喜欢游戏。 )

　　　私(わたし)は魚(さかな)が嫌(きら)いです。     ( 我讨厌鱼。 )

　　　私(わたし)は冬(ふゆ)が嫌(きら)いです。      ( 我讨厌冬天。 )

　　　私(わたし)はピーマンが嫌(きら)いです。  ( 我讨厌青椒。 )

 

句型演变过程如下:

 

「名词　は　形容词　です」＋「私は」

　　　　　　　↓

「私　は　名词　は　形容词　です。」出现二个「は」，变成其他意思。

　　　　　　　↓

「私　は　名词　が　形容词　です。」将第二个「は」改成「が」。

 

程度副词

 

以下介绍几个常用的程度副词:

 

「一番」: 最...。程度为100%。

「すごく」「非常に」: 非常...。程度约为90%。

「とても」「かなり」: 很...。程度约为80%。

 

程度副词的用法:

私　は　名词　が　程度副词＋好き／嫌い　です。

 

例：私(わたし) は 懐石料理(かいせきりょうり) が 一番(いちばん) 好(す)きです。( 我最喜欢怀石料理。 )

　　私(わたし) は 懐石料理(かいせきりょうり) が すごく 好(す)きです。( 我超喜欢怀石料理。 )

　　私(わたし) は 懐石料理(かいせきりょうり) が 非常(ひじょう)に 好(す)きです。(我非常喜欢怀石料理。)

　　私(わたし) は 懐石料理(かいせきりょうり) が とても 好(す)きです。(我相当喜欢怀石料理。)

　　私(わたし) は 懐石料理(かいせきりょうり) が かなり 好(す)きです。( 我很喜欢怀石料理。)

 

 

也可以将上句中「好き」换成「嫌い」，程度仍然不变，从「最讨厌」、「非常讨厌」、到「很讨厌」。

 

以上介绍的，都是程度很高的副词(100%─80%)，那么有没有程度低的副词呢?

 

「あまり」：不太...。程度约30%。

「全然（ぜんぜん）」：完全不...。程度为0%。

 

各位有没有注意到，「あまり(不太)」和「全然(完全不)」，都有一个「不」字，因此这二个副词的用法，也和上述的用法不太一样，句尾的形容词必须变成否定。

 

「あまり」「全然」的句型用法:

 私　は　名词　が　程度副词　好き／嫌いではありません。

 

例：私(わたし)は刺身(さしみ)があまり好(す)きではありません。( 我不太喜欢生鱼片。)

　　私(わたし)は茶碗蒸(ちゃわんむ)しがあまり嫌(きら)いではありません。(我不太讨厌蒸蛋。)

　　私(わたし)は納豆(なっとう)が全然好(ぜんぜんす)きではありません。( 我完全不喜欢纳豆。)

　　私(わたし)は味噌汁(みそしる)が全然嫌(ぜんぜんきら)いではありません。(我完全不讨厌味噌汤。)

 

 

会话例：

 

A：ケンは彼女(かのじょ)のこと、どう思(おも)います？( Ken，你觉得她如何? )

B：うん…あまり嫌(きら)いではありませんが…(嗯…虽然不太讨厌…)

A：じゃ、好(ず)きってことですね。(那么，就是喜欢啰! )

B：いや、好(す)きでもありません…(不，也不算是喜欢… )

A：全然好(ぜんぜんす)きではありませんか。(完全不喜欢吗?)

B：ううん、少(すこ)し好(す)き……かな？( 也不是，是有一点喜欢…吧…)
