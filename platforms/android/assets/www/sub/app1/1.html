動詞的時態

將動詞時態用法，全部塞進一張表格裡的話，就會像下面這樣 :

時態 對應的動詞形式
過去式 ～た　～ました
現在式 ～ている　～ています
未來式 動詞原形　動詞ます形

過去式: 表示在過去某時間點發生的動作。

現在式: 表示現在正在從事的動作。

未來式: 表示將來、或是等一下將去做的動作。

如果你正在閱讀這篇教學文章，那麼代表你應該對於日文文法有了一定程度的認識，因此，我們本回將進行較進一步的文法說明─關於日文的時態。

首先， 我們先來定義一下，什麼是過去式、現在式、以及未來式。

過去式

過去           現在           未來

時間軸

在過去某時間點發生的動作

現在式

過去           現在          未來

時間軸

在現在當下的時間點進行的動作

未來式

過去              現在        未來

時間軸

在未來某時間點會發生、進行的動作。

如同上表所述，日文當中，動詞原形既可以當作現在式，也可以當作未來式使用。但是一般在會話中，動詞原形給人的感覺，會比較偏向未來式，也就是用來表示還沒發生的事情、或是現在正要從事的動作。請對照著中文看，會比較容易體會語感。

例:

私(わたし)は学校(がっこう)に行(い)く。

我要去上學 ( 給人感覺是現在還沒開始上學，之後才會開始)

私(わたし)は夜(よる)ご飯(はん)を食(た)べます。

我要去吃飯。( 給人感覺是現在還沒開始吃飯，之後才會吃)

私(わたし)は風呂(ふろ)に入(はい)ります。

我要去洗澡。( 給人感覺是現在還沒洗澡，之後才會去洗)

私(わたし)は明日学校(あしたがっこう)に行(い)きます。

我明天要去學校 (未來式)

由於動詞原形兼具現在式和未來式的功能，且未來式的感覺較強烈，因此，如果要特定表示「現在當下正在做的事情」，則必須使用動詞的「現在進行式」。

例：

私(わたし)はご飯(はん)を食(た)べます。

（我要吃飯。）

私(わたし)はご飯(はん)を食(た)べています。

（我正在吃飯。）

総統(そうとう)はアメリカを訪(たず)ねる。

（總統將造訪美國。）

総統(そうとう)はアメリカを訪(たず)ねている。

（總統正在造訪美國。）

私(わたし)は風呂(ふろ)に入(はい)る。

（我要洗澡。）

私(わたし)は風呂(ふろ)に入(はい)っている。

（我正在洗澡。）

因此，我們將動詞原形定位為+未來式*，意思是+我即將去做…*，

將ている定位為現在式，意思是+我現在正在做…*。

以中文為母語的日文學習者，很多人會將動詞原形和ている的用法弄錯(包括我在內)。

例:

×(　)　弟(おとうと)は部屋(へや)で宿題(しゅくだい)をする。

○(　)　弟(おとうと)は部屋(へや)で宿題(しゅくだい)をしている。

（弟弟在房間裡寫作業。）

中文的意思是「弟弟正在房間中寫作業」，第一句為未來式，表示「弟弟等一下會在房間寫作業」，意思不同，因此必須使用第二句「ている」的形式。

相同的例子還有:

×(　)　彼女(かのじょ)は大学院(だいがくいん)で勉強(べんきょう)する。

○(　)　彼女(かのじょ)は大学院(だいがくいん)で勉強(べんきょう)している。

（她在研究所唸書。）

×(　)　父(ちち)は新聞(しんぶん)を読(よ)む。

○(　)　父(ちち)は新聞(しんぶん)を読(よ)んでいる。

（爸爸在看報紙。）

簡單來說，當你在寫作文或說話時，

想表達之前發生過的事情: 　過去式（～た／～ました）

想表達現在正在進行的動作: 現在式（～ている／～ています）

想表達將來、或是稍後會進行的動作: 未來式（動詞原形）

我這麼寫，一定會被許多學術派的教授砍頭，不過這的確是最合乎日本人在說日文時的心理和使用方式，就我的經驗，如此也能大大減少外國人在使用日文時發生時態錯誤的機率。

變化過程的表示

將日文中表示動作、狀態過程的基本二項句型，整理如下 :

變化過程的表示  
過去→現在 ～てきた　～てきました
現在→未来 ～ていく　～ていきます

「～てきた」：「動詞て形」＋「来る」の過去形

表示從過去到現在的變化情形。

相當於中文的「…起來」

「～ていく」：「動詞て形」＋「行く」

表示從現在到未來的變化情形。

相當於中文的「…逐漸…」

「～てきた」圖解

過去            現在          未來

時間軸

「～ていく」圖解

過去              現在         未來

時間軸

例:

外食(がいしょく)の人(ひと)が多(おお)くなる。

（外食人口會變多。）未來式

外食(がいしょく)の人(ひと)が多(おお)くなった。

（外食人口變多了。）過去式

外食(がいしょく)の人(ひと)が多(おお)くなってきた。

（外食人口變多起來了。）過去到現在的變化

外食(がいしょく)の人(ひと)が多(おお)くなっていく。

（外食人口會逐漸變多下去。）現在到未來的變化

中小企業(ちゅうしょうきぎょう)の数が少(すく)なくなる。

（中小企業的數量會減少。）未來式

中小企業(ちゅうしょうきぎょう)の数(かず)が少(すく)なくなった。

（中小企業的數量減少了。）過去了

中小企業(ちゅうしょうきぎょう)の数(かず)が少(すく)なくなってきた。

（中小企業的數量變少了。）過去到現在

中小企業(ちゅうしょうきぎょう)の数(かず)が少(すく)なくなっていく。

（中小企業的數量會逐漸減少下去。）現在到未來

ゴミの量(りょう)が増(ふ)える。

（垃圾量會增加。）未來式

ゴミの量(りょう)が増(ふ)えた。

（垃圾量增加了。）過去式

ゴミの量(りょう)が増(ふ)えてきた。

（垃圾量增加起來了。）過去到現在

ゴミの量(りょう)が増(ふ)えていく。

（垃圾量會逐漸增加下去。）現在到未來
