单元24    动词て形について

            (关于动词て形)

 

学习目标

 

①　学习「动词て形」的变化方法。

②　学习「动词て形」的基本使用方法。

 

锵锵锵锵，本单元开始，将堂堂进入日文动词变化的关键点:「动词て形」，噢，我彷佛可以听见许多人的哀号，若你之前曾经立志学好日文，后来又无法持续下去的话，那么我想以「动词て形」为首的动词变化，可能是将你击沈的其中一枚巨大飞弹。各位听过日文学习的80/20法则吗? 日文有三块大石头，分别是五十音、动词变化和日文敬语。有80%想学日文的人，在五十音时就宣告放弃，只有20%的人成功记熟五十音；这些20%的人，又有80%在日文动词变化(特别是动词て形之后)时投降，只有20%的人努力克服了这个关卡；然后这些20%的人，又有80%在日文敬语的学习上迟迟无法前进，只有其中20%能够学有所成。

 

先不论商业和正式场合，其实如果日文敬语讲得不好，那么在日常会话中并不会带来非常大的困扰，但是「五十音」和「动词变化」就不是那么回事了。「五十音」相当于英文的字母，学不好五十音而想学好日文，就像车子没有轮胎一样不切实际，其重要性不言而喻。然而，以「动词て形」为首的动词变化，则会大大影响口语会话的流畅度，有些人的日文听起来

 

非常流畅，有些人却听起来断断续续、讲不出一个完整句子，其原因除了发音之外，文法熟悉度也是重要关键，而文法熟悉度中，动词变化的熟习度又至关紧要。如果五十音是车子的轮胎，那么动词变化就是油门和煞车，车子能够动起来，靠得是轮胎，但是车子能开多快、开多稳，靠的则是油门和煞车。

 

本单元中，我们的重点，就是教会各位「动词て形」的变化和基本使用方法。「动词て形」有什么功用吗? 简单来说，它就像是万能胶，能够将日文中的许多句子、许多动词黏在一起，使其变成一个完整的语句。我们在学校英文课中学过，动词和动词不能放在一起，例如「People eat drink」这句话基本上是错的，必须改成「People eat, and drink」，英文中用「and」连接动词和动词。

 

换成日文的话，「私はご饭を食べるコーラを饮む。」这句话也是错的，因为「食べる」和「饮む」二个动词不能在同一句话中，会打架。但是用「动词て形」黏起来就没有问题，「私はご饭を食べて、コーラを饮む。」

因此，某种程度上来说，「て形」和英文「and」的功能很像，专门当作万能胶，连接复数的动词或句子。这就是「て形」的基本用法，讲完了。是不是没有想象中的困难?

 

至于为什么「动词て形」等动词变化会让80%的人宣告放弃呢?  有二个原因，第一，动词变化的规则有些多，第二，也是最主要的原因，老师教的方法太复杂。正因为动词变化对于中文母语者不好理解，因此教授方法更显得重要，要教得很复杂、或是浅显易懂，要将简单的东西教得很复杂、或是将复杂的东西教得简单，全凭日文老师的功力和技巧。然而老师的教学方式，则会大大影响到学习者的学习意愿和效率，有些人学习顺畅、没遭遇什么理解上的困难，但是有些人却痛苦到快要升天。这不是因为后者比较笨，只是因为使用不一样的学习教材和方法。

 

关于「动词て形」等动词变化，我们参考了许许多多的书籍和日语教师的经验方法，包括观摩东吴大学和台湾大学的课程(我是这二所学校毕业的)，整理出既浅显易懂、容易记忆，同时文法上精准无比的学习方法。

请跟着我们一起学习。

 

动词て形

 

还记得我们之前学过，动词分为三类:第一类、第二类和不规则吗? 这次我们也是依照相同的分类，进行动词变化。同样地，只有第一类动词需要动点脑筋，第二类和不规则的变化方式都相当简单。

 

另外，我们在当初介绍日文动词时，就要各位一并记住「动词原形」和「动词ます形」吗? 就是为了现在。 在学习日文动词变化(无论哪一种都一样)时，使用动词原形是最轻松简单、方便记忆的方式!

 

我们介绍如何将「动词原形」变化为「动词て形」。

 

第一类动词(最难的)

 

字尾为「す」的动词

 

て形变化方法：将最后的字母由U换成I，加上「て」

 

例：貸(か)す　kasu　→　　貸(か)して  kasite

　　指(さ)す　sasu　→　　指(さ)して  sasite

　　無(な)くす  nakusu  →　　無(な)くして  nakusite

　　刺(さ)す　sasu　→　　刺(さ)して  sasite

　　出(だ)す　dasu　→　　出(だ)して  dasite

　　戻(もど)す　modosu　→　　戻(もど)して  modosite

 

字尾为「く?ぐ」的动词

 　

基本上相同，将最后的字母由「U」换成「I」，加上「て」，但是由于实在太难念了，例如「书く」→「书きて」「闻く」→「闻きて」，大家可以念念看「书きて」「闻きて」，还真不好念，因此就偷懒，将「~きて」改念为「～いて」，念念看「书いて」「闻いて」，是不是比较好念啦?

 

て形变化方法：将假名「く?ぐ」，改为「いて?いで」

 

例：

書(か)く　→　書(か)いて

引(ひ)く　→　引(ひ)いて

描(えが)く　→　描(えが)いて

響(ひび)く　→　響(ひび)いて

続(つづ)く　→　続(つづ)いて　　

嗅(か)ぐ　→　嗅(か)いで

凌(しの)ぐ　→　凌(しの)いで

泳(およ)ぐ　→　泳(およ)いで

仰(あお)ぐ　→　仰(あお)いで　

　　

 

字尾为「う?つ?る」的动词

 

仔细看，在「う?つ?る」三个假名中，都可以看到小小的假名「つ」:

因此，「う?つ?る」变成「て形」的方法，就是加上促音「っ」后，直接加上「て」，这样比较容易记忆。

 

て形变化方法：将假名「う?つ?る」，改为「って」。

 

例：

吸(す)う　→　吸(す)って

買(か)う　→　買(か)って

言(い)う　→　言(い)って

打(う)つ　→　打(う)って

待(ま)つ　→　待(ま)って

立(た)つ　→　立(た)って

切(き)る　→　切(き)って

帰(かえ)る　→　帰って

送(おく)る　→　送(おく)って

売(う)る　→　売(う)って

字尾为「ぶ?ぬ?む」的动词

 

最后是假名「ぶ?ぬ?む」，这三个假名的「て形」比较难记忆，但还是有方法。

 

请大声念出「ぶ?ぬ?む」这三个假名的发音，然后用手摸摸自己喉咙的地方，有没有发现当我们发音时，这个部分会强烈振动。这个振动的部分是声带，我们在发出某些有声假名(像是ぶ?ぬ?む)时，声带会强烈振动，发出某些无声假名(像是か?す?し)时，声带的振动就不会那么强烈。其中，声带振动最为持续且强烈的，就是鼻音「ん」，当我们发出「ん」时，喉咙声带的部分会产生持续且强烈的振动。

 

我们可以利用这个性质来记忆て形。
「ぶ?ぬ?む」和「ん」相同，发音时声带会强烈振动，因此它们的て形:

 

て形变化方法：将「ぶ?ぬ?む」改为「んで」。

 

例：

 

飛(と)ぶ　→　飛(と)んで

運(はこ)ぶ　→　運(はこ)んで

遊(あそ)ぶ　→　遊(あそ)んで

読(よ)む　→　読(よ)んで

住(す)む　→　住(す)んで

噛(か)む　→　噛(か)んで

死(し)ぬ　→　死(し)んで

 

 

 

第二类动词

 

て形变化方法: 将动词字尾的「る」直接换成「て」。

 

例：

 

食(た)べる　→　食(た)べて

見(み)る　→　見(み)て

開(ひら)ける　→　開(ひら)けて

寝(ね)る　→　寝(ね)て

始(はじ)める　→　始(はじ)めて

集(あつ)める　→　集(あつ)めて

忘(わす)れる  →  忘(わす)れて

 

不规则动词

 

不规则动词只有二个，直接记起来吧!

 

する　→　して

来る（くる）　→　来て（きて）

 

て形用法

 

动词て形的最主要用法，就是连接句子和句子，以及连接动词和动词。

简单来说，就是将复数句子合成为一个句子。

 

基本用法:  (句子一)。(句子二)。

　　　　　　　　　↓

           (句子一)＋て、(句子二)。

 

例：

 

(描述晚上做了什么事时)

 

例：私はご饭を食べる。私はテレビを见る。私は宿题をする。

　　　　　　　　　　↓

×　私はご饭を食べるテレビを见る宿题をする。

○　私はご饭を食べて、テレビを见て、宿题をする（します）。

 

(描述如何乘车上学时)

 

 

例：私は歩く。私は电车に乗る。私はバスに乗る。

　　　　　　　　　　↓

私は歩いて、电车に乗って、バスに乗る（乗ります）。

 

 

基本用法二:  动词て形＋动词て形＋动词。

 

当我们一次使用多个动词时，也可以使用て形，将所有动词都串在一起，直接变成一句话。

 

例：

私(わたし)は本屋(ほんや)で本(ほん)を買(か)う。本(ほん)を開(ひら)く。本(ほん)を読(よ)む。

私(わたし)は本屋(ほんや)で本(ほん)を買(か)って、開(ひら)いて、読(よ)む（読(よ)みます）。

( 我在书店买书、打开、阅读。 )

 

私(わたし)はパソコンで字(じ)を打(う)つ。インターネットする。印刷(いんさつ)する。

私(わたし)はパソコンで字(じ)を打(う)って、インターネットして、印刷(いんさつ)する（します）。

( 我用计算机打字、上网、打印。 )

 

名词和形容词的て形

 

那么，当我们想一次讲很多名词、或形容词时，该怎么办呢?

其实，名词和形容词也て形，功能相同，只是规则简单多了。

 

名词?な形容词のて形：「名词?な形容词」＋で

 

例：

 

私(わたし)はサラリーマンです。私(わたし)は父親(ちちおや)です。

私(わたし)はサラリーマンで、父親(ちちおや)です。

( 我是上班族，也是父亲。 )

 

彼女(かのじょ)は英語(えいご)の先生(せんせい)です。彼女(かのじょ)は動物園(どうぶつえん)のボランティアです。

彼女(かのじょ)は英語(えいご)の先生(せんせい)で、動物園(どうぶつえん)のボランティアです。

(  她是英文老师，也是动物园的义工。 )

 

基隆(キールン)の夜市(よいち)は有名(ゆうめい)です。基隆(キールン)の夜市(よいち)は賑(にぎ)やかです。

基隆(キールン)の夜市(よいち)は有名(ゆうめい)で、賑(にぎ)やかです。

(  基隆夜市很有名，也很热闹。 )

 

あの公園(こうえん)はきれいです。あの公園(こうえん)は静(しず)かです。あの公園(こうえん)は有名(ゆうめい)です。

あの公園(こうえん)はきれいで、静(しず)かで、有名(ゆうめい)です。

(  那座公园很漂亮、安静、又有名。)　

 

い形容词のて形：「い形容词」－「い」＋「くて」

 

例：

 

大(おお)きい　→　大(おお)きくて

小(ちい)さい　→　小(ちい)さくて

広(ひろ)い　→　広(ひろ)くて

狭(せま)い　→　狭(せま)くて

 

例：

あの日本料理店(にほんりょうりてん)は広(ひろ)いです。美味(おい)しいです。安(やす)いです。

あの日本料理店(にほんりょうりてん)は広(ひろ)くて、美味しくて、安(やす)いです。

(  那间日本料理店既宽广、好吃、又便宜。 )

私(わたし)の部屋(へや)は狭(せま)いです。暗(くら)いです。きたないです。

私(わたし)の部屋(へや)は狭(せま)くて、暗(くら)くて、きたないです。

(  我的房间既小、又暗、又脏。 )
