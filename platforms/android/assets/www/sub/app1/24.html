单元37    日本语の可能形用法

             ( 日文的可能形用法)

 

学习目标

 ①　学习日文动词的「可能形」用法

②　学习「可能形」的相关用法和句型

 

 

本单元的学习重点为日文动词的「可能用法」。没错，我们又要学习动词变化了。所谓的「可能用法」，为表示「能够做什么事」的用法，相当于中文的「会…能够…」，例如「见る→见られる(能看到)」「食べる→食べられる(能够吃)」「会う→会える(能够见面)」等等。

 

不是说短时间内不会再碰到动词变化了吗!

 

我彷佛可以听到各位怒吼的声音，不过请先听我解释一下吧。其实本来动词「可能用法」这个动词变化，是预计放在更之后的单元再进行介绍的，想说先介绍一些生活中常用的文法和例句后，等到量累积到一定程度之后，再回来介绍动词变化，学习起来会较有效率、也会比较轻松。

 

但是，当我在编写文法相关教学文章时，特别是在构思文法例句时，无论如何注意，总会不小心用到这个「可能用法」。由于我们的文法教学已经进入了中级程度，像「私は学生です」这种幼儿园句型，相信大家都熟到

 

不行了，继续使用这种例句，相信对各位日语能力的提升并不会有帮助，但是，如果想造出稍微深一点的例句，又无可避免地会使用到「会…能够…」等等「可能用法」的用法。

 

例：（もらう）

　　その本を取ってもらえますか。

　　（から?だから）

　　おなかが痛いから、学校に行けない。

　　おなかが痛い。だから、学校に行けない。

 

因此，为了我们之后教材例句的多样性、也为了提升各位的日文能力，我们将在本单元中，先介绍日文动词「可能形」的变化方式和使用方法，以及类似句型用法的比较。

 

 

可能形用法

 

所谓日文动词的「可能形用法」，意思是「能够做什么事」，相当于中文的「会…能够…」。以下我们将简称为「可能形」。

 

动词可能形的变化方式

 

第一类动词

 

将动词罗马拼音字尾的「U」改成「E」 之后，加上「る」。

 

例：

書(か)く　kaku →　書(か)け kake　→　書(か)ける

泳(およ)ぐ　oyogu →　泳(およ)げ　oyoge  →　泳(およ)げる

言(い)う　iu →　言(い)え　ie  →　言(い)える

待(ま)つ　matsu  →　待(ま)て　mate  →　待(ま)てる

読(よ)む　yomu  →　読(よ)め　yome  →　読(よ)める

 

第二类动词

 

直接去掉动词字尾的「る」，加上「られる」。

 

食(た)べる　→　食(た)べられる

着(き)る　→　着(き)られる

忘(わす)れる　→　忘(わす)れられる

寝(ね)る　→　寝(ね)られる

開(あ)ける　→　開(あ)けられる

見(み)る　→　見(み)られる　

 

不规则动词

没有规则，只能直接记起来。

 

する　→　できる

勉強(べんきょう)する　→　勉強(べんきょう)できる

来(く)る　→　来(こ)られる

 

 

 

 

 

 

 

基本句型 : ～は（可能形）。

 

～は～を／が（可能形）。

 

例：

 

私(わたし)は走(はし)れる。

(我会跑步。)

私(わたし)は走(はし)れない。

(我不会跑步。)

彼(かれ)は泳(およ)げる。

(他会游泳。)

彼女(かのじょ)は飛行機(ひこうき)に乗(の)れない。

(她无法坐飞机。)

弟(おとうと)は朝起(あさおき)られない。

(弟弟早上起不来。)

妹(いもうと)は料理(りょうり)できない。

(妹妹不会做菜。)

 

祖父(そふ)はインターネットが使(つか)える。

(祖父会用网络。)

祖母(そぼ)は自分(じぶん)で海外旅行(かいがいりょこう)ができる。

(祖母能够自己去国外旅行。)

 

 

 

木村(きむら)さんは一戸建(いっこだ)てが買(か)える。

(木村先生能够买透天屋。)

朱(しゅ)さんは1000個(こ)の新(あたら)しい英単語(えいたんご)が覚(おぼ)えられる。

(朱先生能够记得1000个新英文单字。)

学校(がっこう)の地下室(ちかしつ)ではシャワーが浴(あ)びられる。

(学校地下室能够淋浴。)

 

 

一般句型中，动词前方使用助词「を」，但是如果后方是「可能形」的话，也可以使用助词「が」。一般的情况下，使用「が」的频率较高。

 

例：

 

私は纳豆を食べます。　私は纳豆が食べられます。

私はパソコンを使います。　私はパソコンが使えます。

私は仕事をします。　私は仕事ができます。

 

另外，在口语会话中，会将第二类动词「可能形（～られる）」中的「ら」省略，成为「～れる」的形式。虽然在学术界中，这是不符合传统文法的日语(又称「ら抜き言叶」），不过现在日本约有80%的人都这样使用，反倒是很少人在说话时使用「可能形（～られる）」这样的形式了。不过，在文章或正式场合，仍然会使用「可能形（～られる）」，请特别注意。

 

例：

 

私(わたし)は納豆(なっとう)が食(た)べられます。→私(わたし)は納豆(なっとう)が食(た)べれます。

私(わたし)は勉強会(べんきょうかい)に来(こ)られます。→私(わたし)は勉強会(べんきょうかい)に来(こ)れます。

私(わたし)は朝起(あさお)きられません。→私(わたし)は朝起(あさお)きれません。

 

 

 

ほかの可能用法

 

除了以上介绍的动词可能形之外，还有二种特别的可能用法

 

见える    ( 看得见 )

闻こえる  ( 听得见 )

 

以上二种可能用法，和我们刚才介绍过的可能形变化方法不同。

普通的可能形变化方法为:

 

 

见る　→　见られる ( 可以看见 )

闻く　→　闻ける   ( 可以听见 )

 

因此，「见る」「闻く」这二个动词，有二种可能形用法:

 

见る　:　见られる ( 可以看见 )  见える    ( 看得见 )

闻く　:　闻ける   ( 可以听见 )  闻こえる  ( 听得见 )

 

二者的差别在于 :

 

见える : 

 

中文为「看得见」，表示某物体或景物不受任何阻隔、自然而然地进入视野。

 

例：この窓(まど)から富士山(ふじさん)が見(み)える。

    (从这扇窗户看得见富士山。)

　　頭(あたま)をあげると、太陽(たいよう)が見(み)える。

    (抬起头就看得见太阳。)

　　学校(がっこう)では沢山(たくさん)の学生(がくせい)が見(み)える。

    (学校看得见许多学生。)

　　動物園(どうぶつえん)ではいろいろな動物(どうぶつ)が見(み)える。

    (动物园看得见各种动物。)

　　阿里山(ありざん)では綺麗(きれい)な朝日(あさひ)が見(み)える。

    (在阿里山看得见漂亮的日出。)

　　目(め)が見(み)える／目(め)が見(み)えない

    (眼睛看得见／看不见)

 

 

见られる :

 

中文为「可以看见」，表示经由自己的个人判断，去选择看特定的东西。

 

 

例：

 

あのホテルの屋上(おくじょう)に行(い)けば、富士山(ふじさん)が見(み)られる。

(到那栋旅馆的顶楼，就能看见富士山。)

学校(がっこう)では午後(ごご)１時(じ)に飛行機(ひこうき)が見(み)られる。

(在学校，下午一点能看见飞机。)　　

動物園(どうぶつえん)でパンダが見(み)られる。

(在动物园能看见猫熊。)

阿里山(ありざん)なら、綺麗(きれい)な朝日(あさひ)が見(み)られる。

(如果是阿里山，那么就能看到漂亮的日出。)

あの映画館(えいがかん)で「AVATAR」という映画(えいが)が見(み)られる。

(在那座电影院里可以看阿凡达。)

　　

 

 

闻こえる:

 

中文为「听得见」，表示某物体或景物不受任何阻隔、自然而然地进入耳朵中。

 

闻ける: 

 

中文为「可以听见」，表示经由自己的个人判断，去听特定的声音。

 

例：

 

朝(あさ)ニワトリの鳴(な)き声(ごえ)が聞(き)こえる。

(早上听得见公鸡叫。)

テレビを付(つ)けると、アナンサーの声(こえ)が聞(き)ける。

(转开电视后，就可以听到主播的声音。)

夜中(よなか)に隣(となり)の人(ひと)が喧嘩(けんか)する声が聞(き)こえる。

(半夜听得到隔壁邻居吵架的声音。)

MP3プレーヤーを持(も)っていると、沢山(たくさん)の音楽(おんがく)が聞(き)ける。

(有MP3随身听的话，就能够听很多音乐。)

CD屋(や)ではいろいろな音楽(おんがく)が聞(き)こえる。

(在唱片行听得见许多音乐)  可能是店里广播

CD屋(や)ではいろいろな音楽(おんがく)が聞(き)ける。
