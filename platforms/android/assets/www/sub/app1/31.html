单元31    顺序の言い方

             (先后顺序的说法)

学习目标

 

①  学习表示「…的时候」的句型「～ときに」

②  学习表示「在…之前」的句型「～前に」

③  学习表示「在…之前」的「て形」と「～てから」句型

④  学习表示「在…之后」的句型「～た后で」

 

本单元要介绍的是日文中「动作先后顺序」的说法，具体来说，就是中文当中「…的时候…」「在…之前…」「在…之后…」等等句型，用于表示二个动作间的时间先后关系，例如：「ご饭を食べる前に、手を洗う」「日本语を勉强する前に、日本文化に兴味を持てなければなりません」等等，在日常生活中经常使用，特别是用在叙述事情发展、以及给予建议时。

 

我们在本单元要学习的有四项句型，分别为表示「…的时候」「…之前」「…之后」等句型用法。其中会用到我们学过的「动词て形」和「动词た形」，因此如果对于这二项动词变化尚不熟练的话，请再多加练习，我们往后将会继续用到这些常用的动词变化形式。

 

其中，「～てから」「～前に」「～后で」的用法较相似，因此在后半的篇幅中，也将介绍如何区别这几个相似的用法、以及如何依照不同的场合来使用不同的句型，请试着体会其中些微不同的语感。

 

～ときに

 

「～ときに」的汉字写做「～时に」，意思为「在…的时候」，表示在某动作进行的同时，发生另一项动作。例如:「在洗澡的时候，电话响了。」

文法方面，由于「～ときに」的「とき」属于名词，因此和接续名词的方法完全相同。

 

基本用法: 动词原形／ている＋ときに

　　   　 名词＋のときに

　　　    な形容词＋なときに

　　　    い形容词＋ときに

 

例：風呂(ふろ)に入(はい)っているときに、電話(でんわ)が鳴(な)りました。

( 在洗澡的时候，电话响了。)

　　スーパーで買物(かいもの)するときに、大学(だいがく)の先生(せんせい)に会(あ)いました。

( 在超市买东西的时候，遇到了大学老师。 )

　　部屋(へや)を掃除(そうじ)しているときに、昔(むかし)の写真(しゃしん)を見付(みつ)けました。

( 在打扫房间的时候，找到了以前的照片。 )

　　夜寝(よるね)ているときに、母(はは)は急(きゅう)に大声(おおごえ)で私(わたし)を呼(よ)びました。

( 晚上睡觉的时候，妈妈突然大声叫我。)

　　パソコンを使(つか)うときに、画面(がめん)がフリーズしました。

( 在用计算机的时候，画面当机了。)

　　面接(めんせつ)するときに、スーツを着(き)なければなりません。

( 在面试的时候，必须穿着西装。)

　　必要(ひつよう)なときに、私(わたし)に連絡(れんらく)してください。

( 有需要的时候，请联络我。)

 

动词て形

 

我们之前学过，「动词て形」可以用来接续二个句子或是动词。

 

例如: 歯を磨いて、朝ごはんを食べます。

( 刷完牙、吃早饭 )　

 

其实这就是最基本的、表示时间先后顺序的用法。试想一下，当我们听到「刷完牙、吃早饭」这样的中文句子时，下意识里就会觉得是「先刷完牙、再吃早饭」，「刷牙」在先，「吃早饭」在后吧!  日文也相同，如果只是单纯接续二个句子，那么听者会自然而然地认为，先讲时间较早，而后讲的时间较晚。这就是最基本的表示时间先后的用法。

 

 

基本用法:  ( 句子1 ) 动词て形、( 句子2 )

 ( 先…，再…。)

 

例：

 

朝起(あさお)きて、ご飯(はん)を食(た)べて、学校(がっこう)に行(い)きます。

( 早上起来，吃饭，去学校。)

家(うち)に帰(かえ)って、テレビを見(み)て、宿題(しゅくだい)をします。

( 回到家，看电视，写作业。)

スーパーに行って、晩(ばん)ご飯(はん)の食材(しょくざい)を買(か)って、果物(くだもの)を买いました。

( 去了超市、买了晚餐的食材、买了水果。)

電源(でんげん)を入(い)れて、パソコンを起動(きどう)して、インターネットに接続(せつぞく)します。

( 按下开关、启动计算机、连上网络。)

雨(あめ)が降(ふ)って、風(かぜ)が吹(ふ)いて、桜(さくら)の花(はな)が落(お)ちます。

( 下雨、刮风、樱花掉落一地。)

 

～前(まえ)に

 

「～前に」句型，顾名思义，相当于中文的「在…之前」，表示在进行某动作之前、先做某项动作。和「动词て形」不同，这个句型很明确地表示出时间前后顺序的关系，而且整句话的重点在后半句，例如「ご饭を食べる前に、手を洗います。( 吃饭前，先洗手。) 」，整句话想强调的在于后句「手を洗います」 。

 

文法用法和「～ときに」大致相同。

 

 

基本句型: 　A ＋前(まえ)に、B。

          ( 在A之前，先B。)

 

基本用法: 动词原形＋前(まえ)に

　　　　　名词＋の前(まえ)に

　　　　　い形容词＋前(まえ)に

　　　　　な形容词＋な前(まえ)に

 

例：

 

教室(きょうしつ)に入(はい)る前(まえ)に、先生(せんせい)に挨拶(あいさつ)します。

( 进教室之前，先向老师打招呼。)

夜寝(よるね)る前(まえ)に、風呂(ふろ)に入(はい)ります。

( 晚上睡觉前，先洗澡。)

日本(にほん)に留学(りゅうがく)する前(まえ)に、日本語(にほんご)をマスターします。

( 去日本留学之前，先学好日文。 )

料理(りょうり)を作(つく)る前(まえ)に、ちゃんと包丁(ほうちょう)を洗(あら)わなければなりません。

( 在做菜之前，必须先将菜刀好好洗干净。 )

ホテルの予約(よやく)の前(まえ)に、宿泊(しゅくはく)の日(ひ)を決(き)めてください。

( 预约旅馆之前，请先决定住宿日期。 )

日本語能力試験(にほんごのうりょくしけん)を申(もう)し込(こ)む前(まえ)に、受験願書(じゅけんがんしょ)を買(か)わなければなりません。

( 申请日文检定之前，必须先买申请表。)

 

～てから

 

刚才介绍过，「动词て形」能够表示先后顺序，而「动词て形＋から」这个用法，则相当于「动词て形」的加强语气，表示顺序的语感更为强烈，相当于中文「…之后」。

 

「～てから」和「～前に」的不同之处:

 

「～前に」: 重点在后半句。

例：寝る前に、宿题をします。

 

「～てから」: 表示动作连续性，重点在前半句。

例：手を洗ってから、ご饭を食べます。

 

基本句型: 　A　てから、B。

( 先A，再B。)

文法用法: 动词て形＋から、( 句子 )。

 

例：

 

シャワーを浴(あ)びてから、テレビを見(み)ます。

( 冲完澡之后，看电视。 )

よく考(かんが)えてから、返事(へんじ)をします。

( 好好考虑之后，再给予回答。 )

医者(いしゃ)に見(み)てもらってから、家族(かぞく)に知(し)らせます。

( 给医生看过之后，再通知家人。 )

昼(ひる)ごはんを食(た)べてから、アルバイトに行きます。

( 吃完午饭后，再前去打。 )

私(わたし)は買物(かいもの)するときに、実物(じつぶつ)を見(み)てから買(か)います。

( 我买东西时，都是先看过实品之后再买。)

 

 

～た後(あと)で

 

「～た后で」句型，用到我们之前学过的「动词过去式(动词た形)」，照字面解释，就是「在…之后」之意。「～た后で」和「～前に」是互相对应的句型，一个表示「在…之前」、一个表示「在…之后」。

 

基本用法: 「动词た形」＋后で、～。

 

例：

 

宿題(しゅくだい)をした後(あと)で、友達(ともだち)と遊(あそ)びに行(い)きました。

( 写完作业后，和朋友去玩了。 )

テレビを見(み)た後(あと)で、風呂(ふろ)に入(はい)りました。

( 看完电视后，去洗了澡。 )

クラスメートに質問(しつもん)した后で、彼達(かれたち)の反応(はんのう)を見(み)ます。

( 问完同学之后，看看他们的反应。  )

本屋(ほんや)でビジネス書(しょ)を買(か)った後(あと)で、本棚(ほんだな)を買(か)いに行(い)きました。

( 在书店买了商业书籍后，去买了书架。  )

野球(やきゅう)の練習(れんしゅう)をした后で、みんなで食事(しょくじ)に行(い)きました。

( 棒球练习完了之后，大家一起去吃了饭。 )

 

 

 

三项句型的比较

 

例句：在睡觉前，先写作业

 

「～前に」

寝る前に、宿题をします。

（睡觉前去写作业。）

重点、话者想强调的是在后半句。

 

「～た后で」

宿题をした后で、寝ます。

（写完作业之后去睡觉。）

重点、话者想强调的是在后半句。

 

「～てから」

宿题をしてから、寝ます。

（写完作业之后才去睡觉。）

重点、话者想强调的是在前半句。
