①  理解表示距离远近指示词「こそあど」的基础用法及句型。

 

 

中文当中，我们常会指着某样东西说「这个?那个?哪个」，对着一处场所说「这里?那里?哪里」，依照距离远近，来指称我们所看到的事物。这是种很方便的代名词，当我们不晓得东西名字时，比如点餐时，只要指着菜单说「我要这个」就解决了。在日文中，也有同样的用法。这类用语，语言学上称为「指示词」，包括「这里那里」「这个那个」「这些那些」等等，简单来说，就是用来指称东西时用的字汇。

 

日文的基本指示词为:

 

これ：相當於中文的「這個」。用来指距离自己近的东西。　

それ：相當於中文的「那個」。用来指距离对方较近的东西。

あれ：相當於中文的「那一個」。用来指距离双方都远的东西。

どれ：相當於中文的「哪個」。疑问词。用来询问。

基本句型用法

 

①(指示詞) は (名詞) です。

 

　例：これ は 小(しょう)説(せつ) です。 (这是小说。)

　　　それ は ジュース です。 (那是果汁。)

　　　あれ は カーテン です。 (那是窗帘。)

　　　

②  (名詞) は (疑問詞) ですか。

 

　　例：小(しょう)説(せつ) は どれ ですか。(小说是哪一个?)

　　　　これ は 何(なん) ですか。 (这个是什么? )

　　

③  (指示詞) は (名詞) の (名詞) です。

 

　　例：これ は 私(わたし)の小(しょう)説(せつ) です。(这是我的小说。)

それ は 彼(かれ)のジュース です。 (那是他的果汁。)

あれ は 店(てん)長(ちょう)のカーテン です。(那是店长的窗帘。)

 

 

图解

 

情况一: 当物体距离A方较近时

 

 


A                      B

  

 

 

 

A:これは何(なん)ですか。(这是什么? )

B:それは正(せい)方(ほう)形(けい)です。(那是正方形。)

A：なるほど、これは正(せい)方(ほう)形(けい)ですね。(原来如此，这是正方形啊。)

 

 

情况二: 当物体距离B方较近时

 

 

 

　　

　A　　　　　　　　　　　B　　　

 

 

A:それは何(なん)ですか。(那是什么? )

B:これは正(せい)方(ほう)形(けい)です。(这是正方形。)

A：なるほど、それは正(せい)方(ほう)形(けい)ですね。(原来如此，那是正方形啊。)

 

 

情况三:当物体距离双方都很远时

 

 

 

 

 


　　 A　　　　　　　　　　  B　　　　

 

 

A:あれは何(なん)ですか。(那是什么? )

B:あれは正(せい)方(ほう)形(けい)です。(那是正方形。)

A：なるほど、あれは正(せい)方(ほう)形(けい)ですね。(原来如此，那是正方形啊。)

情况四:提出疑问时

 

 

          

   A         　          B

　　　　　？？？？　

 

A：正(せい)方(ほう)形(けい)はどれですか。(正方形是哪一个啊? )

B：さあ、私(わたし)もわかりませんね。(呃..我也不知道)

 

 

 

「これ?それ?あれ?どれ」用于指称一件东西，若是改写成「ここ?そこ?あそこ?どこ」，则变成指一个「地点」。例如学校、家里、餐厅、医院、咖啡厅等等地点。

 

ここ：相当于中文的「这里」。用来指距离自己近的场所。　

そこ：相当于中文的「那里」。用来指距离对方较近的场所。

あそこ：相当于中文的「那～里」。用来指距离双方都远的场所。

どこ：相当于中文的「哪里」。疑问词。用来询问。

 

例：？　これは私(わたし)の学(がっ)校(こう)です。（？这个是我的学校。）

　　Ｏ　ここは私(わたし)の学(がっ)校(こう)です。（○这里是我的学校。）

　　？　それは病(びょう)院(いん)ですね。（？那个是医院。）

　　Ｏ　そこは病(びょう)院(いん)ですね。（○那里是医院。）

　　？　あれは喫(きっ)茶(さ)店(てん)です。（？那个是咖啡厅。）

　　Ｏ　あそこは喫(きっ)茶(さ)店(てん)です。（○那里是咖啡厅。）

 

　　Ｏ　そこはどこですか。（○那里是哪里? ）

 

有没有发现，「これ?それ?あれ?どれ」和「ここ?そこ?あそこ?どこ」前面的「こ?そ?あ?ど」部分完全一样。称为日文的「こそあど」系统。应用范围广泛，像是「这位那位哪位」、「这次那次哪次」，也是一样以「こ?そ?あ?ど」这个系统来表示。

 

 

那么，如果我们不想讲「これ」，而想直接说出名称(例如这只狗、这部车)时，该怎么办呢?

 

「これ?それ?あれ?どれ」＋「名詞」

＝「こそあど」＋「の」＋「、名詞」

 

例：　これ(這個)＋犬(狗)＝？これ犬　○この犬(いぬ) (这只狗)

 

それ(那個)＋車(車)＝？それ車　○その車(くるま) (这部车)

 

あれ(那個)＋人(人)＝？あれ人　○あの人(ひと) (那个人)

 

 

「の」这个助词，有一个大家都十分熟悉用法，就是当作中文的「的」。

 

例：これはノートです。 (这是笔记本。)

　　これは私(わたし)のノートです。(这是我的笔记本。)

　　このノートは私(わたし)のノートです。(这本笔记本是我的笔记本。)

　　このノートは私(わたし)のです。(这本笔记本是我的。)

　　私(わたし)のノートはどれですか。(我的笔记本是哪一个? )

　　

それは誰(だれ)のパソコンですか。(那是谁的计算机? )

これは私(わたし)のパソコンです。(这是我的计算机。)

A:あれは鈴(すず)木(き)さんの車(くるま)ですか。(那是铃木先生的车吗? )

B: はい、そうです。あれは鈴(すず)木(き)さんの車(くるま)です。

(是的，没错。那是铃木先生的车。)

 

いいえ、違(ちが)います。あれは鈴(すず)木(き)さんの車(くるま)ではありません。

(不，不是。那不是铃木先生的车。)

会话例：

（二位女生在百货公司逛街时）

 

A：これ、見てー　       (你看这个~)

B: どれどれ～           (哪个哪个~)

A：これこれ～           (这个这个~)

B：あ～それか～いいね   (啊~那个吗~真不错呢)

A：でしょう～           (对吧~)
