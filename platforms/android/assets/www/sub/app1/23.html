单元23    日本语の逆接用法

             (日文中的转折语气)

学习目标

 ①　了解表示转折语气的「が」「けど」「のに」「くせに」用法

②　学习连接词「ですが」「でも」「しかし」「なのに」

 

我们在单元二十一时，介绍过日文原因理由的说法，以及「から」「ので」二个助词。「因为…所以…」这样的句型，称为「顺接用法」，所谓的「顺接用法」，指的是按照「原因→结果」的顺序描述的句型，「因为今天下雨，所以远足取消」，「今天下雨」是原因，「远足取消」是结果，因此这是一句「顺接用法」。

 

有「顺接用法」，那么当然就会有「逆接用法」，「逆接用法」为按照「动作→意料之外的结果」顺序描述的句型。我们可以换个比较容易理解的中文说法: 「转折语气」。简单来说，转折语气就是「虽然…但是…」这样的句型，前面是动作、后面是出乎意料之外的结果。例如: 「虽然台风来了，但是还是有人去海边」，「台风来了」是发生的动作，「有人去海边」是意料之外的结果，因此这种句型称为「逆接用法(转折语气)」。

 

我们本单元的主题，就是要学会日文中的「转折语气」说法。简单整理如下:

 

 

「顺接用法(正常语气)」: 「原因→结果」→「因为…所以…」

「逆接用法(转折语气)」: 「动作→意料之外的结果」→「虽然…但是…」

 

日文中表示「虽然…但是…」句型的助词，常见的有「が」「けど」「のに」「くせに」四个助词，之后的篇幅将介绍其使用方法和使用时机。另外，既然有表示「原因理由」的连接词(如だから?なので)，那么也会有表示「虽然…但是…」的连接词，用以连结二个句子，我们也会一并介绍「ですが」「でも」「しかし」「なのに」四个连接词。

 

 

助词×４

 

「逆接用法(转折语气)」的基本句型用法如下:

 

基本句型: 

(动作)＋「が／けれども／のに／くせに」、(意料之外的结果)

 (虽然…但是…)

 

经常使用于转折语气的助词，主要有以下四个:

 

「が」

 

表示轻微的转折语气，相当于中文的「虽然…但是…」，主要用于文章或正式场合中，为书面用语。

 

文法: 　名词?な形容词＋だが／ですが

　　　　い形容词?动词＋が

 

例：

 

この店(みせ)は高(たか)いですが、全然(ぜんぜん)美味(おい)しくありません。

( 虽然这间餐厅很贵，但是一点都不好吃。)

明日(あした)からは夏休(なつやす)みですが、宿題(しゅくだい)がいっぱいあります。

( 虽然明天开始是暑假，但是有很多作业。 )

パソコンを買(か)いましたが、キーボードがありません。

( 虽然买了计算机，但是却没有键盘。 )

彼女(かのじょ)はかわいいですが、わがままな人(ひと)です。

( 虽然她很可爱，但是却很任性。 )

北川(きたがわ)さんはたくさん食(た)べましたが、全然太(ぜんぜんふと)りません。

( 虽然北川小姐吃了很多，但是完全不会变胖。 )

テレビを付(つ)けましたが、画面(がめん)がなかなか出(で)ません。

( 虽然开了电视，但是却没有画面。 )

 

「けれども」

 

和「が」相同，表示轻微的转折语气，相当于中文的「虽然…但是…」。主要用于口语会话中，为口头用语。在日常会话中，也会使用「けれど」「けど」的形式。

 

文法: 名词?な形容词＋だけれども／ですけれども

　　　い形容词?动词＋けれども

 

例(れい)：

 

この店(みせ)は高(たか)いけれども、全然(ぜんぜん)美味(おい)しくありません。

( 虽然这间餐厅很贵，但是一点都不好吃。 )

あの歌手(かしゅ)は有名(ゆうめい)だけれども、歌(うた)が下手(へた)です。

( 虽然那位歌手很有名，但是却很不会唱歌。 )

明日(あした)からは夏休みだけれども、宿題(しゅくだい)がいっぱいあります。

( 虽然从明天开始是暑假，但是有很多作业。 )

パソコンを買(か)いましたけれども、キーボードがありません。

( 虽然买了计算机，但是却没有键盘。)

彼女(かのじょ)はかわいいけれども、わがままな人(ひと)です。

( 虽然她很可爱，但是却很任性。)

北川(きたがわ)さんはたくさん食(た)べましたけれども、全然太(ぜんぜんふと)りません。

( 虽然北川小姐吃了很多，但是却完全不会胖。 )

テレビを付(つ)けましたけれども、画面(がめん)がなかなか出(で)ません。

( 虽然开了电视，但是却没有画面。 )

 

 

「のに」

 

表示较强烈的转折语气，相当于中文的「明明…但是…」。

 

文法:   名词?な形容词＋なのに

　　　  い形容词?动词＋のに

 

 

例(れい)：夏(なつ)なのに、陽明山(ようめいざん)で雪(ゆき)が降(お)りました。

( 明明是夏天，但是阳明山却下雪。)

　　もう25歳(さい)なのに、両親(りょうしん)からお小遣(こづか)いをもらいます。

( 明明已经二十五岁了，但是却还是领父母亲的零用钱。)

　　あの店(みせ)の弁当(べんとう)は美味しいのに、40元(げん)もかかりません。

    ( 那间店的便当明明很好吃，但是却花不到四十元。 )

　　この絵(え)がきれいなのに、誰(だれ)も買(か)いません。

    ( 那幅画明明很漂亮，但是却没有人买。 )

　　もう朝(あさ)なのに、なかなか起(お)きることができません。

    ( 明明已经是早上，但是却怎么样都起不来。 )

　　目覚(めざ)まし時計(どけい)をかけたのに、なかなか鳴(な)りません。

    ( 明明已经设定了闹钟，但是却没有响。 )

 

「くせに」

 

表示最强烈的转折语气，有强烈不满、抱怨的感觉，相当于中文的「明明是…但是却…」。主要用在抱怨、批评的时候。

 

文法接续方式和名词相同:

 名词＋のくせに

な形容词＋なくせに

い形容词?动词＋くせに

 

例(れい)：

 

給料(きゅうりょう)をたくさんもらうくせに、全然仕事(ぜんぜんしごと)をしません。

( 明明领了很多薪水，但是却完全不工作。 )

お金(かね)を振(ふ)り込(こ)んだくせに、商品(しょうひん)がまったく来(き)ません。

( 明明将钱汇过去了，但是商品却完全没有送来。 )

日本語先生(にほんごせんせい)のくせに、日本語(にほんご)で自己紹介(じこしょうかい)することもできません。

( 明明是日文老师，但是却连用日文自我介绍都不会。 )

お金持(かねも)ちのくせに、スーパーでさんざん値切(ねぎ)りをしました。

( 明明是有钱人，但是却在超市不断地杀价。 )

この車(くるま)はとんでもない高(たか)いくせに、すぐ故障(こしょう)します。

( 明明是贵得要死的车子，但是却动不动就故障。 )

ここの公務員(こうむいん)は暇(ひま)なくせに、ひたすら文句(もんく)を言(い)います。

( 这里的公务员明明很闲，但是却一直抱怨东抱怨西。)

 

 

连接词×４

 

以上介绍的助词，和连接词的对应关系如下:

 

「が」→「ですが」

「介于『が』和『のに』之间」→「でも」「しかし」

「のに」→「なのに」

 

 

 

基本句型:  (发生的动作)。连接词、 ( 意料之外的结果)。

 

「ですが」

 

表示轻微的转折语气，相当于中文的「但是…」，主要用于文章或正式场合中，为书面用语。

 

例：この店(みせ)は高(たか)いです。ですが、全然(ぜんぜん)美味(おい)しくありません。

( 这间餐厅很贵。但是，却一点都不好吃。)

 

「でも」

 

表示略为强烈的转折语气，相当于中文的「但是…」，主要用于会话中，为口头用语。

 

例：明日(あした)からは夏休みです。でも、宿題(しゅくだい)がいっぱいあります。

( 明天开始是暑假。但是，有很多作业。 )

 

「しかし」

 

和「でも」相同，但主要用于文章和正式场合中，为书面用语。

 

例：テレビを付(つ)けました。しかし、画面(がめん)がなかなか出(で)ません。

( 开了电视。但是，画面却一直出不来。 )

 

「なのに」

 

表示较强烈的转折语气，相当于中文的「明明…但是…」。

 

例：この絵(え)がきれいです。なのに、誰(だれ)も買(か)いません。

( 这幅画很漂亮。但是，却没有人买。)
