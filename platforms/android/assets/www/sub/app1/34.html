单元34    する?なる

 

 

学习目标

 ①　了解「する」「なる」的意思

②　学会「名词＋する?なる」的用法

③  学会「形容词＋する?なる」的用法

 

本单元的主角为「する」和「なる」二个单字。

 

「才二个单字，几行就解决了，为什么还要另外辟一个单元介绍!?」

 

我们彷佛可以听到各位心里的吶喊和疑惑，为什么要花一个单元的时间介绍「する」「なる」这二个动词呢?  其实，这二个动词可没有外表看起来那么单纯，单独这二个字的意思很好懂，但是它们就像口香糖一样，可以「黏」在名词、形容词、动词等等词类的后面，赋予整句话不同的意思，这种特殊用法，在学习时就必须特别注意了。

 

「する」这个动词，大概已经在例句中出现过三百遍了，属于不规则动词，中文是「做」的意思，可以接在汉字之后，形成新的动词，例如：「勉强する」「仕事する」等等。而「なる」这个动词，是「成为…」的意思，可以用在「医者になる」「歌手になる」等等的例句当中。以上为该字本身

 

的意思，但是当「する」和「なる」和名词、形容词等等一起连用时，那么情况可就大不相同了。

 

「する」「なる」若是和名词、形容词等其他词类结合时，会由「做」「成为」等具体意思，变成「使…变成…」「…自然成为…」等抽象的意思。呃，做个简单易懂的比喻，当东尼．史塔克单独一人时，只是普通的人类，但是穿上钢铁装时，就是能够飞天遁地的钢铁人了，我发誓这是我所能想到最好的比喻。

 

「＋名词」：歌手になる　歌手にする

「＋形容词」：美味しくなる　美味しくする

 

「する」「なる」还可以接副词和动词等等，本单元由于篇幅关系，只挑了最实用的名词和形容词介绍，不过其实意思都相同，只要了解「する」「なる」和名词、形容词接续时的意思和用法，往后遇上其他词类时，也能很迅速地理解。

 

する?なる

 

「する」

 

不规则动词，相当于中文「做…」的意思。

前面多接助词「を」。

 

基本用法: ～をする／します。

 

例：

 

宿題(しゅくだい)をする。　( 写作业。)

仕事(しごと)をする。    ( 工作。 )

料理(りょうり)をする。   ( 做菜。 )

買物(かいもの)をする。   ( 买东西。 )

復習(ふくしゅう)をする。  ( 复习。 )

 

 

也可以直接将「する」接在汉字之后，形成新的动词。

 

例：

 

勉強(べんきょう)する。  ( 学习。 )

登校(とうこう)する。   ( 上学。 )

仕事(しごと)する。    ( 工作。 )

購入(こうにゅう)する。  ( 买入。 )

展示(てんじ)する。    ( 展示。 )

 

「なる」

 

第一类动词，相当于中文「成为…」的意思。

前面多接助词「に」。

 

基本用法: ～になる／なります。

 

例：

 

教師(きょうし)になる。      ( 成为老师。 )

漫画家(まんがか)になる。     ( 成为漫画家。 )

 

宇宙飛行士(うちゅうひこうし)になる。 ( 成为航天员。 )

有名(ゆうめい)な人(ひと)になる。  ( 成为有名的人。 )

お金持(かねも)ちになる。   ( 成为有钱人。 )

 

名词＋する?なる

 

「なる」

 

「名词＋なる」的基本用法和「なる」相似，相当于中文「…变成…」

 

 

基本用法:  ～　は　（名词）　に　なる。

 

例：

 

私(わたし)は大統領(だいとうりょう)になる。   

( 我成为总统。)

彼(かれ)は日本語教師(にほんごきょうし)になる。    

( 我成为日文老师。 )

彼女(かのじょ)は物理(ぶつり)の博士(はかせ)になる。 

( 她成为物理博士。 )

村上(むらかみ)さんはノーベル賞(しょう)の作家(さっか)になる。 

( 村上先生成为诺贝尔奖的作家。)

国中(くになか)さんは有名(ゆうめい)な俳優(はいゆう)になる。

( 国中小姐成为有名的演员。 )

 

「する」

 

「名词＋する」的意思为「将…变成…」，指经由外力，使某样东西变成另一样东西。

 

基本用法:  ～は A（対象）を  B（名词） に  する。

(  以外力将A变成B。 )

 

在原始用法「をする」的中间，加入「（名词）に」，表示变化的对象。

 

 

例：

 

私(わたし)は書(か)いた原稿(げんこう)を本(ほん)にする。 

( 我将写完的原稿制成书。)

彼女(かのじょ)はあの野良犬(のらいぬ)をペットにする。

( 她将那只野狗当作宠物。)

農人(のうにん)たちは取(と)った稲(いね)を白(しろ)い米(こめ)にする。

( 农人们将割下的稻子制成白米。 )

母(はは)はこの古(ふる)い布を雑巾(ぞうきん)にする。

( 母亲将旧布制成抹布。 )

私(わたし)はノートパソコンをテレビにする。

( 我将笔电当作电视。)

あのバカは二本(にほん)の鉛筆(えんぴつ)を箸(はし)にして、ご飯(はん)を食(た)べる。

( 那个笨蛋将二只铅笔当作筷子来吃饭。 )

 

 

形容词＋する?なる

 

「形容词＋なる」的意思为「…自然变成…」，表示该物体不经由外力、自然而然地变成某个状态。

 

 

基本用法：い形容词－「い」＋「くなる」

　　　　　な形容词＋　に　なる

 

例：

 

水(みず)が熱(あつ)くなる。( 水变热。 )

背(せ)が高(たか)くなる。 ( 身高变高。 )

眠(ねむ)くなる。( 变得想睡。 )

氷山(ひょうざん)が小(ちい)さくなる。( 冰山变小。 )

部屋(へや)がきれいになる。( 房间变干净了。 )

病人(びょうにん)が元気(げんき)になる。( 病人变得有精神。)

周(まわ)りが静(しず)かになる。  ( 周围变安静。 )

日本語(にほんご)が上手(じょうず)になる。( 日文变得拿手。 )

 

「形容词＋する」的意思为「使…变成…」，表示经由外力，使某物体变成某个状态。

 

基本用法：                          

（対象）を（い形容词）－「い」＋「くする」

（対象）を（な形容词）に　する

 

例：

 

水(みず)を熱(あつ)くする。      ( 使水变热；煮热水)

音量を大(おお)きくする。   ( 将音量调大 )

料理(りょうり)を美味(おい)しくする。( 将料理变得好吃)

白(しろ)い紙(かみ)を黒(くろ)くする。  ( 将白纸染黑 )

部屋(へや)をきれいにする。 ( 将房间变干净)

クラスを静かにする。 ( 让全班安静)

彼女(かのじょ)を幸(しあわ)せにする。( 让她幸福)

日本語(にほんご)を上手(じょうず)にする。( 让日文能力变好)

 

简单整理

 

＋なる：指该物不受外力影响，自然而然变成某种状态。

        「…自然变成…／…自然成为…」

＋する：指使用外力，让该物由原本的状态变成某种状态。

        「使…变成…／使…成为…」
