单元32    「同时」を表す言い方

             (表示同时的说法)

 

学习目标

 ①　学习表示「一边…一边…」的「～ながら、～」句型。

②　学习表示「…，直接…」「～たまま、～」

 

这次我们要学习的内容，和前一单元有些类似，不过因为篇幅过长，因此分成二个单元介绍，这样各位学习起来也较省力(笑)。

 

上一单元介绍的是「时间先后顺序」的说法，而本单元将介绍「同时…」的说法。解释成「同时…」也有点不太精确，我们将介绍二个句型: 「～ながら、～」和「～たまま、～」。「～ながら、～」相当于中文的「一边…一边…」，同时进行二项动作，例如：「テレビを见ながら、ご饭を食べます」「音楽を闻きながら、宿题をします」。「～たまま、～」则相当于「…，就直接…」，用来描述意料之外发生的动作情况，例如：「スーツを着たまま、寝ました(穿着西装，就直接睡着了)」「靴を履いたまま、リビングに入りました( 穿着鞋子，就直接进来客厅里了)」等等例句。

 

本单元的句型用法，也会用到之前介绍过的动词变化，分别为「动词ます形」和「动词た形」。嘿，我们说过动词变化很重要吧，就是指这类的情况，有很多很多很多日常生活的会话句型，都和动词变化息息相关、锁得紧紧的，如果不熟悉动词变化，那么可能只能说出断断续续、单字词组程度的日文，而无法流畅地进行沟通。各位也许会见到许多日语学习者，拥

 

有非常丰富的字汇量，但是说话时只能说出单字，而无法将所有单字符串起来形成完整流畅的句子，这就是文法不熟悉的关系。如果是日文的情况，那么特别是指不熟悉动词变化的情形。

 

ながら

 

「～ながら、～」为表示二项动作同时进行的句型，相当于中文的「一边…一边…」。这里需要特别注意的是，必须将二个动作中最主要的动作，接在「ながら」之后。例如「テレビを见る」「ご饭を食べる」，由于是「吃饭顺便看电视」而不是「看电视顺便吃饭」，「テレビを见る」是主要动作，「ご饭を食べる」为顺便动作，因此必须写成「テレビを见ながら、ご饭を食べる」，而不能写成「ご饭を食べながら、テレビを见ます」。

 

基本文法: 「动词ます形」－「ます」＋ながら、～。

 

基本句型: 　A　ながら、B。

(一边A，一边B )  B为主要动作，A为顺便的动作

 

例：

 

テレビを見(み)る。　ご飯(はん)を食(た)べる。　

テレビを見(み)ながら、ご飯(はん)を食(た)べます。

( 一边看电视，一边吃饭。)

 

歩(ある)く。　食(た)べる。　

歩(ある)きながら、アイスクリームを食(た)べます。

( 一边走路，一边吃冰淇淋。)

 

 

話(はな)す。運転(うんてん)する。　

友達(ともだち)と話(はな)しながら、車(くるま)を運転(うんてん)します。

( 一边和朋友讲话，一边开车。)

 

抱(かか)える。　飛(と)び込(こ)む。

彼(かれ)は大(おお)きい荷物(にもつ)を抱(かか)えながら、電車(でんしゃ)に飛(と)び込(こ)みました。

( 他一边抱着很大的行李，一边冲进电车里。 )

 

買物(かいもの)する。　考(かんが)える。　

母(はは)は買物(かいもの)しながら、今晩(こんばん)のレシピを考(かんが)えています。

( 母亲一边买东西，一边思考今晚的菜单。 )

 

仕事(しごと)する。　通(かよ)う。

彼(かれ)は仕事(しごと)しながら、大学(だいがく)の夜間部(やかんぶ)に通(かよ)っています。

( 他一边工作，一边在大学夜间部念书。 )

 

～たまま

 

「～たまま」句型，为持续前面动作的情况下，进行后面的动作，相当于中文的「…，就直接…」。「まま」原意为「一直；照样」，而前面必须接「动词た形」，由于前面是已经做过的动作，因此必须使用表示过去式的「动词た形」。

 

基本文法: 「动词た形」＋まま、～。

 

基本句型: 　Aたまま、B。

(  A，就直接B。)

 

另外，这个句型较常用在描述「意想不到」的事情上。

 

例 : ○　スーツを着たまま、寝ました。

　　 ×　パジャマを着たまま、寝ました。

 

例：

 

テレビを付(つ)けたまま、ソファで寝(ね)ました。

( 开着电视，就直接在沙发上睡着了。 )

靴(くつ)を履(は)いたまま、部屋(へや)に入(はい)りました。

( 穿着鞋子，就直接进了房间。 )

ドアを開(あ)けたまま、外(そと)に出掛(でか)けました。

( 门开着，就直接出去了。 )

車(くるま)を店(みせ)の前(まえ)に止(と)めたまま、店(みせ)に食事(しょくじ)に行(い)きました。

( 将车子停在店门口，就直接进去店里吃东西了。 )

彼(かれ)は家出(いえで)をしたまま、何(なん)の連絡(れんらく)もありませんでした。

( 他离家出走后，就一直没有任何音讯。 )

彼(かれ)は「さよなら」と言(い)って東京(とうきょう)に行(い)ったまま、二度(にど)と帰(かえ)りませんでした。

( 他说了一句”再见”后去了东京，就再也没有回来过了。 )

彼女(かのじょ)はランドセルをかけたまま、マラソンを走(はし)りました。

( 她背着书包，就直接去跑马拉松了。 )
