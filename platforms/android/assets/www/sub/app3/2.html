形容詞的語尾變化總整理

學習目標

①　復習形容詞的分類

②　復習形容詞在不同時態中的語尾變化

③　復習其他常見的形容詞變化

本單元我們將復習日文中「形容詞」的相關用法，包括形容詞的分類、時態和其他語尾變化。

日文當中的形容詞系統，和中文最大的不同點，就在於日文形容詞分成二大類：い形容詞、な形容詞。中文形容詞語尾，統一都使用「的」，例如「好吃的、美麗的、有名的」，日文則大不相同，い形容詞和な形容詞不但語尾不同，後面連接名詞時的變化方式也不同。另外，中文當中並沒有「時態」的概念，形容詞也不會隨著現在或過去的不同而使用不同型式，就這一點來說，日文中不但動詞會有時態變化，連形容詞都會有時態的變化，因此在學習時也必須多加注意。

聽起來很複雜嗎?  其實並不會，歸納起來只有以下三項重點:

①　形容詞的分類

②　形容詞的時態

③　形容詞的敬語和常體

別擔心，這些文法大家在基礎文法的階段一定都學過，如本單元開頭所說，現在只是要為各位進行「復習」的工作，讓各位的日文基礎更加穩固，因此，在閱讀時，只要確保沒有忘記這些重要文法概念就可以了。

形容詞分類

日文中的形容詞，分為二類: 「い形容詞」和「な形容詞」

い形容詞: 結尾為「い」的形容詞

な形容詞: 結尾不是「い」的形容詞

二類形容詞的差別，主要反映在「後面接續名詞」的情況

い形容詞: 直接接續名詞

な形容詞: 必須加上假名「な」之後才能接續名詞

例：

美味しい　料理(りょうり)　   （ 美味的料理。 ）

まずい　デザート    （ 難吃的甜點。 ）

大(おお)きい　時計(とけい)        （ 大的時鐘。 ）

小(ちい)さい　花(はな)          （ 小的花。 ）

広(ひろ)い　アパート       （ 寬廣的公寓。 ）

狭(せま)い　部屋(へや)           （ 狹小的房間。 ）

静(しず)か　な　講演(こうえん)      （ 寂靜的演講。 ）

賑(にぎ)やか　な　夜市(やじ)     （ 熱鬧的夜市。 ）

元気(げんき)　な　子供        （ 有活力的小孩。）

親切(しんせつ)　な　店員(てんいん)       （ 親切的店員。 ）

上手(じょうず)　な　解説(かいせつ)       （ 厲害的解說。 ）

色々(いろいろ)　な　国(こく)          （ 各式各樣的國家。 ）

但是，也有例外 ，有一些語尾為「い」的形容詞，其實屬於「な形容詞」

きれい

→　雖然語尾是「い」，但是漢字寫法為「綺麗」，語尾看不到假名「い」，

因此屬於「な形容詞」

有名（ゆうめい）

→　漢字寫法「有名」，語尾也看不到假名「い」，屬於「な形容詞」

嫌い（きらい）

→　純粹例外，屬於「な形容詞」，需特別注意。

例：

きれいな服(ふく)。（ 漂亮的衣服。 ）

有名(ゆうめい)な歌手(かしゅ)。（ 有名的歌手。 ）

嫌(きら)いな食(た)べ物(もの)。（ 討厭的食物。 ）

( 形容詞語尾變化，見下頁。)

形容詞的常體變化

い形容詞：おいしい

 

肯定

否定

現在

 おいしい おいしくない

過去

 おいしかった おいしくなかった

原則: 「った」表示過去，「ない」表示否定。

例：

カレーライスがおいしい。        （ 咖哩飯很好吃。）

カレーライスがおいしかった。    （ 之前的咖哩飯很好吃。 ）

カレーライスがおいしくない。    （ 咖哩飯不好吃。 ）

カレーライスがおいしくなかった。（ 之前的咖哩飯不好吃。 ）

な形容詞：静(しず)か

 

肯定

否定

現在

 静かだ 静かではない

過去

 静かだった 静かではなかった

原則: 「った」表示過去，「ない」表示否定。

例：

公園は静かだ。　（ 公園很安靜。 ）

公園は静かだった。  （ 公園之前很安靜。 ）

公園は静かではない。  （ 公園不安靜。 ）

公園は静かではなかった。  （ 之前公園不安靜。 ）

形容詞的敬體變化（です・ます形）

い形容詞：おいしい

 

肯定

否定

現在

 おいしいです。 おいしくありません。おいしくないです。

過去

 おいしかったです。 おいしくありませんでした。おいしくなかったです。

原則:

①　將常體直接加上「です」，就可以成為敬體。

②　否定形有二種說法，可以直接將常體加上「です」，也可以使用

「ありません」，基本上二種説法都是正確的。

現在否定: ◯　おいしくないです　 　(較接近年輕人用語)

◯　おいしくありません

過去否定: ◯　おいしくなかったです　(較接近年輕人用語)

◯　おいしくありませんでした

例：

ラーメンがおいしいです。

（ 拉麵很好吃。 ）

ラーメンがおいしかったです。

（ 之前的拉麵很好吃。 ）

ラーメンがおいしくないです。／おいしくありません。

（ 拉麵不好吃。 ）

ラーメンがおいしくなかったです。／おいしくありませんでした。

（ 之前的拉麵不好吃。 ）

な形容詞：静(しず)か

 

肯定

否定

現在

 静かです。 静かではないです。静かではありません。

過去

 静かでした。 静かではなかったです。静かではありませんでした。

原則:

肯定形: 將「だ」「だった」→「です」「でした」。

否定形: 將常體直接加上「です」、或是使用「ありません」。

現在否定: 静かではないです。 (較接近年輕人用語)

静かではありません。

過去否定: 静かではなかったです。(較接近年輕人用語)

静かではありませんでした。

其他形容詞語尾變化

其他形容詞語尾變化有:

①　形容詞「ば」形

②　形容詞名詞化：「形容詞＋さ・み」

③　形容詞動詞化：「形容詞＋がる」

①　形容詞「ば」形

形容詞「ば」形 : 用於假設句型時，相當於中文的「…的話」之意。

變化方法:  (い形容詞)　去「い」 加「ければ」

(な形容詞)　加「であれば」

例：

美味(おい)しい　→　美味(おい)しければ  （ 如果好吃的話 ）

大(おお)きい　→　大(おお)きければ　   （ 如果大的話 ）

広(ひろ)い　→　広(ひろ)ければ         （ 如果寬廣的話 ）

元気(げんき)　→　元気(げんき)であれば      （ 如果有活力的話 ）

親切(しんせつ)　→　親切(しんせつ)であれば    （ 如果親切的話 ）

安全(あんぜん)　→　安全(あんぜん)であれば    （ 如果安全的話 ）

料理(りょうり)が美味(おい)しければ、毎日行(まいにちい)く。

（ 如果料理好吃的話，每天都去。 ）

時計(とけい)が大(おお)きければ、年配者(ねんぱいしゃ)も見(み)やすいでしょう。

（ 如果時鐘大一點的話，高齡者也很容易看吧。 ）

アパートが広(ひろ)ければ、たくさん買(か)い物(もの)ができる。

（ 如果公寓大一點的話，就可以買很多東西。 ）

子供(こども)は元気(げんき)であれば、それを越(こ)したことはない。

（ 如果小孩有活力的話，就沒有比這個更好的事了。 ）

解説(かいせつ)が親切(しんせつ)であれば、お客様(きゃくさま)もどんどん増(ふ)えるでしょう。

（ 如果解說相當親切的話，客人也會逐漸增多吧。 ）

安全(あんぜん)であれば、俺(おれ)はジェットコースターに乗ってもいいよ。

（ 如果安全的話，那麼要我坐雲宵飛車也可以喔。 ）

形容詞名詞化

變化方法: い形容詞－「い」＋「さ・み」

な形容詞－「い」＋「さ」

「い形容詞＋さ」「い形容詞＋み」，有什麼地方不一樣呢?

「さ」：表示程度，具體可以計算。

「み」：表示状態，抽象不可計算。

「い形容詞」

楽しさ:  表示快樂的程度，很快樂、普通快樂、不快樂?

楽しみ:  表示快樂的狀態本身，可以說成「樂趣」。

「な形容詞」

主要使用「な形容詞＋さ」，而不使用「な形容詞＋い」的形式。

静かさ：表示安靜的程度，安靜到不行、普通安靜、一點都不安靜等等。

静かみ：不使用

形容詞動詞化

變化方法: い形容詞－「い」＋「がる」

な形容詞＋「がる」

形容詞： 形容心理的状態。

形容詞＋がる： 將心理状態表露於外表的樣子。

面白い  : 心裡覺得很有趣。只能用於第一人稱。

面白がる:  外表表現出覺得很有趣的樣子。用於第二三人稱。

例：

このゲームは面白(おもしろ)い。（ 這個遊戲很有趣。 ）

彼(かれ)はこのゲームを面白(おもしろ)がる。（ 看樣子他覺得這個遊戲很有趣。 ）

学校(がっこう)に行(い)くのがいやだ。（ 討厭去學校。 ）

彼女(かのじょ)は学校(がっこう)に行(い)くのをいやがる。（ 看樣子她討厭去學校）

關於「形容詞名詞化」和「形容詞動詞化」的詳細介紹，請參照文法教材第五十四單元「形容詞的名詞化．動詞化」。
