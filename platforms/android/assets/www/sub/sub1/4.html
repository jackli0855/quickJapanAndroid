[ 單元4 ] 日本語の「こ・そ・あ・ど」

Back

講義本文:

學習目標

①  理解表示距離遠近指示詞「こそあど」的基礎用法及句型。

中文當中，我們常會指著某樣東西說「這個・那個・哪個」，對著一處場所說「這裡・那裡・哪裡」，依照距離遠近，來指稱我們所看到的事物。這是種很方便的代名詞，當我們不曉得東西名字時，比如點餐時，只要指著菜單說「我要這個」就解決了。在日文中，也有同樣的用法。這類用語，語言學上稱為「指示詞」，包括「這裡那裡」「這個那個」「這些那些」等等，簡單來說，就是用來指稱東西時用的字彙。

日文的基本指示詞為:

これ：相當於中文的「這個」。用來指距離自己近的東西。

それ：相當於中文的「那個」。用來指距離對方較近的東西。

あれ：相當於中文的「那一個」。用來指距離雙方都遠的東西。

どれ：相當於中文的「哪個」。疑問詞。用來詢問。

基本句型用法

① (指示詞) は (名詞) です。

例：これ は 小説(しょうせつ) です。 (這是小說。)

それ は ジュース です。 (那是果汁。)

あれ は カーテン です。 (那是窗簾。)

②  (名詞) は (疑問詞) ですか。

例：小説(しょうせつ) は どれ ですか。(小說是哪一個?)

これ は 何(なん) ですか。 (這個是什麼? )

③  (指示詞) は (名詞) の (名詞) です。

例：これ は 私(わたし)の小説(しょうせつ) です。(這是我的小說。)

それ は 彼(かれ)のジュース です。 (那是他的果汁。)

あれ は 店長(てんちょう)のカーテン です。(那是店長的窗簾。)

圖解

情況一: 當物體距離A方較近時

A                      B

A:これは何(なん)ですか。(這是什麼? )

B:それは正方形(せいほうけい)です。(那是正方形。)

A：なるほど、これは正方形(せいほうけい)ですね。(原來如此，這是正方形啊。)

情況二: 當物體距離B方較近時

A　　　　　　　　　　　B

A:それは何(なん)ですか。(那是什麼? )

B:これは正方形(せいほうけい)です。(這是正方形。)

A：なるほど、それは正方形(せいほうけい)ですね。(原來如此，那是正方形啊。)

情況三:當物體距離雙方都很遠時

A　　　　　　　　　　  B

A:あれは何(なん)ですか。(那是什麼? )

B:あれは正方形(せいほうけい)です。(那是正方形。)

A：なるほど、あれは正方形(せいほうけい)ですね。(原來如此，那是正方形啊。)

情況四:提出疑問時

A         　          B

A：正方形(せいほうけい)はどれですか。(正方形是哪一個啊? )

B：さあ、私(わたし)もわかりませんね。(呃..我也不知道)

「これ・それ・あれ・どれ」為指稱一件東西，若是改寫成「ここ・そこ・あそこ・どこ」，則變成指一個「地點」。例如學校家裡餐廳醫院咖啡廳等等地點。

ここ：相當於中文的「這裡」。用來指距離自己近的場所。

そこ：相當於中文的「那裡」。用來指距離對方較近的場所。

あそこ：相當於中文的「那裡」。用來指距離雙方都遠的場所。

どこ：相當於中文的「哪裡」。疑問詞。用來詢問。

例：X　これは私(わたし)の学校(がっこう)です。（×這個是我的學校。）

O　ここは私(わたし)の学校(がっこう)です。（○這裡是我的學校。）

X　それは病院(びょういん)ですね。（×那個是醫院。）

O　そこは病院(びょういん)ですね。（○那裡是醫院。）

X　あれは喫茶店(きっさてん)です。（×那個是咖啡廳。）

O　あそこは喫茶店(きっさてん)です。（○那裡是咖啡廳。）

O　そこはどこですか。（○那裡是哪裡? ）

有沒有發現，「これ・それ・あれ・どれ」和「ここ・そこ・あそこ・どこ」前面的「こ・そ・あ・ど」的部分是完全一樣的。稱為日文的「こそあど」系統。應用範圍廣泛，像是「這位那位哪位」、「這次那次哪次」，也是一樣以「こ・そ・あ・ど」這個系統來表示。

那麼，如果我們不想講「これ」，而想直接說出名稱(例如這隻狗 這部車)時，該怎麼辦呢?

「これ・それ・あれ・どれ」＋「名詞」

＝「こそあど」＋「の」＋「、名詞」

例：　これ(這個)＋犬(狗)＝×これ犬　○この犬 (這隻狗)

それ(那個)＋車(車)＝×それ車　○その車 (這部車)

あれ(那個)＋人(人)＝×あれ人　○あの人 (那個人)

「の」這個助詞，還有一個大家都十分熟悉用法，就是當作中文的「的」。

例：これはノートです。 (這是筆記本。)

これは私(わたし)のノートです。(這是我的筆記本。)

このノートは私(わたし)のノートです。(這本筆記本是我的筆記本。)

このノートは私(わたし)のです。(這本筆記本是我的。)

私(わたし)のノートはどれですか。(我的筆記本是哪一個? )

それは誰(だれ)のパソコンですか。(那是誰的電腦? )

これは私(わたし)のパソコンです。(這是我的電腦。)

A:あれは鈴木(すずき)さんの車(くるま)ですか。(那是鈴木先生的車嗎? )

B: はい、そうです。あれは鈴木(すずき)さんの車(くるま)です。

(是的，沒錯。那是鈴木先生的車。)

いいえ、違(ちが)います。あれは鈴木(すずき)さんの車(くるま)ではありません。

(不，不是。那不是鈴木先生的車。)

一般會話時，使用這些指示詞的機會非常頻繁，你或許聽過這個的對話:

会話例：

A：これ、見てー　      (你看這個~)

B:どれどれ～           (哪個哪個~)

A：これこれ～          (這個這個~)

B：あ～それか～いいね  (啊~那個嗎~真不錯呢)

A：でしょう～          (對吧~)
