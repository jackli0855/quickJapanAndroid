我們這回要介紹的是，日文中的形容詞。如同我們在先前的講義中提到的，形容詞的功能是用來形容名詞，如:「漂亮的花」「乾淨的房間」，其中「花」和「房間」是名詞，「漂亮的」和「乾淨的」即是形容詞。形容詞的概念，對中文母語者來說，並不難理解，我們日常生活中無時無刻不用到形容詞，正確且廣地使用形容詞，會讓你的話顯得更為生動，同時更精準地傳達自己的意思(例如比起說「這裡有怪味道」，說「好臭!」更能快速且精準地表達自己想法)。因此，我們將形容詞的學習放在初期階段。

日文中的形容詞，和中文略為不同，主要分為二大類: 「い形容詞」及「な形容詞」。關於名稱的由來，之後會詳細介紹。我們將依序學習如何區分這二種形容詞、形容詞的基本用法句型、以及形容詞的否定形。

二大種類

日文形容詞分為二類:い形容詞和な形容詞，區分方法很簡單。

い形容詞:以假名い結尾的形容詞

。

例：長(なが)い(長的)。 短(みじか)い(短的)。 広(ひろ)い(寬的)。 狭(せま)い(窄的)。

深(ふか)い(深的)。 浅(あさ)い(淺的)。  おいしい(好吃的)。 まずい(難吃的)。

寂(さび)しい(寂寞的)。 忙(いそが)しい(忙碌的)。 楽(たの)しい(快樂的)。

暖(あたた)かい(溫暖的)。

名稱由來: 以い形容詞來修飾名詞時。

使用「い形容詞＋名詞」的方式。

例：長(なが)い傘(かさ) (很長的傘)。 短(みじか)い鉛筆(えんぴつ) (很短的鉛筆)。

広(ひろ)い部屋(へや) (很寬的房間)。狭(せま)い部屋(へや) (很窄的房間)。

深(ふか)い湖(みずうみ) (很深的湖)。 浅(あさ)い川(かわ) (很淺的河)。

おいしいデザート(很好吃的甜點)。まずい料理(りょうり) (很難吃的料理)。

寂(さび)しい人(ひと) (很寂寞的人)。 忙(いそが)しい会社員(かいしゃいん) (很忙碌的上班族)。

楽(たの)しい一日(いちにち) (很快樂的一天)。暖(あたた)かい手(て) (很溫暖的手)。

な形容詞:不以假名い結尾的形容詞

例：静(しず)か (安靜的)。 親切(しんせつ) (親切的)。 賑(にぎ)やか (熱鬧的)。

便利(べんり) (方便的)。元気(げんき) (有精神的)。 簡単(かんたん)(簡單的)。

上手(じょうず) (拿手的)。 下手(へた) (不拿手的)。

名稱由來: 以な形容詞來修飾名詞時。

使用「な形容詞＋な＋名詞」的方式。

例：静(しず)かな教室(きょうしつ) (安靜的教室)。 親切(しんせつ)な先生(せんせい) (親切的老師)。

賑(にぎ)やかなスーパー (熱鬧的超市)。 便利(べんり)な所(ところ) (方便的地方)。

元気(げんき)な子供(こども) (朝氣十足的小孩)。 簡単(かんたん)な質問(しつもん) (簡單的問題)。

上手(じょうず)なコック (厲害的廚師)。下手(へた)な選手(せんしゅ) (憋腳的選手) 。

另外，有些形容詞經常會讓人產生混亂。

例：有名（ゆうめい；有名的）・綺麗（きれい；漂亮的）

嫌い（きらい；討厭的）

前面二個形容詞，雖然寫成假名時，是以「い」結尾，但是由於寫成漢字時，後面並不會出現「い」，因此將它們歸類在「な形容詞」。最後的「嫌い」算是特例，雖然是「い」結尾，但仍屬於「な形容詞」。

例：有名(ゆうめい)な歌手(かしゅ) (有名的歌手)。きれいな公園(こうえん)(乾淨漂亮的公園)。

嫌(きら)いな店員(てんいん) (討人厭的店員)。

形容詞的肯定句型

以形容詞來修飾名詞時，除了使用「形容詞＋名詞」的形式外，我們也可以運用「～は～です」這個基本句型。二種都可以，意思上沒有區分。

肯定句型: (名詞) は (い形容詞/な形容詞) です。

例：長(なが)い傘(かさ)　→　傘は長いです。

短(みじか)い鉛筆(えんぴつ)　→　鉛筆は短いです。

おいしいデザート　→　デザートはおいしいです。

静(しず)かな教室　→　教室は静かです。

親切(しんせつ)な先生　→　先生は親切です。

賑(にぎ)やかなスーパー　→　スーパーは賑やかです。

形容詞的否定句型

形容詞的否定用法，各自不同。

い形容詞：將字尾的「い」改成「く」後，加上「ありません」。

例：(傘不長)、  (鉛筆不短)、  (甜點不好吃)

長(なが)い傘(かさ)　→　傘(かさ)は長(なが)いです →　傘(かさ)は長(なが)くありません。

短(みじか)い鉛筆(えんぴつ)　→　鉛筆(えんぴつ)は短(みじか)いです　→　鉛筆(えんぴつ)は短(みじか)くありません。

おいしいデザート　→　デザートはおいしいです　→

デザートはおいしくありません。

な形容詞：直接加上「ではありません」。

例：(教室不安靜)、 (老師不親切)、 (超市不熱鬧)

静(しず)かな教室→教室(きょうしつ)は静(しず)かです→教室(きょうしつ)は静(しず)かではありません。

親切(しんせつ)な先生(せんせい)→先生(せんせい)は親切(しんせつ)です→先生(せんせい)は親切(しんせつ)ではありません。

賑(にぎ)やかなスーパー　→　スーパーは賑(にぎ)やかです。　→

スーパーは賑(にぎ)やかではありません。
